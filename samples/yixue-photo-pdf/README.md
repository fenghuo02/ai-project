## 后续完善建议（生产版本应加入）

⚠️ 这是**纯 CSS 模拟**的样书。生产版本里这些应替换为真实素材：

1. **场景图位置（第 14-17 页 / 第 19-20 页）**
   - 当前用 CSS 渐变 + 大字"開/生/景/休/月/春"模拟
   - 生产版应**插入用即梦/Nano Banana 跑出的真实壁纸**
2. **客户姓氏印章**
   - 当前用占位字"主"
   - 生产版应**根据客户实际姓名刻章**（线上篆刻服务一枚 5-15 元）
3. **手写体字段**（序章 / 终章寄语）
   - 当前用思源宋体替代
   - 生产版应购买/找设计师**真正的硬笔楷书或钢笔体**
4. **24 节气和 7 节日完整版**
   - 当前为节选样书（春六节气 / 中秋 + 春节）
   - 标准档应展开到 24 节气全列 + 7 节日全列

## 技术栈

- HTML + CSS（纯静态，无 JS）
- WeasyPrint v66 转 PDF
- Noto Serif CJK SC 嵌入字体（Google 开源，可商用）
- A4 / 210mm × 297mm / @page 规则

## 如何重新生成 PDF

```bash
pip install weasyprint
python3 -c "from weasyprint import HTML; HTML('mingshu.html').write_pdf('mingshu-sample.pdf')"
```

或在浏览器中打开 `mingshu.html`，按 `Cmd/Ctrl + P` → 另存为 PDF
（浏览器版可能字体效果略有不同）
