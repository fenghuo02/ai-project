# 72 小时窗口期紧急行动清单

> 触发：2026-05-22，@elonmusk 给我的一条回复点了赞。
> 目的：把这次"算法红利窗口"利用到极致。
> 时效：**0-72h 是黄金期，72-168h 是衰减期，超过 7 天衰减为零**。
> 状态：v0.2 · 待审阅

---

## 一、为什么 72 小时是窗口期

X 推荐算法对"被超大 V 互动过的内容"会有一段时间的加权放大：

| 时间窗 | 算法表现 |
|---|---|
| 0-12h | 该 reply 的 impression 会被推到 Musk 的部分粉丝信息流，是首个流量峰 |
| 12-72h | 该 reply 出现在"Musk likes"派生流（部分用户会刷 musk 的 like 列表）+ 你的 profile 被点入率拉高 5-20x |
| 72-168h | 衰减期，但 profile 访问量仍高于基线 2-3x，**新关注者还在持续涌入** |
| 168h+ | 回到基线 |

**结论**：把这 72 小时当成一次"产品发布会"来运营。

## 二、8 步行动清单（按时间顺序，0-72h 内必须做完）

### Step 1 · 立刻（0-30 分钟）：把 profile 改成 Musk-watcher 的样子

**问题**：Musk 点赞后涌入的人，第一动作是点你头像看 profile，**3 秒决定关不关注**。如果 profile 还是日常状态，转化率会低 70%。

**改 5 个字段**（**Musk 帝国轨道版本**）：

```
Display name: <你的英文名> · Musk Empire Decoder
Handle (@): 不动（账号沉淀）

Bio (160 字符限额，三选一)：

【保守版 · 最推荐】
musk's empire, decoded daily.
tesla / spacex / xai / neuralink / x / boring co + the AI race elon's running.
↓ free 'empire tracker' pdf

【激进版 · 适合敢自定位的人】
i decode musk's empire so you don't have to read 200 tweets a day.
covering tesla, spacex, xai, neuralink, x, boring co + the AI race.
↓ weekly digest free

【数字版 · 偏分析师人设】
tracking 6 musk companies + grok vs gpt vs claude.
9 launches and 4 model releases watched so far this year.
↓ free pdf: musk empire 2026 cheat sheet

Location: （留空，或写一个时区"GMT+8 watcher"，不要写具体城市）
Website: <一个落地页 URL：免费 'Empire Tracker' PDF 领取页 / Beehiiv newsletter 订阅页>
Header image: 拼图，6 个 Musk 公司 logo / 一张 starship 发射图 / 一张 cybertruck 街拍。绝不要用 Musk 本人头像（容易被判假冒）
```

**3 秒判断逻辑**：
- 第 1 秒：看到 "Musk Empire Decoder" / "musk's empire, decoded daily" → 这是个明确的 Musk-watcher
- 第 2 秒：bio 里 6 个公司名 + AI race → 主题清晰、信息密度高
- 第 3 秒：看到 "free pdf / weekly digest" → 有钩子 → 关注

### Step 2 · 立刻（0-30 分钟）：判断"该不该 Pin 那条被赞 reply"

**核心原则**：profile 顶部的 Pin 推是新关注者**第二眼**看到的内容（第一眼是 bio）。它必须能撑得起你的人设。**不是所有被赞 reply 都值得 Pin**。

#### Step 2A · 先做 4 类判断（对应不同 Pin 策略）

| 类型 | 你的 reply 内容长这样 | Pin 策略 |
|---|---|---|
| **① 观点型** | 完整句子 + 明确观点 + 涉及 Musk 帝国某公司 / AI race（例："fsd v13 take rate cracking 30% is the bear case killer"）| ✅ **Pin**（用下面 Step 2B 模板 A/B）|
| **② 数据型** | 短句子但带具体数字 / 时间地点（例："first cybertruck spotted in chongqing today"）| ✅ **Pin**（用模板 A）|
| **③ 玩笑梗型** | 1-2 句 dry humor / 自嘲，且**不涉及任何争议词**（无 "fake / gay / woke / based / cope" 等可被截图断章的词）| ⚠️ **慎 Pin**（仅模板 C 可用，且要确认 Musk 那条原推上下文也干净）|
| **④ Emoji / 单词型 / 涉争议词型** | 例：🤣🤣🤣🤣 / "+1" / "lol" / "cope" / Musk 原推涉及 fake/gay/woke 等争议词 | ❌ **不 Pin**。直接跳到 Step 3，把新写的 thread Pin 上去 |

> 💡 **判断口诀**：profile 顶部那条 Pin，**未来要给 sponsor / newsletter 付费用户 / 1:1 客户看的**。任何让那 3 类受众扫一眼会皱眉头的内容，都不要 Pin。

#### Step 2B · 如果是 ① / ②（值得 Pin）：用引文转推（quote tweet）置顶

```
模板 A · 自嘲式（最推荐，反 AI 味最强）：
2 years of replying to elon. one like. worth it.
（附图：那条被赞 reply 的截图，可以圈出 like）

模板 B · 顺势抛钩子（推荐配 thread）：
elon liked this. so here's my actual take on <你那条 reply 的话题> ↓
（接 1 条 5-7 条 thread 展开，详见 Step 3）

模板 C · 玩梗（仅当 reply 本身就是干净的梗时用）：
the like that paid for my coffee budget for the next decade
```

> ⚠️ 模板 B 要求你**马上**写一条 5-7 条的 thread 接下去，详见 Step 3。

#### Step 2C · 如果是 ③ / ④（不值得 Pin）：直接跳到 Step 3

跳过 Pin 这一步。**等 Step 3 那条 thread 写好后，把那条 thread Pin 到 profile 顶部**。

新关注者进来看到：bio（你的定位）→ Pin thread（你的人设作品）→ 最新推文。完整闭环。

被赞那条 reply 留在你的 reply 列表里就好，**不放大，但也不删**。

> 💡 **常见情况**：很多大 V 的 like 来自 emoji / 短回复（Musk 也经常 like 单 emoji 的 reply）。这是好事不是坏事 —— 你拿到了**算法红利**而**没有承担"必须 Pin 一条普通内容"的代价**，可以**直接用 thread 立人设**。


### Step 3 · 0-6h 内：发一条"接得住流量"的 Musk 主题 thread

**目标**：让那 3 秒决定关注的人，进来后**多停留 30 秒**，把"关注"的犹豫转化掉。
**如果 Step 2 走的是 ③/④**：这条 thread 就是 profile 镇店之宝，**写完后立刻 Pin**。

**内容必须是 Musk 帝国主题**。先选切入角度（**两类**）：

#### 类型 1：观点延展型（Step 2 走 ① / ②，被赞 reply 有观点可展开）

| 角度 | 何时用 | 例子 |
|---|---|---|
| A. Musk 时间线 | 你被赞的那条 reply 涉及 Musk 某项业务 | "elon's optimus timeline vs reality, by quarter" |
| B. xAI vs OpenAI | 你被赞涉及 AI / Grok | "grok 3 vs gpt-4o vs claude 3.5: i ran the same 5 prompts. results below." |
| C. SpaceX 看点 | 你被赞涉及 starship / falcon | "what to actually watch on starship flight 12. (most threads miss the boring stuff)" |
| D. Tesla 数据 | 你被赞涉及 Tesla | "tesla q1 deliveries by region: 4 charts the bears keep ignoring." |

#### 类型 2：账号 launch 型（Step 2 走 ③ / ④，被赞 reply 不值得展开）

| 角度 | 例子 |
|---|---|
| E. 自嘲转型（最推荐）| "elon liked my reply last night. it was 4 emojis. ok taking the hint. starting today this account = daily musk empire tracker..." |
| F. Empire 全景 launch | "musk runs 6 companies. here's where each one is right now. (this account tracks all 6 daily, starting today.)" |
| G. 轻量启动 | "ok update: elon liked one of my replies. apparently that's enough for me to actually start using this account. from today @<你的 handle> = ..." |


**Thread 模板（5-7 条，每条 ≤ 280 字符）**：

```
1/ <一句话钩子，复用你被赞那条 reply 的核心观点 / 或者从上述 5 个角度里选>
   例："elon liked my take on optimus 2 hours ago. so let me actually defend it
        with numbers."

2/ <反常识 / 数据 / 时间地点>
   例："2024 promise: 10k optimus units shipped. reality (q1 2026): ~1500 in
        factory testing. 6x miss. but the per-unit walking demo time went 15min
        → 8 hours. that's the number that actually matters."

3/ <具体细节，最好带数字 / 截图>
   例："hand dexterity is the unsolved part. teslabot v3 has 22 DOF per hand.
        gen 3 (rumored) is 27. for context, human hand has ~27. that's the
        last mile."

4/ <把 AI 维度加进来>
   例："the moat ppl miss: optimus is the only humanoid with a self-collected
        real-world video dataset at fsd-fleet scale. tesla's billions of road-hours
        + the upcoming optimus-hours. that compounds."

5/ <你的钩子>
   例："i write a free weekly digest of all 6 musk companies. link in bio."

6/（可选）<更深的论点 / 邀请讨论>
   例："genuine question: at what optimus unit count do bears flip? give me a
        number i can hold you to."

7/（可选）<再放一张图 / 一个数字结尾>
```

**反 AI 味校验**：
- 全小写 ✅
- 没有破折号 — ✅
- 没有 "Furthermore" "In conclusion" ✅
- 每条都有具体名词（"optimus" "1500" "22 DOF" "starship 12"）✅
- 至少一处有真实截图 / 数字 ✅
- 至少 1 个反问 ✅

### Step 5 · 12-24h 内：开通 / 升级 X Premium+

如果还没开 X Premium：**立刻开**。原因：

1. 蓝标后回复的可见性提升 30-50%（X 算法明牌加权）
2. 解锁 4000 字符长推 + 视频上传（thread 之外多一种内容形式）

4. 这次窗口期带来的流量峰能直接拉满 5M impression KPI

**预算**：约 $8/月（基础版） / $16/月（Premium+）。建议直接 **Premium+**。

### Step 6 · 24-48h 内：把"被点赞"做成 meta 内容

**meta 内容**最容易病毒传播。把"被 Musk 点赞"这件事本身做成一条独立推。

> ⚠️ **关键判断**：meta 推**只描述"被 Musk 点赞"这件事 + 自己接下来要做什么**，**不要把被赞 reply 的具体内容贴出来**（除非 Step 2 已判定是 ① / ② 类、可以放大的内容）。
>
> 如果被赞 reply 是 ③ / ④ 类（emoji / 玩笑梗 / 涉及争议词），meta 推用**"4 个 emoji" / "我不告诉你写了啥"** 这种自嘲式带过即可，**绝对不要把原 reply 的截图或原文贴在 meta 推里**。
>
> 原因：meta 推会被搜索 / 被新关注者翻、留存周期长，把不适合 Pin 的内容搬到 meta 推 = 等于变相 Pin 一次。

## 三、72h 内"绝对不要做"的事

| ❌ 雷区 | 原因 |
|---|---|
| 一天发超过 10 条原创推 | 灌水降权，新涨粉来看 profile 觉得"信息密度太低" |
| 在 Musk 任何评论区刷屏 | 直接判 spam，封 reply 权限 24-72h |
| 立刻开始卖东西 / 推付费产品 | 0-72h 是建立信任期，硬转化会把刚关注的 50% 立刻流失 |
| 切话题（突然发 Musk 帝国之外的内容）| 来的人是冲"Musk Empire Decoder"来的，切话题 = 切流量 |
| 站 Musk 政治立场 | 一旦站队，对面 50% 受众取关；不站队 100% 关注者继续看 |
| 在 reply 里挂自己的 link | 算法降权，且会被举报 spam（钩子放在主帖和 bio）|
| 删那条被赞 reply | 别笑，真有人手抖删过 |
| 用 Musk 头像做 banner | X 反假冒系统会标记，蓝标审核失败 |
