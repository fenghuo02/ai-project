# 渝山东方美学 · 客户面合规红线

> **背景**：项目赛道是「东方美学命主肖像」，但与"算命/玄学"赛道高度相邻。
> 小红书 / 公众号 / 微信对这条边界的关键词检测很严，**一次违规可能限流 7 天，三次限流到死**。
> 这份文件记录所有客户面文档（小红书笔记、公众号文章、PDF 钩子、自动回复、菜单文案、视觉模板上的文字）必须遵守的禁词清单。

---

## 一、禁词清单（按敏感度分 4 级）

### 🚫 L1 · 算命术语（绝对禁，触发即限流）

```
算命  看相  八字  用神  调候  命格  格局
奇门  八门  死门  惊门  休门  开门  生门  伤门  杜门  景门
```

替换为：
- 东方美学 / 命主画像 / 本命色 / 应景色 / 调性
- 八种生活场景（**不要叫"八门"**）：开拓 / 疗愈 / 兴隆 / 出击 / 沉浸 / 表达 / 释怀 / 心安

### 🚫 L2 · 命理周边词（同样致命，比 L1 更隐蔽）

这一组是项目过程中**逐步发现**的雷点，最初的禁词清单**没**包含：

```
四柱  命理  占卜  玄学  大师  师傅  风水
命运  吉凶  运势  测命  化解  转运  改运
```

特别注意：
- **「玄学」单独使用高敏**（"玄学占卜类"在 PR C2 被替换为"非色彩美学方向的咨询服务"）
- **「四柱」是命理硬术语**（"用四柱里的五行属性"在 PR C2 被替换为"用东方传统的色彩理论体系"）
- **「命理」/「师傅」即使在 B 端 BD 上下文也要避**（"不做命理咨询" → "我们做的是色彩美学定制"）

### 🚫 L3 · 否定句式陷阱（**最隐蔽的雷点**）

**腾讯系 / 小红书的关键词检测只匹配关键词，不区分否定**。所以：

```
❌ "不算命、不预测"          → 仍含"算命"，会被检测
❌ "我们这边不是算命的方向"   → 同上
❌ "你不是来算命的"           → 同上
```

✅ 替换为完全不含禁词的表述：

```
✅ "不下结论、不做预测"
✅ "我们做的是色彩美学定制"
✅ "我们看东方美学的本命色"  （**这是项目的标准转向话术**，记住）
```

> ⚠️ 唯一例外：在 [`docs/playbook/wechat-onboarding.md`](../../docs/playbook/wechat-onboarding.md) §九 Q6 + [`prompts-content.md`](../../docs/playbook/prompts-content.md) Prompt 6 的「算命党评论标准转向话术」里，会出现"我们不算命"这种否定句式 —— 这些是**给主理人 / AI 看的反向场景演示**，不是直接发给用户的客户面文案。客户面文案的 transition 用 ✅ 区里的话术。

### 🚫 L4 · 营销违禁 + 私域导流

```
最 / 第一 / 绝对 / 永远 / 包治 / 包过 / 包准
免费送 / 速来抢 / 数量有限 / 仅此 X 名

微信 / VX / 加我 / 私聊 / 加 V
```

替换为：
- 营销：「整理了一份」「主页找我」「公众号回复 X 即可」「到店即赠」
- 私域：「主页」「后台回复」「自取」「预约表单」「商务邮箱」

> ⚠️ 公众号自动回复**严禁**直接给个人微信号 / 二维码图片 —— 腾讯系会判定为"诱导加私域"，轻则限流，重则封号。

### 🚫 L5 · 天干日主（出现在客户面正文 = 触发"算命"联想）

```
丁火  己土  壬水  乙木  甲木  丙火  戊土  庚金  辛金  癸水
```

虽然这些词不在腾讯系 / 小红书的明确禁词清单里，但普通读者一看就触发"这不就是算命？"的反应。

替换为柔和的季节调性表述：
- ✅ 「12 月生 · 红调」（不写"丁火"）
- ✅ 「8 月生 · 暖调」（不写"己土"）
- ✅ 「7 月生 · 深调」（不写"壬水"）
- ✅ 「2 月生 · 春调」（不写"乙木"）

> 内部生产端（[`mingshu-builder/`](../../mingshu-builder/) / [`swatch-builder/palette/colors.yaml`](../../swatch-builder/palette/colors.yaml) / `data.yaml`）允许保留天干日主 —— 那是给主理人和工具看的，不会出现在客户交付物里。

## 二、合规扫描两轮制（每次发布前必跑）

写完任何客户面文档（笔记 / 长文 / PDF / 自动回复 / 菜单），**必须跑两轮扫描**：

### 🔍 第一轮：禁词扫描

```bash
grep -nE "算命|看相|八字|用神|调候|命格|奇门|八门|死门|惊门" <file>
grep -nE "测命|化解|转运|改运|风水|吉凶|命运|运势|玄学|命理|师傅|大师" <file>
grep -nE "微信|VX|加我|私聊|免费送|绝对|永远|包治|包准" <file>
grep -nE "丁火|己土|壬水|乙木|甲木|丙火|戊土|庚金|辛金|癸水" <file>
```

**预期结果**：所有命中**只能**出现在以下 4 个上下文之一，不能出现在客户面正文：
1. 禁词清单本身（`❌ 算命 / 看相 ...`）
2. 自查清单条目（`- [ ] 没出现「算命 ...」`）
3. 给主理人 / AI 的开发指引（如 `wechat-onboarding.md` §九 Q6 标准转向话术）
4. 注释或注解（`> ⚠️ 严禁出现 ...`）

如果命中出现在第 5 类上下文（客户实际能看到的正文）→ 必须改。

### 🔍 第二轮：否定句式扫描

```bash
grep -nE "不算命|不是算命|不看相|不预测.{0,8}(命|未来|吉凶)" <file>
```

任何命中都要改成完全不含禁词的表述（参考 §一 L3 的 ✅ 区）。

## 三、自查清单模板（嵌入每个客户面文档）

每个客户面文档最后必须有一节 §合规自查 / §X 自查清单，至少包含以下项：

```markdown
## §合规自查（每次发布前过一遍）

- [ ] 没出现「算命 / 看相 / 八字 / 用神 / 调候 / 命格 / 奇门 / 八门 / 死门 / 惊门」
- [ ] 没出现「测命 / 化解 / 转运 / 改运 / 风水 / 吉凶 / 命运 / 运势」
- [ ] 没出现「大师 / 玄学 / 命理 / 师傅」单独使用
- [ ] 没出现「最 / 第一 / 绝对 / 永远 / 包治 / 免费送 / 速来抢」
- [ ] 没出现「微信 / VX / 加我 / 私聊」（用「主页 / 后台回复 / 自取」替代）
- [ ] 没出现否定句式陷阱（「不算命」「不是算命」等仍含禁词的表述）
- [ ] 天干日主词（丁火 / 己土 / 壬水 / 乙木 等）0 次出现在客户面正文
- [ ] HEX 色值与 swatch-builder/palette/colors.yaml 一致
```

参考实现：[`docs/playbook/lead-magnet-19-colors.md`](../../docs/playbook/lead-magnet-19-colors.md) §五（12 项最严版本）。

## 五、三件套对外命名约定（v2 新增 · 2026-05）

> **背景**：主理人脑子里和内部沟通时，经常会用「转运肖像 / 转运壁纸 / 命理书」这三个词
> 描述自己想做的事——这是真实业务诉求，**内部沟通继续用没问题**。
> 但只要进入客户面（小红书 / 抖音 / 视频号 / B 站 / 公众号 / PDF / 自动回复 / 合同 / 视频文案），
> **这三个词全部触雷**（"转运"=L2 禁、"开运"=L2 等价词、"命理"=L2 禁）。
> 故全项目统一翻译口径如下，所有客户面文档和 Prompt 输出都必须用右列。

### 5.1 三件套标准命名表

| 内部沟通用语 | 客户面/平台对外用语（择一即可，按场景选） |
|---|---|
| 转运肖像 / 五行肖像 | **命主画像** ⭐ / 五行美学肖像 / 本命气场画像 |
| 转运壁纸 / 开运壁纸 / 改运壁纸 | **节气壁纸** ⭐ / 应景壁纸 / 节庆美学壁纸 |
| 命理书 / 命理报告 | **命书** ⭐ / 美学命书 / 东方美学之书 |

> **优先用 ⭐ 标记的主推命名**——已在业务计划书 §1.3、playbook 多处确立，
> 改用其他名是合规等价但会增加跨文档不一致风险。

### 5.2 派生用词的对外说法

| 内部 / 口语 | 对外用语 |
|---|---|
| 转运照片 / 开运照片 | **节气写真 / 应景写真 / 节令美学写真** |
| 五行算法 / 排盘 / 起局 | 东方色彩理论体系 / 调候配色逻辑（仅公众号深度文可用，短视频/小红书禁用）|
| 给客户改运 / 让客户运势变好 | 帮客户找到本命色 / 把对自己的期待具象化 / 让一种情绪有了形状 |

### 5.3 工程化扫描（每次 PR 前过一遍）

```bash
# 扫所有客户面 md 是否漏了"转运/改运/改命/开运"未翻译
grep -nE "转运|改运|改命|开运" docs/playbook/*.md docs/business-plan-*.md \
  | grep -vE "禁词|❌|不可做|对外|合规|不出现|没出现|高危|限流|替换|严禁"
```

**预期结果**：0 命中。如果有命中，必须改成本表 §5.1/5.2 的对外用语。

> 目前已通过本扫描的文件：`prompts-yixue-photo.md` v2.1.1（PR-1, 2026-05）。
