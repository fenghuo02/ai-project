# 渝山东方美学 · 文创 SKU 总览

> 对应《商业计划书 v1.6》§十七「文创用品产品线」。三档矩阵覆盖不同人群，预售 + 小批量降低库存压力。
>
> 合规总则（所有 SKU 必须遵守）：
> - **无功效宣称**：不得出现改运/转运/招财/辟邪/养生/疗效等表述（广告法 + 封建迷信红线）。
> - **客户面命名**只围绕「节气 / 五行美学 / 命主画像 / 节气壁纸 / 命书 / 纪念」，不得算命占卜话术。
> - **AIGC 标识**：含 AI 生成元素的图样，须在详情页与成品上标注「含 AI 辅助生成」。
> - **实拍铁律**：涉及人像一律实拍，AI 仅用于无人脸背景；绝不 deepfake / 网图。
> - **产品标签**：实物须含材质、执行标准、厂名厂址、必要警示语。
> - 特殊品类（食品/茶/香薰/化妆品/儿童用品）须取得资质，否则不上架。
