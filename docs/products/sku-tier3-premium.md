## SKU-301　命主画像 · 装裱艺术微喷

- **客户面文案**：将你的「命主画像」以收藏级艺术微喷输出，专业装裱，可挂于厅堂书房的东方美学之作。
- **价格**：¥2,999 起（按尺寸/装裱）
- **交付物**：艺术微喷成品 + 实木/铝框装裱 + 收藏证书（含 AIGC 标识说明）。
- **规格**：博物馆级艺术纸 / 收藏微喷；尺寸【A2/A1 可选】；人像 100% 实拍为底，背景 AI 辅助并标注。
- **生产流程**：实拍 → 配色合成 → 收藏级输出 → 装裱 → 质检 → 顺丰加固发货。
- **个人信息**：肖像与个人信息授权，明确商用范围与期限。
- **售后**：定制按单生产，定稿后不退；运输破损 7 日内凭开箱视频包换。

- **价格**：¥6,800
- **交付物**：实拍精修片 + 命主画像（数字+装裱）+ 美学纪念册精装本。
- **生产流程**：拍摄（实拍铁律）→ 精修 → 画像合成 → 美学纪念册排版印装 → 整套交付。
- **合规标注**：套装内任何含 AI 元素的成果均标注；文案不作运势/功效表述。
- **售后**：服务+商品组合，拍摄服务按合同；实物定制定稿后不退。

- **价格**：¥19,800 起（按方案）

- **生产流程**：需求沟通 → 方案与报价 → 合同与定金 → 设计打样 → 批量生产 → 验收。
- **合规与合同**：签**B 端定制合同**（交付标准、知识产权、验收、违约、发票）；批量生产留**质检与留样**记录。
- **售后**：按合同验收条款，质量问题包修/换。

### 三档运营要点

- **全部按单生产，零库存**；先签合同收定金（建议 30%–50%）再投产。
- B 端与顶配款必须**走合同 + 对公开票**，明确 IP、验收、违约。
- 严守红线：高端品最易被「升值/改运/风水功效」话术带偏，文案统一只谈**工艺、美学、限量、纪念**。
