# 「山海有灵」系列 · 防伪设计方案

> 全系列通用防伪标准。目标：**降低仿冒、保护收藏价值、支撑限量溢价**，同时**合规、不作功效宣称**。
> 各神兽文档中的「防伪暗记」只标注该兽**专属隐藏暗记位置**，其余通用要素见本文件。

## 一、三级防伪体系（按 SKU 档位配置）

### L1 · 入门级（易耗品 / 盲盒 / 挂件）
- **镭射变色防伪标**：贴于吊牌/盒底，倾斜变色 + 撕毁留「VOID」痕。
- **刮层验证码**：每件唯一 16 位码，刮开后到官方小程序「扫码验真」，一码一验、二次查询提示「已验证 N 次」。

### L2 · 进阶级（桌面摆件标准款）
- **UV 隐形荧光暗纹**：底座/主体隐蔽处印**专属神兽灵纹**，紫外灯（365nm）下显形，肉眼不可见。
- **底部微缩文字**：激光雕刻「YUSHAN · SHANHAI · SKU编号」微缩字（≤0.3mm），需放大镜辨识。
- **激光编号钢印**：底部唯一序列号，与包装、证书三码合一。

### L3 · 收藏级（GK 限量 / 礼盒）
- **NFC 芯片验真**：内嵌 NFC，手机轻触跳转官方验真页，返回**唯一 UID + 批次 + 限量编号 + 真伪**。
- **数字收藏证书**：每件绑定一张数字证书（官方数据库 / 可选区块链存证），记录编号、发行量、发行日期，可转移登记。
- **限量编号铭牌**：底座金属铭牌「NO. xxx / 限量 yyyy」。

## 二、原创「灵纹」隐藏暗记（全系列彩蛋机制）

- 每只神兽在**固定且隐蔽**的部位藏一处**原创灵纹**（如尾羽内侧、爪垫、底座云纹缝隙），为本项目自创纹样，**仿品难以精确复刻**。
- 暗记图档**不公开**，仅厂内与验真后台留底；各兽具体位置见其文档「防伪暗记」（对外文档只写部位，不公开纹样细节）。

## 三、包装防伪

1. **易碎封口贴**：开封即损，不可完整揭取复用。
2. **专属盒型 + 工艺**：定制盒型 + 烫金 + 压纹 + 专色印刷，提高仿制成本。
3. **隐藏印记**：内盒某处 UV 暗印 + 微缩 logo。
4. **三码合一**：产品钢印号 = 盒身号 = 证书号，缺一即存疑。

## 四、官方验真渠道

- 渠道：官方公众号 / 小程序「山海有灵 · 验真」入口。
- 方式：扫码 / NFC / 输入验证码三选一 → 返回真伪 + 编号 + 批次 + 验证次数。
- 异常提示：异地高频、超量验证自动标记预警。

## 五、设计与知识产权层防伪（治本）

1. **原创纹样库**：系列专属纹样（灵纹、祥云变体、神兽专属符号）做**版权登记**。
2. **外观专利**：对关键造型/底座结构申请**外观设计专利**，为维权留据。
3. **参数保密**：完整色号、比例、模具参数为内部机密，公开文档仅给方向值。
4. **商标**：「山海有灵」及 logo 注册商标，包装与验真页统一使用。

## 六、合规要点（重要）

- 防伪与验真**不得**附带任何功效宣称（招财/辟邪/改运等），仅作真伪与收藏信息。
- NFC / 小程序若收集用户信息，遵循《个人信息保护法》：**最小必要、告知同意、可不注册即验真**（验真不强制登录）。
- 数字证书若用区块链存证，注意合规与数据留存要求；不得宣传为「金融/投资/升值」属性。
- 镭射标、NFC 等供应商须有资质；标签符合产品标识要求。

## 七、成本与配置建议

| 档位 | 防伪配置 | 单件增量成本（估） |
|---|---|---|
| 挂件/盲盒 | L1（镭射标+刮码） | ¥0.5–1.5 |
| 摆件标准款 | L1 + L2（UV暗纹+微缩+钢印） | ¥2–5 |
| GK 收藏款 | L1+L2+L3（NFC+证书+铭牌） | ¥8–20 |

> 原则：**档位越高、防伪越重**，把验真成本投到溢价高、最易被仿的收藏款上。
