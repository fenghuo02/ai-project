## SH-D07　凤凰 Fènghuáng
- **出处**：《南次三经》丹穴之山。
- **介绍**：状如鸡，五彩而有文，首文曰「德」，翼文曰「义」，背文曰「礼」，膺文曰「仁」，腹文曰「信」。百鸟之王，自歌自舞。
- **小故事**：丹穴山顶，凤凰每展翅，五色羽便把朝霞接到人间。山民说，听见它清亮的歌声，就知道该播种了——它从不报凶，只把好时节衔来。
- **萌化设计**：Q 版五彩长尾鸟，头顶小冠羽，尾羽分五色绶带（红黄青白黑→国风化为 朱红/鎏金/黛蓝/月白/黛绿），翅尖描金。
- **配色**：朱红 #C8313B + 鎏金 #C9A24B + 黛蓝 #2E4A6B + 月白 #F2EFE6。
- **核心提示词**：`chibi five-color phoenix "Fenghuang", small crown crest, long flowing five-ribbon tail (vermilion/gold/indigo/white/green), gold-tipped wings, graceful happy pose`
- **尺寸与参数**：摆件整高 120 mm（含长尾上扬）；**特殊件**：长尾羽内置钢丝定型，分件卡扣；挂件尾羽改为缩短软绒款（整高 95 mm）。
- **防伪暗记**：尾羽第 3 根羽轴内侧 UV 灵纹。

## SH-D08　麒麟 Qílín
- **出处**：传世瑞兽，与《山海经》谱系并称。
- **介绍**：麋身、牛尾、龙鳞、独角（角端有肉），仁兽，步不践生草、不履生虫。
- **小故事**：麒麟踏过的草地，连蚂蚁都不惊。老人讲，它走夜路也含着一点温光，给迷途的人照亮回家的田埂——但它从不许愿，只是静静走过。
- **萌化设计**：Q 版独角小神兽，龙鳞 + 鬃毛卷云，蹄如祥云软垫，额前独角描金，颈披小铃。
- **配色**：黛绿 #3A5A40 + 鎏金 #C9A24B + 月白 #F2EFE6 + 朱红点缀。
- **核心提示词**：`chibi qilin (kirin), single gold horn with rounded tip, dragon-scale body, swirling cloud mane, soft hoof pads, tiny neck bell, gentle benevolent face`
- **尺寸与参数**：摆件整高 110 mm；鳞片浮雕 + 局部鎏金描线；挂件鬃毛用卷绒工艺。
- **防伪暗记**：左前蹄垫底 UV 灵纹。

## SH-D09　应龙 Yìnglóng
- **出处**：《大荒东经》《大荒北经》——有翼之龙，助黄帝、大禹。
- **介绍**：背生双翼的神龙，能蓄水行雨；传说曾以尾画地成江河，助禹疏水。
- **小故事**：大水那年，应龙用尾巴在泥地上划了一道又一道，水便顺着痕迹退回了海。它太累，伏在山后睡着了，鳞上还沾着没干的河泥。
- **萌化设计**：Q 版短身翼龙，圆头大眼，背一对小羽翼，鳞片描金，爪抱一颗水珠。
- **配色**：黛蓝 #2E4A6B + 鎏金 #C9A24B + 月白 #F2EFE6 + 朱红须。
- **核心提示词**：`chibi winged dragon "Yinglong", short round body, big eyes, a pair of small feathered wings, gold-edged blue scales, holding a water droplet, swirling cloud`
- **尺寸与参数**：摆件整高 115 mm；**特殊件**：双翼分件卡扣 + 钢丝；水珠为半透明树脂件。
- **防伪暗记**：右翼根部内侧 UV 灵纹。

## SH-D10　重明鸟 Chóngmíngniǎo
- **出处**：志怪谱系神鸟（双瞳）。
- **介绍**：状如鸡而双瞳，鸣声如凤；古人喜其守门的传说（本系列仅作**文化守护**叙事，不作辟邪功效）。
- **小故事**：重明每只眼里都住着一点光。它停在门楣上打盹，夜里有人提灯回来，它便睁开四只眼，像四盏小灯，陪人把门闩好。
- **萌化设计**：Q 版圆胖鸟，双瞳大眼（每眼两点高光），鸡冠化为祥云冠，翅描金。
- **配色**：朱红 #C8313B + 鎏金 #C9A24B + 月白 #F2EFE6。
- **核心提示词**：`chibi double-pupil bird "Chongming", plump round body, big eyes each with two highlights, cloud-shaped crest, gold-edged wings, cheerful`
- **尺寸与参数**：摆件整高 100 mm；挂件圆球绒身款 90 mm。
- **防伪暗记**：尾下绒缝 UV 灵纹。

## SH-D11　鹿蜀 Lùshǔ
- **出处**：《南山经》杻阳之山。
- **介绍**：状如马，白首，身有虎纹，尾赤红，鸣声如人唱谣。
- **小故事**：鹿蜀跑过山脊时，虎纹一闪一闪，像把山风谱成了曲。樵夫说，跟着它的歌声走，总能绕开陡坡，慢慢就到了家。
- **萌化设计**：Q 版小马，白头 + 橙黄身虎纹 + 红尾，蹄轻巧，张口似在哼歌（音符可点缀但非文字）。
- **配色**：月白 #F2EFE6 头 + 橙黄虎纹 + 朱红 #C8313B 尾 + 鎏金描边。
- **核心提示词**：`chibi horse-beast "Lushu", white head, orange-yellow body with tiger stripes, bright red tail, light hooves, singing happy expression`
- **尺寸与参数**：摆件整高 105 mm；红尾内置软钢丝可微调；挂件虎纹烫印。
- **防伪暗记**：红尾尾根 UV 灵纹。

## SH-D12　陆吾 Lùwú
- **出处**：《西次三经》昆仑之丘——「司天之九部及帝之囿时」。
- **介绍**：虎身而九尾，人面而虎爪，昆仑山的守护神。
- **小故事**：陆吾守着昆仑的九道天阶，九条尾巴各搭一阶。它认得每一位上山求学的人，尾巴一摆，便替他们扫开石上的霜。
- **萌化设计**：Q 版虎身（萌化人面→拟人小圆脸），九条短尾呈扇形，虎纹 + 鎏金甲片，威而不凶。
- **配色**：鎏金 #C9A24B + 黛蓝 #2E4A6B 甲 + 月白 #F2EFE6 + 玄黑虎纹。
- **核心提示词**：`chibi guardian "Luwu", tiger body with nine short fan-tails, gentle round humanlike face, gold-edged armor scales, tiger stripes, dignified but cute`
- **尺寸与参数**：摆件整高 115 mm；九尾分层定型；挂件九尾简化为绒尾束。
- **防伪暗记**：第 5 条尾内侧 UV 灵纹。

## SH-D13　英招 Yīngzhāo
- **出处**：《西次三经》槐江之山——「司帝之平圃」。
- **介绍**：马身而人面，虎纹而鸟翼，徇于四海，声如榴。
- **小故事**：英招巡四海，替天帝照看一座花圃。它飞过哪里，就把哪里的花消息记下：东海的桃开了，西山的雪化了——回来一一禀报，从不夸大。
- **萌化设计**：Q 版小马身 + 圆脸（拟人）+ 虎纹 + 一对小鸟翼，飞奔姿。
- **配色**：橙黄虎纹 + 黛蓝翼 + 鎏金 + 月白。
- **核心提示词**：`chibi "Yingzhao", horse body, gentle round humanlike face, tiger stripes, a pair of small bird wings, dynamic flying pose`
- **尺寸与参数**：摆件整高 115 mm（动态飞奔，重心配重底座）；**特殊件**：翼+腾空姿需加大底座支点。
- **防伪暗记**：右后蹄垫 UV 灵纹。

## SH-D14　开明兽 Kāimíngshòu
- **出处**：《海内西经》昆仑之墟——「身大类虎而九首，皆人面，东向立昆仑上」。
- **介绍**：虎身九首（九张面孔），昆仑九门的守门神兽。
- **小故事**：开明有九张脸，朝着九个方向。白天它们轮流值守，夜里便一齐闭眼，只留最中间那张微微睁着——昆仑的门，一夜都没真正合过。
- **萌化设计**：Q 版圆虎身 + 主头放大 + 周围八个迷你小头（萌缩），九头不恐怖、呈环抱状。
- **配色**：鎏金 #C9A24B + 朱红 #C8313B + 月白 + 玄黑勾边。
- **核心提示词**：`chibi nine-headed guardian "Kaiming", round tiger body, one big central cute head plus eight tiny mini-heads arranged around, gold + vermilion, friendly not scary`
- **尺寸与参数**：摆件整高 110 mm（横向较宽 Ø100 底座）；八小头分件精涂。
- **防伪暗记**：背部中央小头之间 UV 灵纹。

## SH-D15　乘黄 Chénghuáng
- **出处**：《海外西经》——「其状如狐，其背上有角」。
- **介绍**：形如狐，背生双角，传为祥瑞之马（神话称乘之可寿，本系列仅作**文化叙事**，不作功效）。
- **小故事**：乘黄背上的两只小角，会接住落下的星子。它驮着星光在草原上慢慢走，把夜走成了一条亮亮的路。
- **萌化设计**：Q 版狐马，蓬尾，背上两只小鹿角描金，眼含星点。
- **配色**：暮粉 #E9B7B0 + 鎏金 #C9A24B + 月白 + 黛蓝点。
- **核心提示词**：`chibi fox-horse "Chenghuang", fluffy tail, two small gold antlers on the back, starry eyes, soft gentle pose`
- **尺寸与参数**：摆件整高 108 mm；背角分件；挂件蓬尾卷绒。
- **防伪暗记**：背角根部 UV 灵纹。

## SH-D16　駮 Bó
- **出处**：《西次四经》中曲之山——「状如马而白身黑尾，一角，虎牙爪，音如鼓，食虎豹」。
- **介绍**：白身黑尾独角，虎牙虎爪，能食虎豹的猛马（本系列走「英气护卫」风，不作辟邪功效）。
- **小故事**：駮立在牧场边的高坡上，虎豹远远见它的独角，就掉头走了。牧人夜里睡得安稳，只当是风把猛兽吹散了——其实是駮一夜没合眼。
- **萌化设计**：Q 版白马 + 黑尾 + 额前独角 + 小虎牙（圆钝），英气挺立。
- **配色**：月白 #F2EFE6 + 玄黑 #1C1C22 尾 + 鎏金角 + 朱红蹄。
- **核心提示词**：`chibi white horse-beast "Bo", white body, black tail, single gold horn, small rounded tiger fangs, brave upright pose`
- **尺寸与参数**：摆件整高 110 mm；独角描金分件；挂件黑尾流苏化。
- **防伪暗记**：独角基座 UV 灵纹。

## SH-D17　鸾鸟 Luánniǎo
- **出处**：《西山经》女床之山——「有鸟焉，其状如翟而五采文，名曰鸾鸟」。
- **介绍**：凤之亲族，五彩有文，鸣声和谐（神话云见则天下安宁，本系列仅作**文化寓意**叙事）。
- **小故事**：鸾鸟成对而飞，一只起调，一只接和，山谷里便有了回声。人们循声搭起凉亭，听它们把整座春天唱完。
- **萌化设计**：Q 版小型彩凤，比凤凰更圆更萌，短尾五彩，成对设计可出「鸾凤」CP 款。
- **配色**：暮粉 #E9B7B0 + 朱红 + 鎏金 + 黛蓝。
- **核心提示词**：`chibi "Luan" bird, rounder cuter mini-phoenix, short five-color tail, sweet singing pose, can be a pair`
- **尺寸与参数**：摆件整高 100 mm；可出双只 CP 套；挂件圆绒身。
- **防伪暗记**：胸前绒缝 UV 灵纹。

## SH-D18　文鳐鱼 Wényáoyú
- **出处**：《西次三经》泰器之山——「鱼身而鸟翼，苍文白首赤喙，常行西海，游于东海，以夜飞」。
- **介绍**：鱼身鸟翼，白首红喙，夜里能飞越东西海。
- **小故事**：每到夜里，文鳐鱼便成群飞起，鳍像帆，翼像风。渔人说，看见海面上一片银光掠过，就知道是它们在替月亮巡海。
- **萌化设计**：Q 版胖鱼 + 一对小翼 + 白头红喙，身覆青纹，悬浮姿（底座做浪花）。
- **配色**：黛蓝 #2E4A6B + 月白 #F2EFE6 首 + 朱红 #C8313B 喙 + 鎏金鳍边。
- **核心提示词**：`chibi flying fish "Wenyao", chubby fish body with a pair of small wings, white head, red beak-mouth, blue patterns, floating over a wave base`
- **尺寸与参数**：摆件整高 100 mm（含浪花底座）；**特殊件**：悬浮姿用透明亚克力支柱；挂件去支柱、加翼。
- **防伪暗记**：右鳍内侧 UV 灵纹。
