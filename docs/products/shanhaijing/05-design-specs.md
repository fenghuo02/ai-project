# 「山海有灵」系列 · 统一设计尺寸与参数标准

> 全系列（神兽谱 + 凶兽谱）通用生产规格。各神兽文档中的「尺寸与参数」只标注**与本标准的差异 / 特殊件**，其余默认遵循本标准。
>
> ⚠️ 所有尺寸为**设计稿目标值**，量产以打样实物为准；公差 ±2mm（毛绒 ±5mm）。Pantone 以国风色卡换算后打样校色。

## 二、品类 A：毛绒小挂件（SH-PLUSH-S）

| 项目 | 参数 |
|---|---|
| 整高（不含挂绳） | 90 mm（标准）；±5 mm |
| 挂绳 + 中国结 | 编绳 +120 mm，含锌合金龙虾扣 + 钥匙环 Ø25 mm |
| 面料 | 短毛绒 / 水晶超柔（5mm 绒）+ 局部烫印/电脑绣花 |
| 填充 | 环保 PP 棉，回弹饱满 |
| 工艺 | 立体缝制 + 绣花眼/鼻 + 烫画纹样 + 流苏 |
| 净重 | 45–70 g |
| 安全 | 玩具安全标准；眼鼻为绣花（无可拆小件）；缝合拉力达标；阻燃面料；标签注「装饰收藏用途/适用年龄」 |
| 吊牌 | 双面：正面 IP 形象 + 编号；背面 含 AI 生成标识 + 防伪二维码（见 06） |

## 三、品类 B：桌面摆件（标准款 SH-FIG-STD）

| 项目 | 参数 |
|---|---|
| 整高（含底座） | 100–120 mm |
| 主体材质 | PVC 搪胶 或 树脂；哑光 UV 涂装 |
| 底座 | 祥云浮雕底座，Ø 70–90 mm，高 12–18 mm |
| 底部 | 防滑硅胶垫 + **激光编号钢印** + UV 隐形暗纹（见 06） |
| 净重 | 150–250 g |
| 涂装 | 哑光为主 + 局部高光/鎏金描线；色号对应色卡 |
| 包装 | 内衬 EVA/纸浆托 + 彩盒（烫金+压纹）+ 易碎封口贴 |

## 四、品类 C：收藏级手办（GK · SH-FIG-GK，限量编号）

| 项目 | 参数 |
|---|---|
| 整高 | 150–220 mm |
| 材质 | Polystone 树脂 / 高品质 PVC，多部件分件涂装 |
| 底座 | 双层祥云/山海地形底座，Ø 100–130 mm，可嵌铭牌 |
| 限量 | 编号钢印 + **NFC 芯片**（一拍验真）+ 数字收藏证书 |
| 净重 | 400–700 g |
| 公差 | ±1.5 mm；分模线打磨处理 |

## 五、品类 D：盲盒（SH-PLUSH-BLIND / SH-FIG-BLIND）

| 项目 | 参数 |
|---|---|
| 单盒 | 标准款挂件或迷你摆件（迷你款整高 60–80 mm） |
| 端盒 | 一端 6 常规 + 概率隐藏款（隐藏比例建议 1/72 或每 2 端 1 隐藏） |
| 盒规 | 单盒 ~70×70×95 mm；防拆封口 |
| 验真 | 每盒附刮层验证码（见 06 · L1） |

## 六、衍生件（SH-DERIV）

| 品类 | 尺寸 | 材质 |
|---|---|---|
| 钥匙扣/包挂 | 主体 40–55 mm + 五金 | 软胶/亚克力 |
| 冰箱贴 | 50–70 mm | 软胶 + 磁片 |
| 金属徽章 | Ø 40–58 mm | 锌合金 + 软珐琅 + 蝴蝶扣 |

## 七、出稿与交付规范（给设计/打样厂）

1. 出稿：三视图（正/侧/背）+ 1:1 比例标注 + 配色 HEX/Pantone + 关键纹样矢量。
2. 特殊结构（翼/尾/角/长身）须标注**内部钢丝定型**或**分件卡扣**位置（见各兽特殊件）。
3. 交付：源文件（AI/PSD/OBJ）+ 打样确认书 + 色卡实样。
4. 所有量产件须预留 06 防伪三要素位置：底部钢印编号、UV 暗纹区、二维码/NFC 位。
