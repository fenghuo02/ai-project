# 凶兽谱（第三弹 · 12 新） · 介绍 / 小故事 / 设计 / 提示词 / 尺寸参数

> 通用风格见 `00-ai-prompts.md`；尺寸见 `05-design-specs.md`；防伪见 `06-anti-counterfeit.md`。
> **凶兽萌酷化原则**：保留「凶」的辨识特征（獠牙/翼/多首/利爪）但走 **酷飒 / 守护 / 反差萌** 风格，**严禁恐怖、血腥、暴力、惊悚**表现；神话「食人/降灾」等情节仅作**文化叙事**，不作任何功效或威吓宣称。本弹**暂不出图**。

## SH-F07　穷奇 Qióngqí（四凶之一）
- **出处**：《西次四经》邽山「状如牛，猬毛，音如獆狗」；《海内北经》「状如虎，有翼」。
- **介绍**：有翼之虎（一说牛形猬毛），四凶之一。本系列取「带翼酷虎」造型。
- **小故事**：穷奇的翼太大，飞起来总把云撞散。后来它学乖了，只在没人的夜里偷偷练飞，把云叠成一座座小山——清晨牧人抬头，还以为昨夜下过一场白雪。
- **萌化设计**：Q 版小虎 + 一对羽翼 + 鬃刺（猬毛软化），酷脸但圆眼，翼描金。
- **配色**：玄黑 #1C1C22 纹 + 鎏金 #C9A24B + 月白 + 朱红眼线。
- **核心提示词**：`chibi winged tiger "Qiongqi", a pair of feathered wings, soft spiky mane, cool but cute round eyes, gold-edged wings, dynamic pose, not scary`
- **尺寸与参数**：摆件整高 115 mm；**特殊件**：翼分件+钢丝；挂件翼软绒化。
- **防伪暗记**：左翼根内侧 UV 灵纹。

## SH-F08　梼杌 Táowù（四凶之一）
- **出处**：《神异经》——「状如虎，毛长，人面虎足，猪口牙」。
- **介绍**：虎身人面、长毛獠牙，四凶之一。本系列取「毛茸獠牙萌」。
- **小故事**：梼杌一身长毛，最怕别人笑它像扫帚。它索性躲进深林，给迷路的小兽当挡风的墙——风雪夜里，那堆"会动的长毛"其实最暖。
- **萌化设计**：Q 版长毛虎身 + 圆脸（拟人化萌）+ 小獠牙（圆钝），蓬松毛流。
- **配色**：暖棕 + 玄黑纹 + 鎏金牙边 + 月白。
- **核心提示词**：`chibi "Taowu", long-haired tiger body, round humanlike cute face, small rounded boar tusks, fluffy fur flow, grumpy-cute expression`
- **尺寸与参数**：摆件整高 110 mm（长毛雕塑感）；挂件长毛绒为主打。
- **防伪暗记**：胸前毛缝 UV 灵纹。

## SH-F09　混沌 Hùndùn（四凶之一）
- **出处**：《山海经·西山经》天山「状如黄囊，赤如丹火，六足四翼，浑敦无面目」。
- **介绍**：无面浑圆、六足四翼的奇兽，四凶之一。本系列主打**无脸憨萌**（最易出爆款）。
- **小故事**：混沌没有脸，所以从不挑食、也从不记仇。它六只脚乱跑，四只翼乱扑，撞翻了谁的篱笆，第二天准用翼把它扇得整整齐齐。
- **萌化设计**：Q 版圆滚滚布袋身（红如丹火），六只小短脚 + 四只小翼，无五官、仅一点腮红，呆萌。
- **配色**：朱红 #C8313B + 鎏金 #C9A24B 描边 + 暮粉腮红。
- **核心提示词**：`chibi faceless "Hundun", round plump bag-like body glowing fire-red, six tiny stubby legs and four little wings, no facial features only blush, derpy adorable`
- **尺寸与参数**：摆件整高 100 mm（圆球造型，重心稳）；六足四翼分件；挂件圆球绒最佳。
- **防伪暗记**：底部足间 UV 灵纹。

## SH-F10　相柳 Xiāngliǔ
- **出处**：《海外北经》《大荒北经》——共工之臣，九首人面蛇身，青色。
- **介绍**：九头蛇身的水怪。本系列取「九头小蛇环抱」萌酷造型。
- **小故事**：相柳九个头常为往哪走吵架：一个想去看海，一个想晒太阳……吵到最后谁也没动，干脆盘成一团睡了，九张脸还嘟着嘴。
- **萌化设计**：Q 版盘卷蛇身 + 九个小头（簇拥成花环状），青鳞描金，表情各异（萌点）。
- **配色**：黛绿 #3A5A40 + 黛蓝 #2E4A6B + 鎏金鳞边 + 月白腹。
- **核心提示词**：`chibi nine-headed serpent "Xiangliu", coiled snake body, nine small cute heads clustered like a ring with different expressions, jade-green gold-edged scales, not scary`
- **尺寸与参数**：摆件整高 95 mm（盘卷低矮、Ø95 底座）；九头分件精涂；挂件简化为三头团。
- **防伪暗记**：盘身最内圈鳞缝 UV 灵纹。

## SH-F11　九婴 Jiǔyīng
- **出处**：《淮南子》——尧时水火之怪，后羿所诛；与《山海经》谱系并叙。
- **介绍**：九头、能喷水吐火的怪兽。本系列取「九头小团 + 水火双色」反差萌。
- **小故事**：九婴左边几个头喷水，右边几个头吐火，自己跟自己打成平手，常年腾着一团温乎乎的雾。小动物们都爱挤过去取暖——又不会太烫。
- **萌化设计**：Q 版团身 + 九小头（一半水蓝、一半火红），喷水/小火苗作软装饰。
- **配色**：黛蓝 #2E4A6B 水 + 朱红 #C8313B 火 + 鎏金 + 月白。
- **核心提示词**：`chibi nine-headed beast "Jiuying", round cluster body, nine tiny heads half water-blue half fire-red, gentle water/flame motifs, playful not scary`
- **尺寸与参数**：摆件整高 100 mm；水/火装饰为半透明树脂件；挂件双色拼绒。
- **防伪暗记**：底座云隙 UV 灵纹。

## SH-F12　化蛇 Huàshé
- **出处**：《中次二经》阳水——「人面而豺身，鸟翼而蛇行，其音如叱呼」。
- **介绍**：人面豺身、鸟翼蛇行的水兆之兽。本系列取「灵巧小水蛇」萌化（仅文化叙事）。
- **小故事**：化蛇游过的水面会先起一圈小涟漪。村里的孩子学会了看它——涟漪一圈圈来，就赶紧把晒的谷子收进屋，果然没多久就落了雨。
- **萌化设计**：Q 版圆脸（拟人）蛇身 + 一对小翼，鳞泛水光，灵动 S 形。
- **配色**：黛蓝 #2E4A6B + 月白 + 鎏金 + 暮粉脸。
- **核心提示词**：`chibi "Huashe", cute round humanlike face, slender serpent body with a pair of small wings, water-sheen scales, lively S-curve, gentle`
- **尺寸与参数**：摆件整高 100 mm（S 形需配重底座）；挂件缩短身、加翼。
- **防伪暗记**：尾尖内侧 UV 灵纹。

## SH-F13　蛊雕 Gǔdiāo
- **出处**：《南次二经》鹿吴之山——「状如雕而有角，其音如婴儿之音」。
- **介绍**：生角的雕，叫声似婴啼。本系列取「独角酷雕」造型。
- **小故事**：蛊雕的叫声像小孩，常把上山的人引得到处找。其实它只是孤单，想有人陪它站在崖顶看云——找到它的人，往往也舍不得走了。
- **萌化设计**：Q 版猛禽 + 头顶小独角 + 利爪（圆钝），翅展威风，眼神酷。
- **配色**：玄黑 #1C1C22 + 鎏金角 #C9A24B + 朱红喙 + 月白纹。
- **核心提示词**：`chibi horned eagle "Gudiao", a small single horn on head, spread wings, rounded talons, cool gaze, black + gold + red beak`
- **尺寸与参数**：摆件整高 115 mm（展翅，配重底座）；翼分件+钢丝；挂件收翼圆身。
- **防伪暗记**：右爪垫 UV 灵纹。

## SH-F14　窫窳 Yàyǔ
- **出处**：《海内西经》《北山经》——本为天神，被杀复活后化为龙首兽身的食人怪。
- **介绍**：龙首兽身（一说蛇身），身世悲情。本系列取「忧郁系酷萌」，讲**和解**主题。
- **小故事**：窫窳总记得自己原来的模样，所以爱对着水面发呆。有一天水里映出一只朝它笑的小兽，它愣了很久——原来那就是现在的、也很好的自己。
- **萌化设计**：Q 版龙首 + 圆兽身，眼神温柔带点忧郁，颈系一条月白小巾（和解意象）。
- **配色**：黛蓝 #2E4A6B + 月白 #F2EFE6 + 鎏金 + 暮粉。
- **核心提示词**：`chibi "Yayu", gentle dragon head on a round beast body, soft slightly melancholic eyes, a moon-white scarf around neck, calm cute, not scary`
- **尺寸与参数**：摆件整高 108 mm；颈巾为软质件；挂件绒身款。
- **防伪暗记**：颈巾内侧 UV 灵纹。

## SH-F15　钦原 Qīnyuán
- **出处**：《西次三经》昆仑——「状如蜂，大如鸳鸯，螫鸟兽则死，螫木则枯」。
- **介绍**：似蜂的巨虫。本系列取「呆萌小毒蜂」造型（毒性仅文化叙事，不作任何功效/威吓）。
- **小故事**：钦原其实很怕扎到朋友，所以总把小刺收得好好的。它整天忙着给花传粉，落在谁的窗台，谁家的花第二年就开得格外热闹。
- **萌化设计**：Q 版圆胖蜂，绒绒身，小翅嗡嗡，尾刺圆钝藏起，抱一朵小花。
- **配色**：鎏金 #C9A24B + 玄黑条 + 月白翅 + 朱红花。
- **核心提示词**：`chibi giant bee "Qinyuan", round fuzzy body, tiny buzzing wings, blunt hidden stinger, holding a little flower, gold-black stripes, adorable`
- **尺寸与参数**：摆件整高 90 mm（圆胖矮）；透明小翅亚克力；挂件圆绒球最佳。
- **防伪暗记**：腹底条纹缝 UV 灵纹。

## SH-F16　凿齿 Záochǐ
- **出处**：《海外南经》《大荒南经》——「齿长五六尺」，持盾执戈，被后羿射杀。
- **介绍**：长牙的兽人战士，执干戈。本系列取「Q 版小战士」造型。
- **小故事**：凿齿的牙太长，吃饭总是最后一个。它干脆改去守桥，用那对长牙替村里挑水、扛木头——后来谁也不嫌它的牙长了，反倒抢着请它帮忙。
- **萌化设计**：Q 版兽人 + 一对长牙（圆钝可爱）+ 迷你小盾小戈（玩具化），憨勇。
- **配色**：黛绿 #3A5A40 + 鎏金 #C9A24B + 月白牙 + 朱红甲带。
- **核心提示词**：`chibi beast-warrior "Zaochi", a pair of long rounded cute tusks, holding a tiny toy shield and dagger-axe, brave derpy pose, not violent`
- **尺寸与参数**：摆件整高 105 mm；盾/戈为分件配饰（无尖锐，符合玩具安全）；挂件去戈留盾。
- **防伪暗记**：盾背内侧 UV 灵纹。

## SH-F17　修蛇 Xiūshé（巴蛇）
- **出处**：《海内南经》——「巴蛇食象，三岁而出其骨」。
- **介绍**：能吞象的巨蟒。本系列取「盘成一团的憨胖大蛇」（吞象仅作**趣味叙事**）。
- **小故事**：修蛇吞过一头象，鼓了三年才消下去。那三年它哪也去不了，只能盘在山谷晒太阳，听小鸟在它背上做窝——它倒觉得，这样的慢日子也挺好。
- **萌化设计**：Q 版超粗圆蟒，盘成蚊香状，圆眼吐小信子，肚子微鼓（彩蛋：隐约象形纹）。
- **配色**：黛绿 #3A5A40 + 鎏金鳞边 + 月白腹 + 朱红信子。
- **核心提示词**：`chibi giant python "Xiushe", very chubby coiled body (spiral), round eyes, tiny tongue, slightly bulging belly, jade-gold scales, lazy cute`
- **尺寸与参数**：摆件整高 80 mm（盘卷矮胖、Ø100 底座）；挂件盘团绒球。
- **防伪暗记**：盘心鳞缝 UV 灵纹（彩蛋：内藏微缩象纹）。

## SH-F18　諸犍 Zhūjiān
- **出处**：《北山经》单张之山——「状如豹而长尾，人首而牛耳，一目」。
- **介绍**：豹身长尾、人首牛耳、独目的奇兽。本系列取「傲娇独眼酷豹」。
- **小故事**：諸犍只有一只眼，看人却最准。它把长尾叼在嘴里假装高冷，其实是怕尾巴扫到别人——独眼一眨，算是它打招呼的方式。
- **萌化设计**：Q 版豹身 + 圆脸（拟人）+ 牛耳 + 一只大眼（独目萌化）+ 超长尾叼口中。
- **配色**：鎏金 #C9A24B 豹纹 + 玄黑斑 + 月白 + 黛蓝眼。
- **核心提示词**：`chibi one-eyed leopard "Zhujian", leopard body, round humanlike face, ox ears, a single big cute eye, very long tail held in mouth, tsundere cool pose`
- **尺寸与参数**：摆件整高 105 mm；长尾内置软钢丝可造型；挂件长尾流苏化。
- **防伪暗记**：长尾尾梢内侧 UV 灵纹。
