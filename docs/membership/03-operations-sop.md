## 二、SOP-01 新会员开通

```
客户付款确认 → 财务登记到账 → 客服建卡
    → 发权益卡+授权书 → 客户签字
    → 交付工单（数字/实体/拍摄分支）
    → 3日内首触欢迎消息
```

**检查清单**  
- [ ] 等级与实付金额匹配  
- [ ] A1 来源已标记（影响分成，不改变权益）  
- [ ] 授权书已归档  
- [ ] 工单已指定交付专员  
- [ ] 禁忌：未承诺「改运」「包满意算命」

## 三、SOP-02 年度壁纸批量发放（L3/L4/L5）

| 步骤 | 责任人 | 时点 |
|------|--------|------|
| M-3 日 | 交付专员 | 完成当季 6 张图质检 |
| M-1 日 | 客服 | 预览链接内部审核 |
| 节气当日 09:00 | 客服 | 推送下载 + 触点文案 |
| M+1 日 | 客服 | 未下载提醒 1 次 |

**异常**：漏发 → 4h 内补发 + 记录「交付异常」表。

## 四、SOP-03 升档（L3→L4、L4→L5）

1. 客服核算补差价并收款（对公）。  
2. 更新 `level` 与 `level_expire`（到期日不变）。  
3. 补开交付工单：  
   - L3→L4：补 38 张壁纸 + 色卡升级 + 拍摄预约  
   - L4→L5：启动纪念册 + 副拍预约 + 文化答疑 2 次额度  
4. 24h 内发送《升档权益补充说明》。

## 五、SOP-04 续费

| 节点 | 动作 |
|------|------|
| T-30 | 自动生成《年度回顾》数据 |
| T-30 | 客服人工发送 + 确认客户意向 |
| T-7 | 发送折扣支付链接 |
| 到账当日 | 续费成功消息 + 更新 expire |
| 未续费 T+60 | 系统自动标记降级「知时」 |

**续费话术要点**：强调「又一年的节气美学陪伴」「纪念册可续章」，不提「明年运势」。

## 六、SOP-05 投诉与升级

```
客户投诉 → 客服首响（按等级 SLA）
    → 可解决：补偿方案（§02 第四节）
    → 不可解决：24h 内升级主理人
    → 重大：登记《合规事件表》+ 律师备查
```

| 等级 | 典型投诉 | 标准处理 |
|------|----------|----------|
| 一般 | 物流慢 | 查单 + 加急 |
| 一般 | 壁纸不喜欢 | 1 次换色重出（同节气） |
| 严重 | 要求加命理内容 | **拒绝** + 重申服务边界 + 可提供退费协商（未交付部分） |
| 严重 | 怀疑虚假宣传 | 出示授权、AIGC 标识、合同 |

## 七、SOP-06 B 端伙伴 onboarding

1. BD 签意向 → 法务审合同模板（P1/P2/P3）。  
2. 定金到账 → 样品册寄送 → 话术培训 1h。  
3. 体验日执行 checklist（三方交接见商业计划书）。  
4. 验收单签字 → 开票 → 录入 `partner_level`。  
5. 机构 VIP 个人 L3 体验码批量生成（限 10）。

## 八、与老师协作 SOP

| 触发 | 老师动作 | 时限 | 交付物 |
|------|----------|------|--------|
| L5 文化答疑预约 | 微信语音 20min | 客户预约后 5 日内 | 答疑纪要（无命理记录） |
| 知识审稿 | 审核节气短文 | 3 日 | 批注 PDF |
| B 端交接 | 现场见证 + 钤印 | 按场次 | 照片存档 |
| 色彩坐标（内部） | 美学用色建议 | 2 日 | **内部**工单，不对客户展示「八字断语」 |

**顾问费**：按《文化顾问服务协议》月结，**不与会员费直接挂钩提成**（避免隐名股东风险，见商业计划书 §18.4）。

## 九、台账与周报

- 新增会员各等级人数  
- 即将到期 30 天内名单

**主理人月报**  
- 续费率、升档率、ARPU  
- L4/L5 占比  
- B 端伙伴 pipeline  
- 合规违规 0 确认

## 十、首期人力配置建议（M1–M6）

| 角色 | 人数 | 说明 |
|------|------|------|
| 主理人 | 1 | 兼交付 + BD |
