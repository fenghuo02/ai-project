# 四时印 · 配套服务体系

> 与 `01-membership-tiers.md` 等级一一对应。服务承诺均围绕**美学交付与体验**，不构成任何命理、医疗、功效结果承诺。

---

## 一、服务体系总图

```
  获客入会          交付履约              留存续费
     │                 │                    │
  注册/升档 ──→  需求确认·授权 ──→  节气触点·满意度
     │                 │                    │
  权益说明         生产·质检·物流          升档·续费·转介绍
     │                 │                    │
  合规确认         售后·档案·验真          会员复盘（季）
```

**服务角色**

| 角色 | 职责 | 不负责 |
|------|------|--------|
| **会员客服** | 咨询、订单、进度、投诉一级处理 | 八字推导、定价审批 |
| **交付专员** | 出图、排版、印厂对接、物流跟踪 | 对外承诺交付日期（需客服确认） |
| **主理人** | L5 纪念册校对会、异常赔付、B 端谈判 | 日常 9.9 订单 |
| **文化顾问（老师）** | L5 文化答疑、B 端交接见证、知识审稿 | 会员销售、合同签署、售后退款 |
| **B 公司客服** | 文创订单、退换货、验真补发 | 摄影档期 |

## 二、全旅程服务包（按阶段）

### 2.1 入会期（T+0 ~ T+3 日）

| 步骤 | 动作 | 标准 | 产出物 |
|------|------|------|--------|
| 1 | 身份识别 | 查询是否老客 / A1 渠道 | CRM 标签 |
| 2 | 权益说明 | 发送《四时印权益卡》PDF（无命理表述） | 客户确认截图 |
| 3 | 授权采集 | 肖像 + 个人信息授权书（SKU-201 同款） | 签字/电子签 |
| 4 | 色彩坐标 | 收集美学用色所需信息（**不说「算命」**） | 内部工单 |
| 5 | 开卡 | 生成会员编号 `YS-年份-序号` | 欢迎消息模板 |

**欢迎消息模板（示例）**  
> 尊敬的「四时印 · {等级}」会员 {姓名}，您的编号是 {编号}。我们将按二十四节气为您更新美学壁纸与色彩建议。本服务仅为东方美学创作与纪念，不含命理推算。如有疑问回复「客服」。

### 2.2 交付期（按产品类型）

#### A. 数字交付（壁纸 / 画像 / 纪念册 PDF）

| 等级 | SLA（自授权齐全起） | 质检 |
|------|---------------------|------|
| L1–L2 单张/八门 | 3 个工作日 | 实拍铁律抽检 + AIGC 标注 |
| L3 年度 24 张 | 首包 7 日；之后每节气 **提前 3 日** 发布 | 批量水印 + 归档 |
| L4 年度 62 张 | 首包 10 日；月度包 5 日 | 同上 |
| L5 纪念册 PDF | 实拍齐套后 **15 个工作日** | `generate-jiniance.js` 出稿 + 人工校对 |

#### B. 实体交付（色卡 / 文创 / 纪念册精装）

| 品类 | SLA | 物流 |
|------|-----|------|
| 档 A/B 色卡 | 7–10 个工作日 | 顺丰标快 |
| 档 C 色卡 | 20–25 个工作日（矿物工艺） | 顺丰保价 |
| SKU-203 纪念册 | 定稿后 15–20 个工作日 | 加固包装 |
| B 公司文创 | 预售款按详情页；现货 48h 发货 | 见 SKU 页 |

#### C. 摄影服务

| 项目 | SLA | 备注 |
|------|-----|------|
| 档期确认 | 下单后 5 个工作日内给 3 个档期 | L4/L5 优先 |
| 拍摄完成 | 档期当日 | 合同含天气备选 1 次 |
| 精修交付 | 拍摄后 7 个工作日 | 15 张起 |
| 入册 | 精修确认后启动纪念册排版 | L5 含 |

### 2.3 触点期（节气服务日历）

> **触点** = 主动服务动作（消息 / 小礼品 / 内容包），非命理提醒。

| 节气节点 | L3 | L4 | L5 |
|----------|----|----|-----|
| 立春、夏至、立秋、冬至 | ✅ 四触点 | ✅ 含于八触点 | ✅ 含于十二触点 |
| 其余 20 节气 | — | ✅ 壁纸推送 + 1 句色彩建议 | ✅ + 可选语音问候（预录，非直播） |
| 春节 / 中秋 | 节日壁纸包 | 加赠实体贺卡（成本 ¥8 内） | 加赠纪念册章节更新（数字） |

**单次触点标准内容**  
1. 节气名称 + 美学意象 1 段（50–80 字）  
2. 壁纸下载链接 / 小程序卡片  
3. 「本月穿搭色」3 色 HEX（来自客户色彩坐标）  
4. 客服值班电话（L5 24h 微信响应）

## 三、分级客服标准

| 指标 | L0–L1 | L2–L3 | L4–L5 | L6 尊藏 |
|------|-------|-------|-------|---------|
| 响应时效 | 48h | 48h | 24h | 2h（工作时间） |
| 渠道 | 微信 / 小程序 | 同上 | 微信优先 + 电话备用 | 专属管家微信 |
| 投诉首响 | 24h | 12h | 4h | 2h |
| 升级主理人 | 否 | 客诉 2 次 | 1 次即升 | 默认主理人知晓 |

**标准禁用回复**  
- ❌「您今年运势不好建议升级」  
- ❌「这个颜色能招财」  
- ✅「这款朱砂色来自节气美学体系，适合作为您春季写真造型参考」

## 四、售后与补偿政策

| 问题类型 | 处理 | 补偿上限 |
|----------|------|----------|
| 数字文件损坏 / 漏发 | 24h 内重发 | — |
| 实体运输破损 | 凭开箱视频 7 日内换货 | 运费我方承担 |
| 色差争议（色卡） | 复寄 1 张对照色 + 说明矿物手工差异 | 不复做全套 |
| 拍摄不满意 | 合同约定范围内 **1 次** 补拍（L4/L5） | 不含妆造二次费 |
| 纪念册印装瑕疵 | 印厂重印 | 单册成本内 |
| 重大延误（超 SLA 50%） | 主理人裁定 | 时令印 200 或文创券 ¥50 |

> **不退款情形**：定稿签字后印装、已交付 ≥50% 年度壁纸、老师顾问费已发生。
