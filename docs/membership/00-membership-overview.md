## 一、设计目标

| 目标 | 说明 |
|------|------|
| **统一入口** | 把分散的「单次 SKU + 年度订阅 + 文创加购 + B 端采购」收敛为可理解的**四时会员**阶梯，降低客户选择成本 |
| **拉高 LTV** | 用「入会 → 节气触达 → 续费 / 升档 → 文创复购 → 纪念册加印」形成 12 个月生命周期 |
| **对齐合规** | 会员权益只围绕**美学创作、纪念、收藏、服务体验**；严禁运势、改运、占卜类「会员特权」表述 |
| **主体清晰** | 摄影 / 数字交付 / 会员运营 → **A 公司**；文创实物 / 电商 → **B 公司**；老师 → **独立文化顾问**（不参与会员销售主体） |
| **可落地** | 首期（M1–M6）可用**微信私域 + 表格台账**跑通；M4+ 再上小程序会员中心（与验真小程序共用账号体系） |

## 二、体系命名：「四时印」会员

> **四时印**：以二十四节气为时间刻度，以「印」为会员身份凭证（数字勋章 + 实体收藏编号同源）。

```
                    ┌─────────────────────────────────────┐
                    │         四时印 · 会员体系              │
                    ├─────────────────────────────────────┤
   C 端（个人）      │  游霞 → 初逢 → 知时 → 四时序/全/典藏  │
                    │              ↘ 山海·尊藏（单次顶配）   │
   B 端（机构）      │  云鉴伙伴 → 锦匣伙伴 → 定制框架协议      │
   内部             │  客服 / 交付 / 文化顾问 服务标准（§02）   │
                    └─────────────────────────────────────┘
```

## 三、会员等级一览（C 端）

| 等级 | 代号 | 获得方式 | 年费/门槛 | 对应原 SKU / 产品 |
|------|------|----------|-----------|-------------------|
| 游霞 | L0 | 注册私域 / 小程序 | 免费 | 引流、验真、社群 |
| 初逢 | L1 | 任意消费 ≥¥99 或购买入门档 | — | SKU 101–103、入门钩子 |
| 知时 | L2 | 八门款成交或 12 月累计 ≥¥800 | — | 八门 ¥299–499、SKU 201 |
| 四时·序 | L3 | 年度订阅 ¥1,999 | ¥1,999/年 | 节庆订阅 + 档 C 简版色卡 |
| 四时·全 | L4 | 年度订阅 ¥2,999 | ¥2,999/年 | 全套订阅 + 档 C 完整色卡 |
| 四时·典藏 | L5 | 年度订阅 ¥3,999 | ¥3,999/年 | 高定订阅 + 美学纪念册数字版 |
| 山海·尊藏 | L6 | 单次高端 / 收藏级 | ¥6,800–30,000+ | SKU 301–304、B 端礼盒 |

> **升档原则**：只升不降（当期年度内）；跨年可按新档续费，未续费降回「知时」或「初逢」（保留历史编号档案）。

## 五、B 端伙伴等级（简表）

| 等级 | 名称 | 门槛 | 核心权益 |
|------|------|------|----------|
| P1 | **云鉴伙伴** | 单场体验日 / 首单礼盒 ≤¥30,000 | 样品册、联合话术、客户经理协同交付 |
| P2 | **锦匣伙伴** | 年度框架 / 累计采购 ≥¥100,000 | 专属 SKU、批量编号、季度复盘会 |
| P3 | **定制框架** | 单笔 ≥¥198,000 或年框 ≥¥500,000 | SKU-303 定制、缂丝/限量档 D、联合品牌（需法务） |

## 六、「时令印」成长值（可选启用 · M4+）

> 不与「算命积分」挂钩，仅作**消费与互动**的兑换凭证。

| 行为 | 时令印 |
|------|--------|
| 注册并完成手机号验证 | +10 |
| 单笔消费每 ¥10 | +1 |
| 年度订阅续费 | +200 |
| 节气壁纸打卡（每月 1 次） | +5 |
| 合规晒单（带话题 #四时印） | +20（审核） |
| 推荐新客首单成交 | +50 |

**兑换示例**（B 公司实物）：300 印 → 五行书签；800 印 → 明信片套装；2000 印 → ¥100 文创券（不与年度订阅叠加超过 15% 折扣）。

## 八、合规红线（会员专用）

1. 会员卡 / 权益说明**不得**出现：改运、招财、辟邪、旺夫、化解、流年吉凶等表述。  
2. 「八字 / 生辰」在会员文案中统一为**「个人色彩坐标 · 美学用色」**。  
3. 含 AI 的会员赠品须在领取页勾选「已知晓 AIGC 标识」。  
4. 会员价宣传遵守广告法，禁止「最低价」「唯一」等绝对化用语（除非可举证）。  
5. 老师仅以「文化顾问」署名出现在**知识审稿 / 直播答疑（美学向）**，不出现在会员合同签约页。
