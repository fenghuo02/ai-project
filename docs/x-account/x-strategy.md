## 一、账号定位

### 一句话人设

> **Musk's empire, decoded daily.**
> tesla / spacex / xai / neuralink / x / boring co + the AI race elon's running.

### 分析师人设三要素

1. **每日跟踪**（daily）：每天有 1-3 条短推 + 5-10 条 reply，让算法和受众形成"打开 X 就能看到"的预期
2. **6 个公司全覆盖**：不偏科，6 个公司轮换，但权重按当前热度调（一般 Tesla + xAI + SpaceX 占 70%）
3. **AI 是横切轴**：所有公司都用 AI 视角再切一刀（Tesla = FSD/Optimus AI / SpaceX = autonomous reentry AI / xAI 本身就是 AI / Neuralink = brain-AI 接口 / X = AI 算法平台 / Boring = autonomous tunneling 远期）

### 不是什么

- ❌ 不是 Musk 个人崇拜账号（不舔，不喊"king elon"）
- ❌ 不是 Musk 黑账号（不混 @DemocracyDocket 那种空头大军）
- ❌ 不是泛 AI 评论员（专注 xAI 这一条线 + AI race 对照）
- ❌ 不是 thread bro（不写"21 lessons from elon"）
- ❌ 不是政治评论员（不碰 Trump / 大选 / 言论自由 / Musk 个人生活）
- ❌ 不是搬运工（不是把别处 Musk 新闻翻译过来抄）

## 二、4 大内容支柱

| 支柱 | 占比 | 形式 | 作用 |
|---|---|---|---|
| **A · Musk Reply Guy** | 45% | 单条 reply | 每天 5-10 条，主要打 @elonmusk / @Tesla / @SpaceX / @xai / @neuralink / @sama / Tesla 大 V，**这是涨粉主引擎** |
| **B · Empire Tracking** | 25% | 单推 / 短 thread | 每天 1-2 条，跟踪具体事件：SpaceX 发射 / Tesla 周交付 / Optimus demo / Grok 更新 / Neuralink 临床 |
| **C · Musk Decoder** | 20% | 短 thread / 长推 | 每周 1-2 条，解码 Musk 的发言（cryptic 推 / 路线图变更 / 组织调整）、Tesla 财报、SpaceX 路演的隐藏信号 |
| **D · AI Race Watch** | 10% | 单推 / 对比图 | 每周 1-2 条，xAI vs OpenAI vs Anthropic 横向比，所有 AI race 的 framing 都通过"Musk 怎么打这个仗"切入 |

### 各支柱发推模板

#### A · Musk Reply Guy（详见 [x-reply-scripts.md](./x-reply-scripts.md)）

#### B · Empire Tracking

**6 种 sub-format**：

```
1. Launch / event live tweet（SpaceX 发射 / Tesla 发布会 / xAI Colossus 上线）：
   "starship flight 12 t-30. things i'm watching:
    - hot stage timing
    - booster catch attempt 4
    - in-orbit refuel test
    last flight failed at min 8. let's see."

2. Numbers tracking（Tesla 周/季交付 / xAI 招聘数 / Optimus 单元数）：
   "tesla q1 2026:
    - global deliveries: 401k (+8% yoy)
    - shanghai: 195k (record)
    - berlin: 48k
    - austin: 73k
    fsd take rate finally cracked 30%. that's the number."

3. Roadmap update（Musk 又改路线图 / Tesla / xAI 又改 timeline）：
   "musk just bumped optimus mass production from 2026 → 2027.
    that's the 4th timeline change for optimus.
    pattern: musk timelines run 2-3x late on production, 0.5-1x late on demo."

4. Spotting / sighting（街拍 cybertruck / fsd v13 in the wild）：
   "first cybertruck spotted in <city>, <date>. screenshot from instagram.
    that's <number> markets cybertruck has reached now."

5. Cross-company connect（Musk 帝国内部协同动作）：
   "noticing: xai colossus 2 announced same week tesla data center expansion
    rumored. those compute clusters might end up sharing infra. that'd be the
    musk empire's quiet moat play."

6. Comparison（与对手）：
   "fsd v13 vs waymo 6th gen, by metric (i ran my own tests last week):
    - intervention rate: <数字>
    - average speed maintained: <数字>
    - urban turn left at unprotected: <数字>"
```

#### C · Musk Decoder

**3 种 sub-format**：

```
1. 解码 Musk 的 cryptic 推（每周 1-2 次）：
   "elon just tweeted '!!!' under @xai's quote post.
    historical pattern: 3 of his 4 last '!!!' replies preceded major releases
    within 14 days. taking notes."

2. 解码财报 / 路演 / 内部信（季度级）：
   "tesla q1 earnings call. 3 things bears missed:
    1/ <具体>
    2/ <具体>
    3/ <具体>
    none of these were in the press release. they were in the q&a buried at min 47."

3. 解码组织调整（不定期）：
   "elon promoted <name> to head <division> last week.
    background: <name> ran <prior project>, came over from <prior co> in <year>.
    this is musk's standard 'engineer i trust over manager from outside' move.
    expect <prediction>."
```

#### D · AI Race Watch（用 Musk 视角切 AI 大局）

```
1. xAI vs OpenAI 阶段性比：
   "grok 4 vs gpt-5, on the same 5 prompts (i ran them yesterday):
    1. coding: gpt-5 still wins, but gap closed from 2x → 1.3x
    2. real-time info: grok wins easy (live x training set is the moat)
    3. long context: claude still wins both
    4. multimodal: gpt-5 wins
    5. humor / personality: grok by a mile, somehow
    
    the AI race is now 3-horse for first time in 2 years."

2. 投资视角（不给投资建议，只给信号）：
   "xai funding round 6 closed at $200B valuation last week.
    for context: openai's $300B, anthropic's $80B, google's <number>.
    the 'one of these is not like the others' joke writes itself but..."

3. AI 路线分歧（用 Musk vs Sam 的公开分歧来 framing）：
   "the open-source vs closed-source AI debate, refereed by who's actually doing what:
    - meta: open
    - mistral: open
    - xai (grok 1, grok 2): open weights eventually
    - openai: closed
    - anthropic: closed
    elon keeps calling sam 'closedai.' the labels match."
```

## 四、变现路线图（M1 → M6）

> Musk-watcher 受众是高净值科技投资人群，**ARPU 比泛 AI 受众高 3-5x**。但前 2 个月只建立信任 + 留 lead，**不卖**。

### M1（第 1 个月）· 0 → 1K-5K 粉

**变现动作**：0（投资期）

**铺垫动作**：
- 开 X Premium+（$16/月，分成资格 + 蓝标加权）
- 开 Beehiiv Newsletter "**Musk Empire Decoder Weekly**"（每周一发，免费）
- 做 **Musk Empire 2026 Cheat Sheet PDF**（1 页 / 6 公司 / 当前进展 + 下一里程碑 + 看点，免费）
- bio link 指向 PDF 领取页（页面顺势收 newsletter 邮箱）

### M2（第 2 个月）· 5K → 12K 粉

**变现动作**：

| 路径 | 收入预期（月）|

## 五、避坑清单（X 平台特有 + Musk 题材特有）

### X 平台通用

| 类别 | 不要做 | 原因 |
|---|---|---|
| 算法 | 一天发超过 15 条原创推 | 灌水降权 |
| 算法 | 在同一大 V 评论区一天回 3 条以上 | 判 spam |
| 算法 | 大量推文带 5+ hashtag | 现在算法不喜欢 hashtag 堆 |
| 合规 | 伪造 Musk / Tesla / SpaceX / xAI 内部消息 | 永封 |
| 合规 | 任何 NSFW / 暴力 / 仇恨内容 | 永封 |
| 变现 | 1 个月内推付费产品 | 信任未建立，转化 = 0 还掉粉 |
| 变现 | 买粉 / 买互动 | X 反作弊会取消营收资格 |

### Musk 题材特有

| 类别 | 不要做 | 原因 |
|---|---|---|
| 政治 | 站 Musk 政治立场（Trump / 大选 / 言论自由口水仗）| 一边 50% 受众取关；不站队两边都看 |
| 政治 | 跟 Musk 黑账号大军（@DemocracyDocket / @ElonMuskAOC 类）混 | 同上反向 |
| 个人 | 评论 Musk 婚姻 / 儿女 / Grimes / 私生活 | 八卦受众和帝国分析受众完全不重合，自毁定位 |
| 个人 | 评论 Musk 的健康 / 用药 | 同上 |
| 法律 | 评论 Musk 跟 SEC / Twitter Inc 老股东的诉讼 | 法律细节门槛高，错一个会被群嘲 |
| 假冒 | 用 Musk 头像做 banner / pfp / 仿冒名 | X 反假冒系统会标记，蓝标审核失败 |
| 假数据 | 拍脑袋编 Tesla 销量 / SpaceX 任务数 | 一旦被 @SawyerMerritt 类专业账号挂出来，账号信誉清零 |
| 蹭车祸 / 事故 | 在 Tesla 车祸 / SpaceX 失败现场推下面凑热闹 | 算法和受众都讨厌 |
