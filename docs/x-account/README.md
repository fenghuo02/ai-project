## 一句话定位（先想清楚，再写一行）

> **A solo AI builder in China — shipping products with AI, posting what I see from the inside.**
>
> （翻译：一个在中国的独立 AI 产品开发者 —— 用 AI 造东西，分享我从内部看到的真实景象。）

为什么这个定位能赢：

1. **稀缺供给**：英文 X 上"中国实操派 AI builder"的真实视角，在 Musk / Naval / levelsio / garrytan / @sama 的关注流里几乎没有竞品。
2. **天然钩子**：Musk 长期关注 Tesla 中国 + 中国 AI 进展，账号天然进入他/他粉丝的语义关联池。
3. **可证伪**：每周晒一次产品截图 / 数字 / 翻车，反 AI 味、反"thread bro"。
4. **可变现**：英文受众的 ARPU 是中文的 5-10x（X Premium 分成 / 付费订阅 / 数字产品 / 1:1 咨询）。

## 核心反 AI 味准则（贯穿所有内容）

每条推文 / 回复发出去前自检这 8 条，命中任何一条就改：

| # | AI 味红线 | 人味替代 |
|---|---------|--------|
| 1 | 句首大写 / 完美标点 | 全小写开头，逗号断句，偶尔无句号结尾 |
| 2 | 破折号 — / 长划线 — | 直接用空格 + 逗号，或者另起一行 |
| 3 | "Absolutely / Indeed / Furthermore / In conclusion" | 删，要么不说，要么 "yeah / fwiw / ngl / tbh" |
| 4 | 三段式 / 三个 bullet | 一句话甩观点 → 一个数字 → 一个反问 |
| 5 | 普世正确的废话（"AI is transforming everything"）| 具体到本周某个产品某个数字某个翻车 |
| 6 | 礼貌过头（"Great point!" "Thanks for sharing!"）| 直接进入分歧 / 补充 / 反问 / 玩笑 |
| 7 | 感叹号超过 1 个 | 0 个感叹号是默认值 |
| 8 | 完整、没有错别字、没有语气词 | 偶尔留一个 typo 不改，留 "lol" "fr" "+1" |

> 反 AI 味的本质不是写错字，是**有具体性 + 有立场 + 有时间地点**。

## 风险红线（一次违规可能毁号）

1. ❌ **不要**为了蹭 Musk 频繁 @ 他或刷屏在他评论区（X 算法会判 spam，封 reply 权限 24-72h）
2. ❌ **不要**伪造 Tesla / OpenAI / xAI 内部消息（X 现在对"impersonation / misinformation"零容忍，可永封）
3. ❌ **不要**买粉 / 买互动（500 粉门槛和 5M impression 门槛被 X 反作弊系统盯得很死，作弊会取消营收资格）
4. ❌ **不要**纯转发翻译墙内中文新闻（这条赛道是"我亲眼看见 + 亲手做的"，不是搬运）
5. ❌ **不要**碰政治敏感（中美关系 / 台海 / 新疆 / 俄乌 / 以巴），账号定位是 builder 不是 commentator
