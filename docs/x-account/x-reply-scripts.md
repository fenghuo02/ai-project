## 一、为什么大多数 reply 一眼 AI 味

| AI 味元素 | 真实人怎么写 |
|---|---|
| `Absolutely! This is a great point.` | （删，不需要捧哏开头）|
| `Furthermore, ...` / `Moreover, ...` | `also` / `+ ` / 直接换行 |
| `In conclusion, ...` | （删，不需要总结）|
| `As an AI / As someone who ...` | 直接说事 |
| `It's important to note that ...` | `worth noting:` / 直接说 |
| `Here are 3 reasons why:` + 3 bullets | 1 句甩出去 |
| em-dash — | 逗号 / 句号 / 换行 |
| 完美大写 + 句号 | 全小写，偶尔无句号 |
| 感叹号超过 1 个 | 0 个是默认 |
| 句子完整，没 typo | 偶尔留一个，不改 |
| `Thank you for sharing this insight!` | （删）|
| 每条 reply 都谨慎 / 礼貌 | 偶尔 dry humor / 偶尔反对 |

## 二、人味词典（直接用，不用每次想）

### 开头词（替代 "Absolutely / Indeed / Great point"）

```
fwiw     - for what it's worth
ngl      - not gonna lie
tbh      - to be honest
ime      - in my experience
imo      - in my opinion
fr       - for real
+1       - 同意
yeah but - 同意一半但有补充
ok but   - 同意但要换角度
oh       - 表示意外
hm       - 表示在思考
wait     - 表示发现矛盾
nah      - 不同意（但不刺）
lol      - 自嘲 / 表达不严肃
no joke  - 接下来要说一件听起来夸张但是真的事
```

### 结尾词（替代 "Hope this helps! / Looking forward to your thoughts!"）

```
（什么都不加）   - 多数情况最好
.                - 一个句号收尾，干净
?                - 抛一个真问题
🫠 / 🤔 / 🥲     - 一个 emoji（最多 1 个）
(don't @ me)     - 自嘲式
ymmv             - your mileage may vary, "因人而异"
tldr;<一句>     - 自己总结一句，放在末尾
```

### 万用过渡（替代 "Furthermore / Moreover / Additionally"）

```
also      
plus     
and      
btw      
oh and   
side note: 
or maybe 
ok wait, 
```

### 显示思考过程的语气词（让 reply 像活人写的）

```
huh,             - 突然意识到
wait wait,       - 临时换思路
ok so,           - 开始解释
the thing is,    - 引出真正核心
honestly?        - 接下来要说实话
genuinely curious: - 真问题（避免被当成钓鱼）
hot take:        - 接下来要说一个有争议的观点
unpopular but:   - 同上
context:         - 接下来要给背景
```

## 九、5 种"切入角度"分类速查

> 每条 reply 写之前先想：我用哪个角度？

| 角度 | 何时用 | 模板锚 |
|---|---|---|
| **A 中国视角** | 推主话题与中国 / Tesla 上海 / 中国 AI 有任何关联 | `from chongqing,` / `lived here 5 years,` / `chinese AI scene fyi:` |
| **B 实操派** | 推主谈技术 / 产品 / 做事 | `tried this last week,` / `shipped a thing using this,` / `ime,` |
| **C 数据 / 反常识** | 推主结论太通用 / 太刻板 | `data point:` / `worth noting:` / `fwiw the actual number is` |
| **D 玩笑 / 梗** | 推主自己在玩梗 / 轻松氛围 | `lmao` / `ok this wins the day` / `(skipping the engineering for once)` |
| **E 真问题** | 推主话题你确实有真问题 | `genuinely curious:` / `legit asking:` / `hot take qn:` |

> ⚠️ 不要全部走"A 中国视角"，会显得只有这一张牌。混着打。
> 一周内分布建议：A 30% / B 30% / C 20% / D 10% / E 10%。

## 十、5 种"反对 / 补充"软化句式（避免被当成杠精）

```
1. agreed in spirit but ime <补充>
2. yeah but the data here is <反驳>
3. ok counter-take from chongqing: <反驳>
4. half agree. the part that lands is <X>. the part that doesn't is <Y>.
5. unpopular but: <反驳>
```

> 反对要"具体到事实"，不要"反对态度"。X 算法和受众都讨厌"杠精",但**带数据/经验的反对会拉粉**。

## 十一、必须避开的雷（每条都见过 X 账号被毁）

| ❌ 雷 | 为什么 |
|---|---|
| 同一句话回到 5 个不同大 V 下 | X 反作弊系统 24h 内会标记，封 reply 权限 |
| 在 reply 里 @ 推主之外的人（特别是 @elonmusk）| 判 spam，封 reply 权限 |
| 一天回同一大 V 超过 3 条 | 同上 |
| 在 reply 里挂 link（你自己的产品 / 落地页）| 算法降权，且会被举报 spam |
| reply 长度超过 280 字符（自动变长推） | 长 reply 在评论区折叠率 80%+，没人看 |
| 用 emoji 列表 / 三个 bullet | 一眼 thread bro，是 AI 味重灾区 |
| reply 里直接卖东西 / 推 newsletter | 信任未建立，效果反向 |
| 政治 / 民族 / 宗教 / 性 主题 reply | 永久禁声 / 永封风险 |
| 假冒任何官号 / 大 V | 永封 |
| reply 里写"as an AI" / "i'm an AI" | 自爆，账号信任度归零 |
| 用 ChatGPT 默认风格回（"Certainly! Here are 3 thoughts:"）| AI 味满级，受众一眼识破 |

## 十二、回复后 30 分钟操作清单

每条 reply 发出后，做这 3 件事：

1. **不要立刻删 / 改**：X 算法对"发后 30 秒内编辑"会标记，影响推荐
2. **30 分钟后看数据**：impression < 100 = 失败，复盘原因（话题切错？时间太晚？太 AI 味？）
3. **如果 reply 起来了**（impression > 1K 或被推主点赞）：
   - **立刻发 1 条独立推**深化这个话题（不是 quote tweet 那条 reply，而是另起炉灶）
   - 把那条 reply 收藏到 [x-reply-scripts.md](./x-reply-scripts.md) 末尾的"已验证模板"部分（自己积累）
