## 「三重切割」原则（所有文件须共同满足）

1. **股权切割**：易学老师对 A、B 两家公司**持股 0**，不出现在股东名册、不做实际控制人。
2. **任职切割**：老师**不在** A、B 任职（不任董事/监事/高管/员工），不签劳动合同、不发工资。
3. **合同主体切割**：老师仅以**自有个体工商户**身份，与运营主体签**文化顾问服务协议**收取顾问费；不作为摄影/文创合同的签约主体或收款主体。

> 合规底线：隔离必须**名实相符**。若实际由老师控制资金、决策、人事，则可能被认定为**隐名股东 / 实际控制人**，切割失效。建议老师报酬采用「**固定顾问费 + 封顶绩效**」，避免「按营业额分成」带来的隐名股东与税务风险。
