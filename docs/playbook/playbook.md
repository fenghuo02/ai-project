## 五、差异化改写规则（防搬运限流）

| 维度 | 小红书版 | 公众号版 |
|------|---------|---------|
| 标题字数 | 18-22 字 | 20-28 字 |
| 标题语气 | "姐妹必看""绝了" | "深度解析""实操指南" |
| 正文字数 | 300-500 | 1500-2500 |
| 开头 | 身份+痛点钩子 | 故事/数据/反常识 |
| 配图 | 9 宫格 3:4 竖图 | 文中横图 2.35:1 |
| 结尾 | 互动提问 | 观点总结+引导关注 |
| 引流 | 简介暗示钩子 | 直接引导公众号菜单 |


## 七、避坑清单

- **违禁词**：最、第一、绝对、国家级、永久、包治
- **违规词**：微信、加我、私聊、VX（小红书严打）
- **限流雷区**：
  - 纯 AI 文（必须人工改写加真实细节）
  - 全网搬运（同一文案多平台同时发）
  - 新号前 7 天频繁发布（先养号）
- **公众号注意**：内容里可以放微信，但不能过度营销
