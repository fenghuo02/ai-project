## 快速开始

### Day 0（准备日）
1. 读完 [`playbook-yixue.md`](./playbook-yixue.md) 了解整体策略（30 分钟）
2. 读 [`m1-procurement-checklist.md`](./m1-procurement-checklist.md) 了解 M1 形象采购节奏
3. 注册小红书账号 + 公众号，**先养号 7 天**（只看不发）

### Day 1-21（M1）
1. 按 [`m1-procurement-checklist.md`](./m1-procurement-checklist.md) 第 1-3 周完成 ¥4,500 形象采购
2. 按 [`lead-magnet-19-colors.md`](./lead-magnet-19-colors.md) §六 10 天计划做出 V0.1 钩子 PDF
3. 按 [`wechat-onboarding.md`](./wechat-onboarding.md) §一 / §二 / §三 配置公众号
4. M1 第 4 周拍第一组样片，正式开始发笔记

### Day 22+（持续运营）
- 用 [`prompts-yixue-photo.md`](./prompts-yixue-photo.md) Prompt 1-13 持续产出图与文案
- 用 [`prompts-content.md`](./prompts-content.md) Prompt 1-7 写笔记 / 公众号长文 / 运营评论 / 数据复盘
- 每周二/五把爆款笔记改写成公众号长文（用 `prompts-content.md` Prompt 3）
- 每月 1 日按 [`prompts-content.md`](./prompts-content.md) Prompt 4 §4 推进钩子升级（M2 速测 → M3 节气壁纸 → M4 主理人手记 → M5 案例集 → M6 钩子复盘）

## 工作流（v1.0）

```
M1 第 1-3 周：按 m1-procurement-checklist.md 完成形象采购（¥4,500）
              按 lead-magnet-19-colors.md 做出 V0.1 钩子 PDF
              按 wechat-onboarding.md 配置公众号 7 个关键词
M1 第 4 周  ：拍第一组样片，发布第 1 周 5 篇笔记
每周一      ：用 prompts-content.md Prompt 5（选题雷达）扫下周选题
每日        ：用 prompts-yixue-photo.md Prompt 5b/13 跑图
              用 prompts-content.md Prompt 1/2 写笔记
              用 prompts-content.md Prompt 6 运营评论区
周二/五     ：用 prompts-content.md Prompt 3 把爆款改公众号长文
周五晚      ：用 prompts-content.md Prompt 7 数据复盘 + 排下周
每月 1 日   ：用 prompts-content.md Prompt 4 刷新引流钩子
```
