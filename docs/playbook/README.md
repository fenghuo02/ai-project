## 快速开始

### Day 0（准备日）
1. 读完 `playbook.md` 了解整体策略
2. 按 `wechat-onboarding.md` 配置公众号
3. 按 `covers-design.md` 在 Canva 准备 9 宫格模板
4. 开始写 `lead-magnet-100-prompts.md` 中的 20 条核心 Prompt（MVP 版 PDF）

### Day 1-3（首波内容）
1. 按 `first-post.md` 发布小红书第 1 篇
2. 按 `post-02-weekly-report.md` 发布第 2 篇
3. 按 `post-03-chatgpt-assistant.md` 发布第 3 篇

### Day 4+（持续运营）
- 用 `prompts.md` 中的 Prompt 持续产出
- 每周五用 Prompt 7 做数据复盘
- 每周二/五用 Prompt 3 把爆款改成公众号长文

## 工作流

```
每周一：用 Prompt 5 扫下周选题
每日  ：用 Prompt 1/2 写小红书笔记
周二五：用 Prompt 3 把爆款改公众号长文
每晚  ：用 Prompt 6 运营评论区
周五晚：用 Prompt 7 复盘数据 + 排下周
每月初：用 Prompt 4 刷新引流钩子
```
