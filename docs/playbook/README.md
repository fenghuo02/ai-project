# Xiaohongshu + WeChat Playbook

小红书 + 公众号双平台运营方案（AI 工具 + 职场效率赛道）

## 文件说明

| 文件 | 内容 |
|------|------|
| [playbook.md](./playbook.md) | 完整运营手册：定位、栏目、30 天内容日历、变现路径 |
| [prompts.md](./prompts.md) | 7 套核心 Prompt：小红书+公众号双平台内容生产工具包 |
| [first-post.md](./first-post.md) | 第 1 篇笔记成品：小红书版 + 公众号版 + 配图 + 引流钩子 |

## 快速开始

1. 读 `playbook.md` 了解整体策略
2. 按 `first-post.md` 发第 1 篇笔记
3. 用 `prompts.md` 持续产出内容

## 工作流

```
每周一：用 Prompt 5 扫下周选题
每日  ：用 Prompt 1/2 写小红书笔记
周二五：用 Prompt 3 把爆款改公众号长文
每晚  ：用 Prompt 6 运营评论区
周五晚：用 Prompt 7 复盘数据 + 排下周
每月初：用 Prompt 4 刷新引流钩子
```
