## 快速开始

### Day 0（准备日）
1. 读完 [`playbook-yixue.md`](./playbook-yixue.md) 了解整体策略（30 分钟，注意 §三 30 天平台分发日历 / §四 4 平台发布节奏）
2. 读 [`m1-procurement-checklist.md`](./m1-procurement-checklist.md) 了解 M1 形象采购节奏
3. 5 个号同步注册：抖音 / 视频号 / B 站 / 公众号 / 小红书，**先养号 7 天**（只看不发）
4. 读 [`bilibili-onboarding.md`](./bilibili-onboarding.md) §一 + [`wechat-onboarding.md`](./wechat-onboarding.md) §一/二/三 配置 B 站 + 公众号

### Day 1-21（M1）
1. 按 [`m1-procurement-checklist.md`](./m1-procurement-checklist.md) 第 1-3 周完成 ¥4,500 形象采购
2. 按 [`lead-magnet-19-colors.md`](./lead-magnet-19-colors.md) §六 10 天计划做出 V0.1 钩子 PDF
3. 按 [`wechat-onboarding.md`](./wechat-onboarding.md) §一 / §二 / §三 配置公众号 7 关键词 + 3×5 菜单
4. 按 [`bilibili-onboarding.md`](./bilibili-onboarding.md) §六 完成 B 站置顶视频 + 第一条月度长视频
5. M1 第 4 周拍第一组样片，正式开始发笔记 + 短视频

### Day 22+（持续运营 · 4 平台矩阵）
- 用 [`prompts-yixue-photo.md`](./prompts-yixue-photo.md) Prompt 1-13 持续产出图与文案
- 用 [`short-video-script-prompts.md`](./short-video-script-prompts.md) Prompt 1 每周产出 2-3 条短视频（抖音/视频号/B 站同发）
- 用 [`prompts-content.md`](./prompts-content.md) Prompt 3 每周二/五写公众号长文，发完后用 Prompt 1 12h 内搬运到小红书
- 用 [`bilibili-onboarding.md`](./bilibili-onboarding.md) §2.2 每月最后一周日发布 1 条 B 站长视频
- 每月 1 日按 [`prompts-content.md`](./prompts-content.md) Prompt 4 推进钩子升级（M2 速测 → M3 节气壁纸 → M4 主理人手记 → M5 案例集 → M6 钩子复盘）

## 工作流（v1.5 · 4 平台矩阵）

```
M1 第 1-3 周：按 m1-procurement-checklist.md 完成形象采购（¥4,500）
              按 lead-magnet-19-colors.md 做出 V0.1 钩子 PDF
              按 wechat-onboarding.md 配置公众号 7 个关键词
              按 bilibili-onboarding.md §一-六 配置 B 站账号 + 置顶视频
M1 第 4 周  ：拍第一组样片
              按 playbook-yixue.md §三 30 天平台分发日历执行第 1 周

每周一      ：用 prompts-content.md Prompt 5（选题雷达）扫下周选题
每周一/三/日：用 short-video-script-prompts.md Prompt 1 三平台同发短视频
              抖音先发 30 分钟 → 视频号 + B 站同步
每周二/五   ：用 prompts-content.md Prompt 3 写公众号长文，晚 20:30 推送
              12h 内用 Prompt 1 搬运到小红书图文版
每天        ：用 prompts-content.md Prompt 6 运营评论区
              四平台共用同一套合规话术
每月最后周日：用 bilibili-onboarding.md §2.2 发 1 条 B 站长视频（月度）
每月特殊场景：用 prompts-content.md Prompt 2 写小红书原创补充（节气/vlog/案例快讯）
周五晚      ：用 prompts-content.md Prompt 7 跑四平台数据复盘
每月 1 日   ：用 prompts-content.md Prompt 4 刷新引流钩子
```

### 月度内容产能矩阵

| 内容类型 | 月度数量 | 单条产出时间 |
|---|---|---|
| 📝 公众号长文 | 8 篇 | 90-120 分钟（原创主阵地）|
| 🔄 小红书图文搬运 | 8 篇 | **30 分钟**（公众号长文搬运）|
| 📱 三平台同发短视频 | 12 条 | 60-90 分钟（一份信息卡 → 三脚本一次性吐出）|
| 🎬 B 站长视频 | 1 条 | 5-7 天（拍摄 + 剪辑 + 字幕）|
| ✍️ 小红书原创补充 | 2-4 条 | 60 分钟 |

**总产能**：约 60-80 小时 / 月，**单兵作战可承担**（参考业务计划书 v1.5 §3）。
