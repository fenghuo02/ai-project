# 小红书 + 公众号双平台 Prompt 工具包

> 配套《playbook.md》使用
> 按 Prompt 1 → 2 → 3 → 4 顺序使用，一次产出双平台内容

---

## Prompt 1：小红书爆款笔记生成器（AI 工具测评版）

```
你是小红书 "AI工具测评" 领域头部博主，熟悉打工人痛点。
请帮我写一篇 AI 工具测评笔记。

【工具名称】：<填>
【核心功能】：<填>
【适用场景】：<职场/学习/副业>
【免费还是付费】：<填>
【我的使用感受】：<填 2-3 句真实体验>

输出要求：
1. 标题 10 版，18-22 字，必须包含以下钩子之一：
   - "打工人"/"i 人"/"社恐" 等身份标签
   - 数字承诺（3 分钟/月入 X/省 X 小时）
   - 反差（加班 vs 准点下班 / 小白 vs 高手）
   - 避雷/安利强情绪

2. 正文结构：
   ① 开头 3 秒钩子：用"加班到 10 点""被老板骂"等具体职场场景切入
   ② 痛点共鸣：列出 3 个打工人真实痛点
   ③ 工具介绍：1 句话说清楚能干什么（不要堆功能）
   ④ 使用步骤：3-5 步，每步配一句大白话说明
   ⑤ 真实效果：给出具体数据（从 X 小时 → Y 分钟）
   ⑥ 埋 1 个小缺点（增加可信度）
   ⑦ 结尾互动：问"你们都用什么 AI 工具？评论区聊聊"

3. 字数 350-450 字，每段不超过 2 行，多空行
4. 自然穿插 3-5 个搜索关键词（帮我标出来）
5. 结尾给 5 个话题标签
6. 给出 9 宫格配图建议（每张图的内容和文字）

【避雷词】：不要出现"最""第一""绝对""微信""加我"等
```

## Prompt 2：小红书爆款笔记生成器（AI 效率教程版）

```
你是小红书 "AI效率教程" 博主。请帮我写一篇教程笔记。

【教程主题】：<填>（例：用 AI 30 秒写周报）
【解决什么问题】：<填>
【需要用到的工具】：<填>
【预期节省时间】：<填>

输出要求：
1. 标题 10 版，18-22 字
   - 带数字："30 秒"、"3 步"、"1 分钟"
   - 带反差："从 2 小时到 10 分钟"
   - 带身份："打工人"、"社畜"、"i 人"

2. 正文结构：
   ① 开头：具体场景 + 痛点（例：周报被老板打回 3 次）
   ② 工具准备：1-2 句说清楚（附链接/名称）
   ③ 3-5 个 Step 详细步骤，每步带上实操话术/提示词
   ④ 效果对比：传统方法 vs AI 方法
   ⑤ 1 个小 tips（进阶玩法）
   ⑥ 结尾引导收藏："收藏慢慢练 / 用过的姐妹回来告诉我效果"

3. 字数 400-500 字
4. 穿插 3-5 个搜索词（标出）
5. 5 个话题标签
6. 9 宫格配图建议（截图步骤为主）

【重点】：一定要给出完整可复制的 Prompt/提示词示例
```

## Prompt 3：公众号深度文改写器

```
你是职场干货类公众号主笔。
请把下面的小红书笔记，改写成一篇 1500-2000 字的公众号深度文。

【小红书原文】：<粘贴>
【公众号定位】：帮打工人用 AI 提升效率和搞副业
【目标读者】：25-35 岁职场人

改写要求：
1. 标题重写：20-28 字，专业感
   - 不要用"姐妹""绝了""yyds"
   - 例：不要"打工人必装！这个 AI 神器绝了"
   - 改成"我用这款 AI 工具，把周报时间从 2 小时压到 10 分钟"

2. 开头重写（150 字内）：
   - 不要"姐妹们""宝子们"
   - 用 1 个真实故事 / 1 组反差数据 / 1 个行业现象切入

3. 正文扩写：
   - 3-5 个 H2 小标题
   - 每个工具/方法要有：原理解释 + 使用场景 + 实操步骤 + 效果数据
   - 加入"我的思考""行业观察"等主观见解
   - 每段不超过 3 行

4. 金句设计：至少 2 句可以单独发朋友圈的话，用加粗标出

5. 结尾：
   - 总结 3 条核心观点
   - 引导"点赞在看""转发同事""关注不迷路"
   - 最后一句埋钩子："明天聊 XXX" 吊住下次打开

6. 配图建议：标出 3-5 处插图位置 + 每张图的说明文字

7. 文末话题标签 3 个

【风格】：专业有温度，像同行前辈分享经验
```

## Prompt 4：引流钩子设计器

```
你是私域引流专家。帮我设计一套合规的引流钩子，
把小红书粉丝导流到公众号。

【我的钩子资源】：
- 100 个打工人 AI 提示词合集
- 20 个 AI 工具免费版清单
- AI 副业起步指南 PDF

【引流目标】：
小红书粉丝 → 公众号关注 → 后续卖课/社群

要求：
1. 给 5 条小红书评论区可以用的"暗语"话术（不违规）
   - 不能出现"微信""公众号""加我"等敏感词
   - 要让老用户一看就懂、新用户产生好奇

2. 给 3 版小红书简介话术（暗示钩子在哪领）

3. 给公众号自动回复话术：
   - 用户回复关键词自动推送哪些内容
   - 设计 3 个关键词（如"AI"、"提示词"、"副业"）

4. 给出钩子迭代计划：
   - 每月主推哪个钩子
   - 什么时候推出新钩子
   - 如何让老粉丝持续关注
```

## Prompt 5：爆款选题雷达（每周用一次）

```
你是小红书选题专家。
请围绕【AI 工具 + 职场效率】这个赛道，
扫描当前 1-2 周的热点话题，生成 20 个爆款选题。

要求：
1. 每个选题命中至少一个钩子：
   ① 强对比（前后对比/反常识）
   ② 数字承诺（X 分钟 / 月入 X / 省 X 元）
   ③ 身份标签（i 人 / 打工人 / 程序员）
   ④ 痛点共鸣（加班 / 被 PUA / 月光）
   ⑤ 信息差（别人不告诉你的）
   ⑥ 避雷反面（千万别买 / 踩坑）

2. 每个选题输出：
   【选题】|【钩子类型】|【目标人群】|【爆款概率 ★1-5】|【变现衔接】

3. 筛出 Top 5 推荐，说明理由
4. 给每个 Top 5 搭配 1 个公众号延展主题（深度版）
```

## Prompt 6：评论区运营助手

```
你是小红书评论区运营专家。
我的笔记主题是：<填写>
下面是评论列表：
<粘贴评论>

请帮我：
1. 识别"高价值评论"（可以引导到公众号/加私域的潜在客户）
2. 对每条评论生成 1 条回复：
   - 有温度，像朋友说话
   - 适时反问，拉长互动
   - 对高价值评论，设计"钩子话术"引导（不能违规）
3. 另外生成 3 条"自问自答"评论
   - 给小号用
   - 烘托氛围 + 带搜索关键词
```

## Prompt 7：数据复盘分析师（每周五用）

```
你是小红书数据运营专家。下面是我近期 10 篇笔记的数据：

<粘贴数据：标题 | 曝光 | 点击率 | 互动率 | 涨粉>

请帮我：
1. 找出表现最好的 2 篇和最差的 2 篇，分别分析原因
2. 对比它们的标题/选题/发布时间/话题标签
3. 给出下周 5 个优化方向（按优先级）
4. 生成下周 7 篇选题计划（每天 1 篇，包括发布时间建议）
5. 建议这周哪篇爆款可以改写成公众号长文
```

---

## 使用顺序建议

```
每周流程：
  周一：用 Prompt 5 扫选题 → 排下周内容
  周二-周六：用 Prompt 1 或 2 写小红书 → 下午发
  周三/周五：用 Prompt 3 把爆款改公众号长文
  每天：用 Prompt 6 运营评论区
  周五晚：用 Prompt 7 复盘数据
  钩子调整：每月第 1 天用 Prompt 4 刷新
```
