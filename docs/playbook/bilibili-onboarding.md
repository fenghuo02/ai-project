> 版本：v1.0 · 2026 春（与业务计划书 v1.5 §3 平台矩阵 + [`short-video-script-prompts.md`](./short-video-script-prompts.md) 配套）

## ⚠️ 平台合规底线（先读这一段）

**B 站的合规边界比抖音宽松，但绝不无禁**——它对**封建迷信 / 玄学预测 / 算命相关**的关键词检测和小红书 / 抖音 / 视频号一致严，多了一层「学术 / 科普 / 文化」上下文宽容度，但**不要把这层宽容度当成可以越界**。

### B 站特别敏感的内容

| 类目 | 处理 |
|---|---|
| **算命 / 看相 / 八字** 等 L1 禁词（详见 [`.kiro/steering/01-compliance-redlines.md`](../../.kiro/steering/01-compliance-redlines.md)）| 与抖音/小红书等同处理：客户面 0 出现 |
| **奇门 / 八门 / 死门 / 惊门 / 用神 / 调候 / 命格** 等 L2 命理词 | **B 站可适度提及（限"科普向"长视频）**，但必须放在「学术参考 / 文化考据 / 古籍引用」框架内，且必须明确"我们不做命理预测"的声明 |
| **天干日主**（丁火 / 庚金等） | **可在 B 站长视频中讲**（属于知识科普），但不做"X 日主就要 X"的宿命论结论 |
| **「不算命」「不是预测」否定句式** | 永远禁用（L3 陷阱）。B 站和其他平台一样会被关键词匹配算法触发 |

### B 站独有的合规要求

- ⚠️ **AIGC 标识**（《人工智能生成合成内容标识办法》2025-09-01 施行）：B 站投稿必须在投稿设置中勾选「**视频内容由 AI 生成或合成**」，未勾选会被识别后限流（不仅 B 站，全平台都需要，详见业务计划书 §11.2）
- ⚠️ **投稿分区**：本号**绝不投到「玄学占卜」「迷信民俗」相关分区**，应投：
  - **生活 → 美食 / 美妆 / 时尚** 区（命主画像 / 节气写真）
  - **生活 → 手工 / 日常** 区（矿物颜料试色 / 装裱花絮）
  - **生活 → 家居家装** 区（节气壁纸 / 居家美学）
  - **知识 → 人文历史 → 文化习俗** 区（节气 / 24 节气壁纸 / 传统色科普长视频）
  - **知识 → 设计·创意** 区（19 色色卡 / 配色逻辑长视频）
- ⚠️ **不开「充电专属」付费视频**前 3 个月——避免被认定为「玄学课程付费」违规

### 与短视频三平台同发的差异

详见 [`short-video-script-prompts.md`](./short-video-script-prompts.md) §三 B 站版脚本要求。简言之：**B 站版多 5-10 秒慢节奏开场 + 末尾不放强 CTA**。

## 二、视频内容矩阵：长视频 + 短视频双轮驱动

> **B 站的核心定位 = 主理人深度 IP + 长视频护城河**。
> 短视频做流量入口（与抖音/视频号同发），长视频做知识沉淀和高客单转化。

### 2.1 内容形态对比

| 形态 | 长度 | 内容类型 | 节奏 | 与公众号/其他平台的关系 |
|---|---|---|---|---|
| **长视频** ⭐ | 5-15 分钟 | 知识科普 / 客户案例完整故事 / 主理人 vlog | **月度 1 条**（含 1 周拍摄 + 1 周剪辑）| **1 篇公众号深度文 = 1 条 B 站长视频**（强复用）|
| **短视频** | 30-90 秒 | 节气仪式感 / 老师答疑 / 客户案例片段 | **每周 2-3 条**（与抖音/视频号同发）| 详见 [`short-video-script-prompts.md`](./short-video-script-prompts.md) B 站版要求 |

### 2.2 长视频 4 类（月度轮换 · 每类一年 3 条）

#### 类型 A：知识体系篇（与公众号长文配套）⭐ 主推

> 一篇 1500-2000 字公众号长文 → 一条 5-10 分钟 B 站长视频。
> 这是 B 站最容易拉开差距的一类内容，知识型用户付费意愿最强。

| 选题 | 出处 | 时长 |
|---|---|---|
| **24 节气与传统色：12 个月你的色谱底色** | 公众号长文 + colors.yaml | 8-10 分钟 |
| **「日主即本命色」是错的——调候配色逻辑入门** | 公众号长文 #2 | 6-8 分钟 |
| **《考工记》中的五色与现代审美** | 公众号长文 #3 | 8-10 分钟 |
| **十二个月，每个月最适合穿什么颜色（附 19 色对照）** | 公众号长文 #4 | 10-12 分钟 |
| **传统色背后的矿物颜料：从朱砂到石青** | 公众号长文 #5 | 8-12 分钟 |
| **节气美学的四步审美法** | 公众号长文 #6 | 6-8 分钟 |

> ⚠️ **B 站长视频可适度讲专业概念**（调候 / 月令 / 用神等 L2 词），
> 但**必须放在「学术参考 / 文化考据」框架内**，且每条视频都要有：
> - 开头 5 秒声明："不下结论、不做预测，只做色彩美学创作"
> - 末尾片尾卡："本视频为东方美学艺术参考，非命理预测"
> - 投稿描述中带「#东方传统色」「#色彩文化」「#人文知识」等学术 tag

#### 类型 B：客户案例完整故事

> 一条 8-12 分钟视频，从客户初见 → 信息采集 → 配色推导 → 出图 → 反馈，
> 全程脱敏（用客户编号 + 化名）。

结构参考：

```
开场（30 秒）：黑屏字幕 + 主理人画外音
  "她说她不知道自己为什么穿什么都不好看，直到那天我们做了这件事"
  
信息采集（1-2 分钟）：工作室对谈记录（脱敏镜头：背影/手部）
  客户的故事 + 出生月份 + 当下心愿（求事业 / 求疗愈 / 求安静）

配色推导（2-3 分钟）：白板写画
  "她出生在 12 月 → 红调 → 但她的本命色不是火热的红"
  "结合她的心愿是疗愈期，主色定在了暖白 + 朱砂点睛"
  ⚠️ 这一段是 B 站独有的「展示思路」内容，抖音/视频号上不会做

出图过程（2-3 分钟）：屏幕录制 + 配色调整
  即梦/Midjourney 实操，3-5 版调整

成品交付（1-2 分钟）：壁纸 / 命书 / 朋友圈封面 全套展示

客户反馈（1 分钟）：脱敏文字 + 主理人独白
  "她说她原来真的有一种颜色'是属于她的'"
```

#### 类型 C：主理人 vlog

> B 站观众喜欢看「真实工作流」+「主理人人格」，这一类视频不追求播放量，追求**人格沉淀**。

| 选题 | 时长 |
|---|---|
| 重庆探店：朝天门面料市场买真丝围巾全过程 | 5-8 分钟 |
| 矿物颜料拆箱：第一次试色朱砂 / 石青 / 雌黄 | 6-10 分钟 |
| 工作室探访：易学老师的茶席与古籍架 | 5-8 分钟 |
| 我的一周：从配色推导到成片交付的真实工作流 | 8-12 分钟 |
| 第一本印刷装裱版命书诞生记 | 10-15 分钟 |

#### 类型 D：跨平台内容沉淀（季度 1 条）

> 每季度做一条**「这季度的 12 篇公众号长文 / 30 条短视频精华整合」**——
> 把分散的内容聚合成一条 B 站长视频，作为**月度复盘 + 知识合集**。
> 这一类视频复用度极高，对老用户友好，对新用户是「入门首选」。

### 2.3 短视频（与抖音 / 视频号同发）

详见 [`short-video-script-prompts.md`](./short-video-script-prompts.md) §三。

简言之，三平台**同一支视频**分发，B 站版本相对抖音版有 3 处微调：

| 微调点 | 抖音版 | B 站版 |
|---|---|---|
| 开场 | 0-3 秒强钩子（直接抛冲突）| **多 5-10 秒慢节奏开场**（铺垫场景 / 主理人独白）|
| 字幕密度 | 高（每 2-3 秒一行）| 中（每 4-6 秒一行，留白更多）|
| 末尾 CTA | "评论 1 想要" / "主页接单" | **不放强 CTA**，仅"想了解更多关注我"|
| 配乐 | 节奏强、重低音 | **古风 / 民谣 / 纯音乐** |
| 投稿分区 | 同城 / 美妆 | 生活 → 美妆 / 文化习俗 / 设计创意 |

## 三、关注后 1-3-7-30 天跟进路径

> B 站没有「关注自动回复」机制，但有「**置顶视频** + **粉丝勋章** + **动态**」三个抓手。

### 第 1 天（关注后 24 小时内 · 用「置顶视频」承接）

**置顶视频**：必须是「**渝山·东方美学是什么**」3-5 分钟自我介绍长视频，包含：

- 主理人是谁
- 渝山·东方美学做什么（命主画像 / 节气壁纸 / 命书 三件套）
- 19 色 / 24 节气 / 调候配色逻辑（一句话带过）
- 公众号关键词领 PDF 钩子（"想要 19 色 PDF？关注公众号'渝山东方美学'回复'本命色'领"）
- 不做预测的边界声明
- 末尾："关注我，每月 1 条长视频 + 每周 2-3 条短视频"

### 第 3 天（用「动态」推送一条爆款老视频）

> B 站动态 = 朋友圈式发文区，可以发文字 + 图 + 转发自己的视频。
> 关注后 3 天，新粉丝会回到主页，看到主理人最近发的动态——**这条动态决定他要不要继续关注**。

模板：

```
✦
最近常被问起：
"为什么我穿什么都不好看？"

3 个月前我也是这么想的。
直到我做了这件事 → [转发自己的爆款长视频]

不下结论，不做预测，
只用色彩做美学创作。

—— 主理人 · 渝山
```

### 第 7 天（用「粉丝勋章」+ 第 2 条动态）

- **粉丝勋章名**："**渝山门下**"（不要叫"弟子""门徒"，避免宗教联想）
- 第 2 条动态：客户案例脱敏图 + 主理人手记 200 字

### 第 30 天（推月度长视频）

新粉丝关注 30 天后，主理人正好发布当月长视频——这是**第一次完整观察主理人内容质量**。
这条视频的留存率决定他是否会成为长期粉丝、是否会去公众号领 PDF、是否会预约 1V1 命主画像。

> ⚠️ 这就是为什么**月度 1 条长视频质量绝不能掉**。

## 四、与公众号 / 抖音 / 视频号 / 小红书的协同 SOP

### 4.1 内容流转图

```
[原创主阵地]                    [视频复用]                  [图文搬运]
公众号长文 (周二/周五)  ────→  B 站长视频 (月度 1 条)
                                                          ↘
拍摄/AI 出图素材        ────→  短视频脚本 ──→  抖音/视频号/B站 同发
                                                          ↘
公众号长文截图 + 9 宫格 ────→──────────────→  小红书图文搬运 (12h 内)
```

### 4.2 一周节奏对照

| 时段 | 公众号 | B 站 | 抖音 / 视频号 | 小红书 |
|---|---|---|---|---|
| 周一 | - | 短视频 1 | 短视频 1（同 B 站）| - |
| 周二 | **长文 1**（科普）| - | - | **长文 1 图文版**（12h 内）|
| 周三 | - | 短视频 2 | 短视频 2 | - |
| 周四 | - | - | - | - |
| 周五 | **长文 2**（案例）| - | - | **长文 2 图文版**（12h 内）|
| 周六 | - | 短视频 3 | 短视频 3 | - |
| 周日 | - | - | 主理人 vlog 短视频 | - |
| 月末 | - | **长视频 1**（与本月某条长文配套）| - | - |

### 4.3 长视频 → 公众号长文的反向复用

> 不是所有内容都从公众号 → B 站。**类型 B（客户案例）和类型 C（主理人 vlog）反向走**：

```
B 站客户案例长视频  ────→  剪辑 1500 字解说稿  ────→  公众号长文
B 站主理人 vlog    ────→  剪辑 1200 字图文 vlog ──→  公众号长文（图多）
```

这样一条内容能跑两个阵地，**月度内容产能 = 4 篇公众号长文 + 1 条 B 站长视频 + 12 条短视频**，单兵作战可承担。

### 客户面（视频内容 + 字幕 + 标题 + 简介 + tag）

- [ ] 没出现「微信 / VX / 加我 / 私聊」（B 站简介中**严禁**直接写微信号 / 二维码图片）

## 七、常见问题 FAQ

### Q1：B 站投稿会被算命/玄学限流吗？

会，且和抖音/小红书一样严。B 站对「迷信」有专门的扣分机制，触发后 7-14 天限流。规避方法：

1. 内容定位为**「东方传统色彩文化 / 24 节气美学 / 设计科普」**
2. 客户面绝不出现 L1/L2 禁词
3. 视频开头声明 + 末尾片尾卡都做到
4. 投稿分区选「文化习俗」「设计创意」「美妆」**不选玄学**

### Q2：长视频 5-15 分钟太长，做不出来怎么办？

冷启动期可以从 **4-6 分钟**起步。重点在**密度**而非长度：

- 第 1 个月：4-6 分钟（科普向，密度高）
- 第 3 个月：6-10 分钟（加入客户案例）
- 第 6 个月+：10-15 分钟（多线叙事）

### Q3：B 站没有公众号那种「关注后自动推 PDF」机制，怎么钩子？

通过**置顶视频 + 视频末尾片尾卡 + 动态**三处提示：

```
"想领 19 色 PDF？关注公众号「渝山东方美学」回复"本命色"领"
```

→ 这一步**故意走公众号**，让 B 站做信任建立 / 公众号做私域沉淀。

### Q4：B 站观众和抖音观众真的不一样吗？

是的，差异极大：

| 维度 | 抖音观众 | B 站观众 |
|---|---|---|
| 期待 | 视觉冲击 + 情绪 | 知识 + 长留存 |
| 容忍长度 | 5-30 秒 | 5-30 分钟 |
| 容忍专业度 | 低（看不懂就划走）| 高（看不懂会查资料）|
| 对「专业派」的反应 | 中性 | **正向**（B 站用户更买账）|

→ 这就是**为什么 B 站长视频可以适度讲专业概念**——B 站是主理人「专业派」叙事最容易跑通的平台。

### Q5：B 站可以推荐微信加好友吗？

**严禁**——和抖音/视频号/小红书一样，B 站简介中直接写微信号或贴二维码会被立即限流。

推荐路径：

```
B 站简介 → "公众号「渝山东方美学」回复'本命色'领 PDF"
```

### Q6：算命党在评论区刷「能帮我看运势吗 / 八字怎么看」怎么办？

参考 [`prompts-content.md`](./prompts-content.md) Prompt 6 标准转向话术，B 站独有版本：

```
谢谢提问～
我们这里**不下结论、不做预测**，只用色彩做美学创作。
如果你对自己的「本命色」感兴趣，
关注公众号「渝山东方美学」回复"本命色"
有一份 19 色 PDF 可以送你 ✦
```

### Q7：B 站长视频要不要开「充电专属」？

**前 3 个月不开**。理由：

- B 站「充电专属」会被部分用户解读为「玄学课程付费」，触发误举报概率高
- 早期目标是建立信任 + 公众号引流，不是 B 站直接变现
- 真正变现路径是 B 站观众 → 公众号 → 1V1 命主画像（299-2999）

> 第 4 月+ 累计 1w 粉后，可以考虑开「充电专属」放置「**24 节气美学合集**」（无玄学元素）这一类作为辅助变现。
