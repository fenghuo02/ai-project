
```

❌ 卡通元素 / Q 版 / 萌化
❌ 现代感几何 / 圆角矩形按钮风格
❌ 西方插画 / Adobe Stock 商务风
❌ 占卜符号 / 八卦图 / 易经卦象
❌ 任何文字（让 AI 生图，文字交给后期叠加）
❌ 真实人脸（除非客户授权，且只做背景替换）
```

## 图 1 · 头条主图（最重要 ⭐）

### 用途规格

- **公众号头条主图**：900 × 383 px（公众号自动裁剪比例 16:6.8）
- **公众号缩略图**（自动生成）：500 × 500 px（取主图中心方形区域）
- **文件大小**：≤ 2 MB

### 设计意图

这张图是**整篇文章的 0-3 秒钩子**，决定了打开率。它要做到：

- 一眼能看出"东方美学"
- 一眼能看出"色彩"主题
- 有"重庆"的地域感（雾 + 朱漆门 + 山城）
- 留白处方便后期叠"重庆姑娘的本命色 / 你是哪一抹"标题

### 完整 prompt（中文版 · 直接复制粘贴）

```
【母版风格】+

构图：俯视微仰角度，重庆山城清晨的一扇朱漆木门特写，
门半开，门内透出柔和的暖光。

主体：
- 一只手（女性手腕，无人脸）从门内伸出，手中捧着一卷展开的色卡纸轴
- 色卡纸轴上展示约 8-10 个东方传统色色块，由暖驼到朱砂的色相过渡
- 手腕上一根细朱砂红色绳，腕骨处一颗小朱砂珠
- 手指优雅地捏着色卡边缘

环境：
- 朱漆门 #C0392B 占画面右侧 1/3，木质质感，门钉有细微岁月痕迹
- 门外是清晨重庆的薄雾（透明米白 #FAF7F2 雾气），远处隐约看到山城轮廓
- 门内透出暖驼色 #C9A66B 暖光，模糊的木质茶席背景
- 门头上方有极淡的「福」字朱漆痕迹（不要清晰可读，意象化）

光线：
- 冬日清晨的阳光从画面左上方斜射
- 朱砂门头被照得发亮
- 手部和色卡上有柔和暖光
- 整体有"破晓"的仪式感

色调：
- 主色：朱砂红 #C0392B + 米白 #FAF7F2 + 暖驼 #C9A66B
- 点缀：烛光金 #D4AF37（门钉 / 朱砂珠的反光）
- 背景雾色低饱和度，前景手 + 色卡饱和度稍高（视觉焦点）

构图细节：
- 画面下半部留白（约 30% 米白雾气区域）方便后期叠标题文字
- 不要任何文字
- 不要全身人像（只露一只手 + 部分手腕）
- 16:7 横幅比例（适配公众号头条 900x383）

调性：
- 像观夏品牌的产品摄影
- 像故宫文创的封面图
- 像宋画《捣练图》局部
- 仪式感 + 东方留白美学
- NOT 节日喜庆 / NOT 民俗大红
```

### 失败重跑调整

| 错误现象 | prompt 末尾追加 |
|---|---|
| 出来全身人物 | "Show ONLY a hand and wrist, NO face, NO full body, NO portrait" |
| 朱砂偏粉红 | "Deep vermillion red like Chinese mineral cinnabar 朱砂, NOT pink, NOT magenta" |
| 太喜庆 / 像春联 | "NOT festive, NOT Spring Festival, NOT folk art. Style of MUJI doing Chinese aesthetic" |
| 没有留白 | "Generous negative space in lower half for title overlay, minimum 30% empty area" |
| 添加了文字 | "Absolutely no text, no characters, no calligraphy in the image" |

### 后期处理

得到 PNG 后：
1. 用美图秀秀/Canva 在画面下方留白处叠加标题：
   ```
   重庆姑娘的本命色
   你是哪一抹
   ```
   字体：思源宋体 / 宋楷，墨青色 #2C3E50，字号 60-80pt
2. 右下角小字加 "AI 辅助创作 · AIGC"（10pt 半透明白）

## 图 2 · 19 色全家桶（第 2 节末）

### 用途规格

- **正文配图**：720 × 720 px（小红书也能复用）
- 文章上下文："我把它们装进一份 38 页的 PDF 里" 之后

### 设计意图

让读者**视觉上感受到** 19 个颜色名字的丰富——不是抽象概念，而是 19 个有质感的色卡。

### 完整 prompt（中文版）

```
【母版风格】+

构图：俯视 90 度平拍，米白色宣纸桌面上整齐铺开的 19 张矿物色卡，
组成一个完整的色彩矩阵。

主体：
- 19 张矩形色卡（约 3cm × 4cm），按色相过渡分组排列，
  分成 3-4 行，每行 5-6 张色卡
- 每张色卡有微妙的矿物颜料颗粒质感（不要纯色块）
- 色卡间留有 0.5cm 的米白色宣纸空隙
- 每张色卡边缘有自然的轻微不规则（手切纸感）

色彩顺序（按色相）：
- 第一行（红橘系）：朱砂 #C0392B → 粉霞 #F0A6A6 → 朱橘 #E67E22 →
  赭石 #A0522D → 暮褐 #6B4423
- 第二行（金驼系）：暖驼 #C9A66B → 烛光金 #D4AF37 → 菊黄 #F4D03F →
  米麦 #F5E6C8
- 第三行（白银系）：米白 #FAF7F2 → 月光银 #E8E6E1
- 第四行（绿系）：苍翠 #2D5F3F → 深竹 #4A6B4F → 嫩绿 #8FB573 →
  青翠 #6FA88A
- 第五行（蓝紫系）：墨青 #2C3E50 → 雾蓝 #5D7B91 → 冷银 #A8B5C7 →
  紫雾 #8B6F8E

背景：
- 米白色 #FAF7F2 宣纸桌面，有极淡的纤维纹理
- 桌面右上角放置一支羊毫小楷毛笔（点缀，虚化）
- 桌面左下角放一方小朱砂方印（指甲盖大小，虚化）

光线：
- 自然光从画面左上方柔和洒下
- 色卡有极淡的右下投影（薄薄一层）
- 整体明亮但不刺眼

调性：
- 像 Pantone 色卡盒打开后的样子
- 但更"东方"——是宋代矿物颜料桌面，不是现代印刷品
- 干净、有秩序、克制

构图：
- 1:1 正方形构图
- 19 张色卡占画面 75%，居中
- 上下各留 12.5% 空白

NOT：
- 不要文字（HEX/色名后期叠）
- 不要英文标签
- 不要圆形色卡（必须是矩形）
- 不要电脑屏幕上的色卡 UI（要实物质感）
```

### 后期处理

每张色卡下方可以叠加微小色名 + HEX（用 Photoshop / Canva）：
```
朱砂  #C0392B
朱橘  #E67E22
... (19 个)
```
字号 8pt，墨青色，宋体。

## 图 3 · W 女士案例示意图（第 4 节末）

### 用途规格

- **正文配图**：800 × 1000 px（4:5 竖版，公众号正文最佳比例）
- 文章上下文："她公司的同事来问她最近是不是涨薪了" 之后

### 设计意图

让读者**视觉化感受到** W 女士的"换色前 vs 换色后"对比，但**不能出现具体人脸**（合规底线 + 隐私）。

### 完整 prompt（中文版）

```
【母版风格】+

构图：拼图式对比构图，左右分屏（左：换色前 / 右：换色后），
中间用极细朱砂红线条分隔。

左侧 (换色前)：
- 一身黑白冷调的女性背影或半身（不显示面部）
- 黑色西装外套 + 冷银色丝巾
- 背景是冷调办公室（淡蓝灰墙面）
- 整体色调：冷银 #A8B5C7 + 墨黑 + 冷白
- 氛围：克制、距离感、有一点点"接不住"的疲态

右侧 (换色后)：
- 同一人侧身或半身（同样不显示面部，重点在服饰与气场）
- 朱橘色 #E67E22 长大衣 + 暖驼色 #C9A66B 围巾
- 胸前别一枚朱砂方印 #C0392B 小别针（特写焦点）
- 背景是温暖的木质茶席空间，柔和暖光
- 整体色调：朱砂 + 暖驼 + 米白
- 氛围：温润、亲和、整个人"亮起来"了

中间分隔：
- 极细朱砂红 #C0392B 竖线
- 线上有极小印章「→」（暗示"转变"）

光线：
- 左侧：冷调冷光
- 右侧：暖调暖光
- 形成强烈对比

整体：
- 4:5 竖版构图
- 像中信出版社的对比插画
- 像 Kinfolk 杂志的图片调性
- 不要任何文字
- 不要面部细节（关键合规底线）
```

### 后期处理

图下方公众号正文叠加图注：

```
[ 案例 W 女士 · 1986 立秋后生 · 已脱敏 ]
左：原 · 黑白冷调
右：推荐 · 朱橘 + 朱砂方印
图为示意 · AI 辅助创作 · AIGC
```

## 图 4 · 朱砂介绍样板页（第 6 节末）

### 用途规格

- **正文配图**：720 × 1080 px（2:3 竖版，模仿 PDF 内页）
- 文章上下文："总共 38 页，免费送" 之后，引导关注公众号

### 设计意图

让读者**直观看到** 38 页 PDF 长什么样——这是引流转化的"信任锚点"。

### 完整 prompt（中文版）

```
【母版风格】+

构图：模拟一本翻开的精装手册，俯视 75 度角，左右两页可见。

主体：一本 A4 大小的精装手册，封面是米白色 #FAF7F2 的宣纸质感，
书脊有暖驼色 #C9A66B 烫金细线条，整本书摊开 90 度，露出里面的内页。

左页：
- 顶部居中：一个超大的「No.1」字样（深朱砂红 #C0392B，宋体，约页面高 15%）
- 中部：一个朱砂红 #C0392B 的大色块（占左页 60%），有矿物颜料颗粒质感
- 色块右下角：「朱砂」两个大字（宋楷，深红色 #2C3E50）
- 底部：一行小字「Vermillion · #C0392B · 火 · 红」（Garamond 斜体）

右页：
- 顶部：一段诗句般的小段落（4 行）—— 模糊不清晰，
  暗示有内容但不喧宾夺主（关键：不要让 AI 试图"写出真实文字"）
- 中部：一组小色卡（5-6 个朱砂相关的渐变色），整齐排列
- 底部：一方小朱砂方印「渝山」（约 2cm 见方）+ 页码「P. 04」

环境：
- 手册放在米白宣纸桌面上
- 桌面左侧露出一根毛笔尾端 + 一方小石头镇纸（虚化）
- 桌面右侧有一个朱砂红色印泥盒（虚化）
- 整体光线：自然侧光从画面左上洒下

光线：
- 翻开页面有极淡阴影（书的厚度感）
- 朱砂色块在光下有真实的颗粒感反光
- 暖驼烫金线在书脊处反光

整体：
- 2:3 竖版构图
- 像 Kinfolk Magazine 内页
- 像观夏产品手册
- 像高级茶馆的茶谱
- 不要任何 AI 生成的清晰文字（除了 "No.1" 和 "朱砂"）
- 整体留白克制，高级感
```

### 后期处理

图片本身已经够干净，**只需在正文图注里写**：

```
图为示意 · 实物 PDF 共 38 页 · 后台回复「本命色」自取
```

## 推荐生成顺序

```
Step 1 ⭐ 先跑图 1（主图，决定点击率）
  → Gemini imagen-3 跑 4 张
  → 选最有"东方美学"调性的 1 张
  → 后期叠标题文字

Step 2 跑图 2（19 色全家桶）
  → 这张相对容易（没有人物）
  → 跑 4 张取最整齐的

Step 3 跑图 3（W 女士对比图）
  → 难度中等（左右对比）
  → 跑 4 张取明暗对比最强烈的

Step 4 跑图 4（PDF 内页样）
  → 跑 4 张取模拟书本最逼真的
```

**总工时**：约 30-40 分钟（含 4 张选优）

## Gemini 特别提示

### 1. 用 imagen-3 模式

Gemini 普通对话窗口出图质量一般，**imagen-3** 是专业模式：

```
请用 imagen-3 模式生成这张图，prompt 如下：
[贴上面的 prompt]
```

### 2. 中文字渲染坑（图 4 特别注意）

Gemini 对中文渲染**容易出错**，prompt 末尾**强调**：

```
Critical: The Chinese characters MUST be exactly:
- 「No.1」 (English numeral, easy)
- 「朱砂」 (two characters: 朱 with cross shape, 砂 with stone radical 石)
- 「渝山」 (already covered in seal)

DO NOT generate any other Chinese characters, especially in the
indistinct text paragraph (keep that area as suggestive blur).
```

### 3. 失败重跑统一加这句

如果出图调性偏现代/西方插画：

```
Style reference: 观夏 (TO SUMMER) brand campaign visuals,
闻献 (Documents) brand minimalism, 故宫文创 product imagery,
Wallpaper magazine East Asia issue.
NOT Adobe Stock, NOT modern flat design, NOT Western illustration.
```
