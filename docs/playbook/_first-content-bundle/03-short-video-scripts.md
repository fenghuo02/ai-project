# 短视频分镜脚本 · 三平台同发

> 平台：抖音 + 视频号 + B 站
> 主题：「测一测：你出生那个月，配哪一抹色？」
> 一支拍摄 → 三平台剪三版（抖音 30-45s / 视频号 45-60s / B 站 60-90s）
> 引流路径：视频末尾 → 评论区/简介 → 公众号「渝山东方美学」回复「本命色」

---

## 一、共用素材清单（拍 1 次）

### 拍摄场地
- 主理人工作室 · 茶席一角 + 靠窗自然光
- 背景：米白宣纸墙 + 一角朱漆门头 / 木茶桌

### 道具（按本命色 19 色全集准备）
- 19 色色卡卡纸（按 colors.yaml HEX 印刷或染色）
- 一支羊毫毛笔 + 一方朱砂方印「渝山」
- 一张本命色卡 PDF 打印件（演示用）
- 主理人本人服装：暖驼色羊绒围巾 + 米白长袍内搭

### 录制核心镜头（拍 1 次用 5-10 分钟）

| # | 镜头 | 时长 | 用途 |
|---|---|---|---|
| A | 主理人手部翻 19 色色卡（特写）| 8s | 视觉钩子 · 各版都用 |
| B | 主理人正面坐拍 · 说话头肩景 | 25s | B 站 + 视频号长版 |
| C | 朱漆门头特写 + 朱砂落印 | 5s | 三平台都用 |
| D | 19 色全家桶 + 标价手写 | 5s | 三平台都用 |
| E | PDF 翻页特写（手指按住）| 5s | B 站长版 + 视频号 |
| F | 主理人侧脸笑着说"评论区告诉我你的生月" | 6s | 三平台都用 · CTA |
| G | 工作室方印特写「渝山」| 3s | 三平台落款 |

## 二、抖音版（30-45s · 短平快 · 0-3s 强钩子）

### 文案完整版（35s）

```
[ 0-3s · 镜头 A ]
镜头：手部快速翻 19 色色卡
画面叠字：「你今天穿的颜色，是你的吗？」
配音（主理人 OS）：
"你今天穿的颜色，是你的吗？"

[ 3-10s · 镜头 D ]
镜头：19 色全家桶平铺，主理人手指逐一点过
画面叠字：「19 抹东方传统色 · 朱砂 · 苍翠 · 暖驼 · 墨青」
配音：
"东方传统色谱里有 19 个颜色，
每一抹，都对应你出生那个月、那一刻的节气。"

[ 10-22s · 镜头 B（说话头肩景）]
镜头：主理人正面，自然光从左前方
画面叠字（小字、底部安全区）：「立春嫩绿 · 立秋暖驼 · 冬至朱砂」
配音：
"我做了一份 38 页的 PDF。
立春出生的人骨子里带嫩绿，
立秋的姑娘骨子里带暖驼，
冬至生的人骨子里带朱砂。
你穿对的时候，气场顺；
穿错的时候，看着像哪里有点疲惫。"

[ 22-30s · 镜头 C + E ]
镜头快切：朱漆门特写 → PDF 翻页 → 朱砂落印
画面叠字：「38 页 PDF · 关注公众号回复『本命色』」
配音：
"想知道你的那一抹？
关注我，公众号回复『本命色』，自取。"

[ 30-35s · 镜头 F · CTA ]
镜头：主理人侧脸笑
画面叠字：「评论区告诉我你的生月」
配音：
"评论区告诉我你的生月，
我告诉你你是哪一抹 ✨"
```

### 抖音版调性约束
- **0-3s 必须有强钩子**：本版用反问 + 视觉冲击（19 色色卡飞速翻动）
- **配音节奏快**：1.2 倍速，不留气口
- **画面叠字**：每屏不超过 12 字
- **结尾强 CTA**：必须有"关注 + 评论生月"双指令
- **背景音乐**：古风 instrumental（推荐：《琵琶语》纯音乐版抖音号#古风BGM 库）

### 抖音 tag（评论区/标题）

```
#东方美学 #传统色 #本命色 #重庆同城 #色彩搭配 #国风
#新中式 #节气 #色卡 #穿搭灵感
```

### 抖音视频简介

```
38 页《19 色本命色 PDF》全集，
关注 → 公众号「渝山东方美学」回复"本命色"自取。
评论区告诉我你的生月 ✨
```

## 三、视频号版（45-60s · 温和 · 朋友圈口吻）

### 文案完整版（55s）

```
[ 0-5s · 镜头 A 慢节奏 ]
镜头：手部慢速翻 19 色色卡（比抖音慢 30%）
画面叠字：「你今天穿的颜色 / 是属于你的色吗？」
配音（主理人 OS · 平和语速）：
"你好呀。你今天穿的颜色，
是属于你的色吗？"

[ 5-15s · 镜头 D + B ]
镜头：19 色全家桶平铺 → 切主理人正面

  「东方传统色谱里有 19 个名字」
  「朱砂、苍翠、暖驼、墨青...」
  「每一个都和节气有关」

朱砂、苍翠、暖驼、墨青——
每一个都和节气有关，
和你出生那个月、那一刻的天地之气有关。"

[ 15-30s · 镜头 B（继续说话）]
镜头：主理人坐姿 · 端起茶盏轻啜一口
画面：无叠字，只有人物 + 自然光
配音：
"立春前后出生的人，骨子里带嫩绿；
立秋前后出生的姑娘，骨子里带暖驼；
冬至那几天来到这世上的，骨子里带朱砂。

我陪重庆十几位朋友找过他们的那一抹色，
他们说，穿对色之后，
出门的状态都不一样了。"

[ 30-42s · 镜头 E + C ]
镜头慢切：PDF 翻页特写 → 朱漆门头 → 朱砂落印「渝山」
画面叠字：「38 页 PDF · 全集免费送」

12 月色卡日历、
还有按生月找到你那一抹的速查表。"

[ 42-55s · 镜头 F · CTA + 镜头 G ]
镜头：主理人微笑面对镜头 → 切方印特写
画面叠字：「关注公众号 → 回复『本命色』自取」
配音：
"如果你想要一份，
关注公众号「渝山东方美学」，
后台回复『本命色』，自动发给你。

评论区告诉我你的生月，
我告诉你你是哪一抹 ✨"

[ 末帧 0.5s · 渝山方印 logo + 公众号二维码 ]
```

### 视频号版调性约束
- **不放强 CTA / 强引流弹窗**（视频号被点赞的概率比强营销高 2-3 倍）
- **朋友圈口吻**：开头"你好呀"，节奏比抖音慢 30%
- **视觉**：长镜头为主（每个镜头 ≥ 5s），少快切
- **背景音乐**：极轻的茶馆白噪声 + 低音古琴，**不要古风燃曲**
- **末帧设计**：右下角放公众号二维码（视频号允许）+ 渝山方印

### 视频号文案（朋友圈式分享文）

```
你好呀。整理了一份 38 页的《19 色 · 重庆人的本命色卡》PDF，
不看流行色，不做个人色彩诊断。
只问一件事：你出生那个月，骨子里带哪一抹色。

关注 → 公众号「渝山东方美学」回复「本命色」自取。
评论里告诉我你的生月，我告诉你 ❤️
```

## 四、B 站版（60-90s · 慢节奏 · 学术框架）

### 文案完整版（75s）

```
[ 0-8s · 镜头 A 极慢节奏 + 镜头 G ]
镜头：手部慢慢拈起 19 色色卡 → 朱砂方印特写「渝山」
画面叠字（章节式标题）：

东方传统色谱、节气、

[ 8-20s · 镜头 D + B（说话头肩景）]

苍翠、深竹、嫩绿、青翠，

暖驼，是秋日傍晚七点麦田被染上的颜色；

[ 35-55s · 镜头 B + E（PDF 翻页穿插）]
镜头：主理人坐姿讲解 → 切 PDF 翻页特写

立春前后出生的人，骨子里带嫩绿；

镜头：朱漆门头慢推 → 主理人讲解 → 一张 W 女士案例图（脱敏）
画面叠字：
  「客户案例 · 1986 年立秋后出生」
  「原: 黑白冷调 → 推: 朱砂主调」
配音：

她试着把通勤的西装换成朱橘色长大衣，

[ 70-75s · 镜头 G · 落款 + 引流 ]
镜头：朱砂方印「渝山」特写 + 渐隐
画面叠字：
  「关注 + 私信『本命色』可获取完整 PDF」
  「下期：12 月色卡日历的来源」
```

### B 站版调性约束
- **节奏慢 50%** vs 抖音
- **学术调**：开头报"第几期" + "色谱来源"具体引用
- **引用宋代典籍**：建立专业人设
- **客户案例脱敏讲述**：增加可信度
- **CTA 弱化**：B 站不能有"关注、点赞、投币"明示，只能用"我们下期见"
- **背景音乐**：古琴独奏（推荐：《幽兰》《广陵散》节选）
- **片头片尾**：建议加 2s 渝山 logo 动画（HTML 模板可截图）

| 项 | 内容 |
|---|---|
| 分区 | 时尚 > 美妆 OR 知识 > 人文历史（建议**时尚分区流量大**）|
| 标签 | `东方美学` `传统色` `中式色彩` `节气` `穿搭灵感` `重庆` `色彩学` |
| 简介 | "渝山东方美学 · 第 1 期 · 我用了 3 年整理东方传统色谱里 19 个颜色名字。关注我，私信回复『本命色』，38 页 PDF 完整版送你 🪶" |

## 七、合规自查（每平台发布前都跑一次）

### 通用 checklist

- [ ] 0 处禁词（算命/看相/八字/用神/调候/命格/奇门/八门/死门/惊门/转运/改运/改命/开运）
- [ ] 0 处微信 / VX / 加我（引流必须经公众号）
- [ ] 三件套对外命名一致：命主画像 / 节气壁纸 / 命书
- [ ] AIGC 标识：视频右下角全程显示"AI 辅助创作 · AIGC"≥10pt 半透白字
  - 即使是真人实拍，也建议加（PDF/壁纸里有 AI 生成内容）
- [ ] 末帧"不下结论 / 不做预测"非承诺声明（B 站必加，抖音可省）

### 各平台特殊红线

| 平台 | 红线 | 触发后果 |
|---|---|---|
| 抖音 | "测一测"标题谨慎用（曾被判玄学测试，限流）| 限流 |
| 视频号 | 不要外链跳第三方 → 用公众号关注 + 后台回复 | 流量打折 |
| B 站 | 不要在分区"占卜玄学"投稿，必须用"时尚 / 知识" | 推荐池小 |
