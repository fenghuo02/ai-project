---

## 设计原则

1. **5 个案例对应 5 类常见命局** —— 让不同读者都能"找到自己"
2. **每个案例都是真实命理推导** —— 不是编故事，是用项目的命理逻辑推
3. **客户姓名脱敏到首字 + 生年** —— W 女士 / Z 先生 / L 老师 / S 小姐 / Y 先生
4. **每位"3 个月后说的那句话"是核心金句** —— 容易引发共鸣 + 朋友圈传播
5. **末页强 CTA** —— 引导回复"预约"

---

## 客户面 vs 内部字段（合规底线）

⚠️ **PDF 排版时严格区分**：

| 字段类型 | 包含什么 | 进入 PDF？ |
|---|---|---|
| **客户面**（P.01-13）| 故事 / 色 / 金句 / 印章 | ✅ 进 PDF |
| **内部命理推导核验**（本文档下半部）| 八字 / 用神 / 喜忌 / 强弱 | ❌ 不进 PDF（仅供主理人 + 老师内部参考）|

PDF 客户面**绝不出现**：八字 / 用神 / 调候 / 命格 / 算命 / 看相 / 转运 / 改运。
**只用对外说法**：命局 / 本命色 / 应景之色 / 避忌色 / 那一抹色。

## 命理推导逐条核验（内部）

### W 女士（已在 PR #31 详细推导）
日主己土秋金 → 偏弱 → 用神丙丁火 → 朱砂朱橘
**项目已有 yaml**：`mingshu-builder/clients/W女士-1986/`

### Z 先生（与主理人 1988-08-08 同类）
- 八字：乙丑 / 甲申 / 壬午 / 戊申
- 日主壬水 + 月令申（金生水，水更旺）
- 强弱：偏强（金生 + 比劫多）
- 用神：戊土（堤岸）+ 丁火（晚霞）
- 喜神：火土
- 忌神：金水
- → 主色暖驼（戊土）+ 辅色烛光金（丁火）
- → 避忌冷银（金）

> 这个推导**和主理人一致**——说明 demo 案例可以引用主理人本人的色，作为亲身证明。

### L 老师（甲木春月）
- 八字：戊午 / 甲寅 / 甲辰 / 丙寅
- 日主甲木 + 月令寅 + 年支午
- 强弱：偏强（木旺 + 火助）
- 但木旺需要的是**继续旺**而不是"被克"
- 用神：甲木自身（比劫）→ 苍翠
- 不用泄秀（本来火够）
- 忌神：金（克木）+ 水浇火（月白色让水旺）
- → 主色苍翠 + 辅色嫩绿

### S 小姐（丁火初夏 · 反直觉案例）
- 八字：壬申 / 乙巳 / 丁丑 / 庚子
- 日主丁火 + 月令巳（火本旺）
- 但年支申、时支子（申子半合水），日支丑（湿土）
- 强弱：偏弱（被周围水土耗掉）
- 用神：木火（生火扶身）
- 喜神：朱橘、粉霞（扶丁火的红橘色调）
- 忌神：墨青（水克火 + 全黑耗火）

> 这个案例**反直觉**——很多人以为"夏天生 = 火旺 = 应该穿冷色"，
> 但实际命局推下来，**周围水太多反而把丁火耗掉**，需要补火。
> 体现"我们看的是整个命局，不是月份"的专业感。

### Y 先生（庚金秋月）
- 八字：乙卯 / 乙酉 / 庚申 / 戊寅
- 日主庚金 + 月令酉（金旺）
- 强弱：偏强（秋金本旺 + 戊土生金）
- 用神：泄秀，但用水还是火？
- 时支寅含丙火，但寅木被酉金冲（木受伤）
- 实际推：用水来泄秀（金生水）
- 但避忌过度强烈的克色（朱橘强火克金）
- → 主色月白 + 辅色月光银（白色泄金气）
- → 避忌朱橘（强烈火克金）
