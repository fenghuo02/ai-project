# 短视频分镜参考图 · AI 生成 Prompt 包

> 配合 `03-short-video-scripts.md` 使用 · 给摄影师/剪辑师看的"目标画面参考"
> 推荐工具：Gemini Imagen 3 > 即梦 > 通义万相
> 创建：2026-05-17

---

## 用途说明

这些 prompt 生成的是**分镜参考图（storyboard）**——

- ✅ **视频本体强烈建议实拍**（主理人真人出镜 + 真实道具，转化率比 AI 视频高 5-10 倍）
- ✅ AI 图作用：**给摄影师 / 自己拍摄前看一眼"目标画面长这样"**
- ✅ 也可以**做视频片头/片尾的过渡静帧**
- ✅ 也可以**做朋友圈预热海报**（视频发布前 12 小时）
- ❌ **不建议** 把 AI 静帧当视频内容主体（缺乏真实感，掉粉）

## 镜头 A · 视觉钩子（0-3s 必出钩子）⭐ 最重要

### 视频里出现位置

- 抖音：0-3s（决定完播率）
- 视频号：0-5s
- B 站：0-8s

### 完整 prompt（中文版）

```
【母版风格】+

构图：极特写镜头，俯视 80 度角拍摄一双女性手部（无人脸）正在
快速翻动 19 张矿物色卡。

主体：
- 双手特写（仅手部和小臂，皮肤细腻自然，无指甲油）
- 19 张约 6cm × 4cm 的矩形色卡，按色相过渡
  从朱砂 #C0392B → 朱橘 #E67E22 → 暖驼 #C9A66B
  → 烛光金 #D4AF37 → 米白 #FAF7F2 → 苍翠 #2D5F3F
  → 墨青 #2C3E50（约 8-10 张可见在画面中）
- 手指优雅地拈着色卡边缘
- 色卡正在被翻动，有 1-2 张色卡在画面中"飞起"的动感（运动模糊）
- 色卡有矿物颜料颗粒质感，不要纯色块
- 手腕一根细朱砂色绳，腕骨处一颗小朱砂珠

环境：
- 米白色 #FAF7F2 宣纸桌面，有极淡纤维纹理
- 桌面右侧虚化的一支羊毫毛笔
- 桌面左侧虚化的一方朱砂方印

光线：
- 自然光从画面右上方斜射
- 色卡有柔和的右下投影
- 整体明亮但不刺眼

色调：朱砂主调 + 米白底 + 暖驼点缀

构图：
- 9:16 竖版构图
- 双手占画面下半部约 50%
- 上半部留白（视频字幕区，约 30%）
- 不要任何文字
- NO face, NO body, only hands visible

调性：
- 像观夏品牌的产品宣传短片首帧
- 像 Kinfolk 视频开场
- 强烈的"打开、揭示"感
- 视觉冲击力 + 仪式感

NOT：
- 不要塑料色卡（必须是宣纸 / 矿物质感）
- 不要现代设计感的色卡（不要 Pantone 那种工业感）
- 不要花哨的指甲油
```

### 静帧 vs 动态

这张图作为**静帧分镜参考**——实际拍摄时让手缓慢翻动色卡，剪映后期加 1.2 倍速 + 镜头微推近，做出"飞速翻动"的动感。

- 一位中国女性，30-40 岁，气质温润内秀

- 颈间一枚朱砂色 #C0392B 圆形玉佩
- 自然微笑，眼神平和坚定，正对镜头
- 不浓妆，妆容自然清淡（强调气质而非美貌）

环境：
- 工作室茶席背景：木质茶桌 + 一盏白瓷茶杯（茶汤暖色）
- 背景墙：米白色 #FAF7F2 宣纸墙，有极淡的水墨晕染
- 墙上挂一幅小型水墨画（远景虚化）
- 右后方一支带红色枝条的小盆景（朱砂色点缀）

光线：
- 柔和自然光从画面左前方洒下
- 主理人面部有自然的明暗过渡
- 背景比主理人暗 1-2 档（突出主体）

色调：米白主色 + 暖驼次色 + 朱砂点缀 + 墨青阴影

构图：
- 9:16 竖版
- 主理人占画面下 2/3
- 上 1/3 留白（视频字幕区）

- 不要任何文字

调性：
- 像观夏代言人的肖像

## 推荐生成顺序

```
Step 1 ⭐ 镜头 A（视觉钩子，最关键）
  → Gemini imagen-3 跑 4 张
  → 选最有"翻动感"的 1 张
  → 给摄影师/自己看，作为拍摄参考

Step 2 镜头 D（19 色全家桶）
  → 这张相对容易（无人物）
  → 跑 4 张选最整齐的
  → 实拍时按这个布局摆色卡

Step 3 镜头 G（落款印章）
  → 极简，跑 2 张就够
  → 用作每条视频的结尾静帧

Step 4 镜头 C / E（场景特写）
  → 让摄影师看后能照着搭场景
  → 可不实际打印参考图，文档里描述够清楚

Step 5（选做）镜头 B / F（主理人正面 / 侧脸）
  → AI 出来的人不是真主理人
  → 仅作为"姿势 + 光线 + 服饰"参考
  → 不要直接当视频内容用
```

**总工时**：30-40 分钟跑完全部参考图

## 跑出来怎么用

### 用法 A · 给摄影师看（最常见）

把 7 张 AI 参考图打印或发给摄影师 →

```
"我想要这种感觉的画面 ×7
- 镜头 A 这种翻色卡的特写
- 镜头 D 这种 19 色平铺
- 镜头 G 这种落印特写
- 其他 4 个按文件里的描述"
```

摄影师看到画面后**对气质 / 光线 / 构图**有概念，拍出来才不会跑偏。

### 用法 B · 视频片头/片尾过渡静帧

剪辑时在两个真人镜头之间插入 1-2 秒 AI 静帧（如镜头 G 落印），增加节奏感。

### 用法 C · 朋友圈预热海报

视频发布前 12 小时，用镜头 A 静帧 + 文字 "今晚 19:00 视频上线" 发朋友圈。

## Gemini 特别提示

### 1. 9:16 竖版的特殊要求

Gemini imagen-3 默认是 16:9 横版。生成竖版必须**显式指定**：

```
aspect ratio: 9:16, vertical mobile portrait orientation,
NOT 16:9, NOT square, NOT 4:3
```

如果 Gemini 仍然给横版，**重跑加这句**：

```
The image MUST be rendered in PORTRAIT orientation
(taller than wide), aspect ratio 9 : 16.
This is for short-video platform (TikTok / Bilibili Vertical / WeChat Channels).
```

### 2. 中文字渲染坑（仅镜头 G 印章 + 镜头 E PDF）

镜头 G 的「渝山」必须正确，加这句：

```
Critical: The Chinese characters「渝」(yú, meaning Chongqing,
must include water radical 氵 on the LEFT) and「山」(shān,
meaning mountain). Reference Qing dynasty seal carving.
DO NOT write 涂 / 俞 / 油 / 川 / 出.
```

### 3. 失败重跑万能修正

如果 Gemini 总把 19 色全家桶画成"色彩斑斓的现代设计"：

```
Reference: traditional Chinese mineral pigments laid out on
xuan paper, like a Song dynasty painter's color palette.
NOT modern Pantone swatches, NOT digital color picker UI,
NOT graphic design swatches.
```

如果场景太"喜庆 / 节日"：

```
Calm, restrained, MUJI-style minimalist Chinese aesthetic.
NOT festive, NOT Spring Festival decorations, NOT folk art.
The mood is meditative, contemplative, soft.
```
