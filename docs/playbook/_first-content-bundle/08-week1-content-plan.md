## 双篇协同机制

### 引流路径设计（第一篇 → 第二篇）

```
读者读第一篇文章「19 色 PDF」
    ↓
关注公众号 → 回「本命色」拿 PDF
    ↓
读者第二天看到第二篇「W 女士的故事」
    ↓
"我也想找到我的色" → 回「案例」看更多 → 回「预约」工作室
    ↓
转化为¥1,800-¥6,800 工作室客户
```

### 关键词自动回复（累计 5 条 + 1 条新 + W2 新增 1 条 ⭐）

| 关键词 | 第几篇引入 | 回复内容 |
|---|---|---|
| `本命色` | 第一篇 | 19 色 PDF 链接 + 引导评论生月 |
| `案例` ⭐ NEW | **第二篇** | 5 个客户故事 PDF + 引导预约 |
| `预约` ⭐ NEW | **第二篇** | 预约填写规范 + 一周回复一次 |
| `重庆` | 第一篇 | 工作室地址（脱敏） |
| `合作` | 第一篇 | B 端 BD 联系方式 |
| `地址` ⭐ NEW | **第二篇** | 完整工作室地址（仅在确认预约后给）|
| **`节气`** ⭐⭐ W2 NEW | **第三篇（W2 周二）** | "小满节气壁纸 5/21 当天 09:00 群发推送 🌿 包含锁屏麦田 / 手机青麦 / 电脑桌面 3 张 4K 高清。如果在 5/21 之前来的，我们会在小满当天首发时单独 push 你一次。" |

### 评论区联动话术

第一篇后台 200 条评论 → 主理人手动逐一回复 → 引导那些"已知道生月"的人去看第二篇。

回复模板：
```
[生月]月生·[本命色]调 ✨
更具体的"怎么用这一抹色"的故事在我今天的第 2 篇里——
[文章链接]

```

### 第一篇 + 第二篇 联动指标

| 指标 | 第一篇红线 | 第二篇红线 | 对比看 |

| 公众号阅读 | > 500 | > 800（联动有加成）| 第二篇应该比第一篇高 60% |
| 关注新增 | > 30 | > 50 | 第二篇带来的关注更精准 |
| "本命色" 关键词触发 | > 50 | 持续触发 | 第一篇主战场 |
| **"案例" 关键词触发** | 0 | **> 20** ⭐ | **第二篇核心**指标 |
| **"预约" 关键词触发** | 0 | **> 5** ⭐ | **转化最关键**指标 |
| 小红书曝光 | > 200 | > 500 | 故事型应该数据更高 |
| 私信咨询数 | < 10 | > 20 | 第二篇应该带来更多 |

### 红灯应急处理

| 红灯 | 触发条件 | 处理 |
|---|---|---|
| **预约触发数 < 5** | 第二篇 48h 后 | 文末再加 1 行"前 5 位预约送实物色卡" |
