## 怎么用（30 秒上手）

### 看效果

```
浏览器打开：preview.html
```

5 个 logo 整齐排列，配色板、HEX 值、使用场景一目了然。

### 下载 SVG（推荐）

每个 logo 卡片右下角有 **下载 SVG** 按钮。SVG 是矢量格式：
- ✅ 无限放大不糊
- ✅ 印刷品质 300dpi+
- ✅ 文件超小（每个 1-3 KB）
- ✅ 可在 Photoshop / Illustrator / Figma / Sketch 直接打开编辑

### 转换 PNG（如果平台需要）

某些平台只接受 PNG（如老版本公众号头像上传）：

**方法 1**：浏览器直接转
1. 浏览器打开 SVG 文件
2. 右键 → "图片另存为" → 选择 PNG 格式
3. 保存成功

**方法 2**：在线转换（推荐 https://cloudconvert.com/svg-to-png）
- 上传 SVG
- 设置目标尺寸（建议 1024×1024 或 2048×2048）
- 转换 → 下载 PNG

**方法 3**：本地软件
- Mac：Preview.app 直接打开 SVG → 导出 PNG
- Win：Inkscape（免费）打开 → 导出 PNG @ 300dpi

## 各平台头像规格

| 平台 | 尺寸 | 用哪个 | 上传到 |
|---|---|---|---|
| **微信公众号** | 144×144 px | ① 主标识 | 公众号后台 → 设置 → 头像 |
| **小红书** | 200×200 px | ① 主标识 | 小红书 App → 我 → 编辑资料 → 头像 |
| **抖音** | 720×720 px | ① 主标识 | 抖音 App → 我 → 编辑资料 → 头像 |
| **视频号** | 720×720 px | ① 主标识 | 视频号助手 → 设置 → 头像 |
| **B 站** | 200×200 px | ① 主标识（或 ④ 单字版，强化深度感）| B 站个人空间 → 编辑资料 |
| **大众点评 / 美团** | 200×200 px | ① 主标识 | 商户后台 |

> **强烈建议**：所有平台用同一个头像（① 主标识），强化跨平台 IP 识别。

## 命名一致性铁律

LOGO 内任何文字 **绝不出现**：

```
❌ 转运肖像 / 改运肖像 / 转运壁纸 / 改运壁纸 / 命理书 / 算命 / 看相
❌ 八字 / 用神 / 调候 / 命格 / 奇门 / 八门 / 死门 / 惊门
❌ 微信 / VX / 大师 / 师傅
```

LOGO 内**只用**：

```
✅ 渝山 / 渝山东方美学 / YUSHAN
✅ ORIENTAL AESTHETICS
✅ 命主画像 / 节气壁纸 / 命书（如果文字版需要列产品）
```

详见 `.kiro/steering/01-compliance-redlines.md` §五。

## 设计逻辑（路演给老师讲的话术）

> "我们的 LOGO 系统不是网络爆款 vi 模板。
>
> **核心是朱砂方印「渝山」纵向双字**——这是宋代到民国一直用的私家印格式，
> 在所有客户面前出现，建立"渝山 = 一个有传承的人"的认知。
>
> 5 个版本对应 5 种使用场景：头像、合同抬头、视频水印、礼盒包装、立牌海报，
> 每一个都用同一套 6 主色（来自 19 色色谱），保证客户在不同地方看到我们时
> **一眼就认得出**。
>
> 这是一个**色彩美学品牌**该有的 LOGO 系统：
> 不是一个图，是一套有调性的视觉语言。"
