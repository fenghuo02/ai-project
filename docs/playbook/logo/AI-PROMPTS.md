## 给 Gemini 的特别建议

Gemini 比即梦更适合**精确按规格出**，但有几个调整建议：

### 1. 用 imagen 模式

Gemini 的 imagen-3 模式对于"几何 + 文字"组合理解最好。如果你用普通对话窗口，先说：

```
请用 imagen-3 帮我生成一张图，prompt 如下：[贴上面的 prompt]
```

### 2. 中文字 + 英文 prompt 混用（Gemini 对中文字渲染坑多）

Gemini 对中文字"渝山"的渲染**容易出错**（写成"涂山"或乱码）。优化策略：

**写法 A · 强调字形**
```
The Chinese character must be written exactly as 「渝」(yú, meaning Chongqing)
and 「山」(shān, meaning mountain), in traditional Song dynasty kaishu calligraphy.
Both characters are vertically stacked inside a vermillion red square seal.
DO NOT confuse with other similar characters like 涂、俞、油、舟、川.
```

**写法 B · 给字形参考**
```
Reference the calligraphy style of 清代私家藏书印 / Qing dynasty private library seal:
- Characters are bold, stocky, ancient-looking
- Brushstrokes have natural variation in thickness
- Strong, rooted feeling — not modern thin sans-serif
```

### 3. 跑 4 张选最好的

每条 prompt 跑 **4 张候选**（Gemini 单次默认就是 4 张），对照原 SVG 设计选最贴近的那张。

### 4. 失败重跑的关键调整

如果出图字错了：

| 错误 | 调整 prompt |
|---|---|
| 「渝」写成「俞」/「涂」 | 加 "the character must include the water radical 氵 on the left side" |
| 印章颜色偏粉红 | 加 "deep vermillion, like Chinese mineral pigment cinnabar 朱砂, NOT pink" |
| 字写成楷书但太纤细 | 加 "thick bold strokes, like a stone-carved seal, NOT thin brush" |
| 整体太"现代设计感" | 加 "looks aged, traditional, ancient — NOT modern flat design" |

## 推荐使用顺序

```
1. 先跑 ① 主标识（最重要 + 最难，先把基础视觉定下来）
   → Gemini imagen 跑 4 张 → 选最好的 1 张
2. 再跑 ② 横版 LOGO（用 ① 的印章作为视觉基础）
   → 把 ① 选定的印章风格作为参考图喂给 Gemini
3. ③④⑤ 是衍生形态，可以基于 ② 横版去裁剪/重组
```

## 与 SVG 版本的关系

| 版本 | 优势 | 劣势 |
|---|---|---|
| SVG（PR #36 现有）| 矢量无限缩放、文件超小、印刷品质 | 视觉感「几何」，少手作温度 |
| AI PNG（本文档生成）| 有手作感、宋画工笔肌理、宣纸晕染真实 | 像素图，缩放有损耗 |

**推荐组合用法**：

| 场景 | 用哪个 |
|---|---|
| 印刷品（合同 / 名片 / 海报）| ⭐ SVG（保证印刷品质）|
| 社交平台头像 | ⭐ AI PNG（有温度更吸眼球）|
| 视频水印 | SVG 转 PNG（精确控位）|
| 命书 PDF 内嵌 | SVG（PDF 格式天然支持）|
| 朋友圈分享图 | AI PNG（视觉冲击力强）|
