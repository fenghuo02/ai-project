### 1.3 扶抑用神简则（用于"身强身弱"判断）

| 八字状态 | 用神方向 | 视觉表达 |
|---------|---------|---------|
| **身强**（日主旺，比劫多/印多） | 用财、官、食伤来"泄秀" | 主色 = 食伤/财色，气质偏外放、表达欲、华丽 |
| **身弱**（日主衰，财官杀重） | 用印、比劫来"扶身" | 主色 = 印/比劫色，气质偏内敛、温润、有依靠感 |
| **平衡**（中和） | 顺应大运 / 流年 | 主色 = 流年喜用色，每年微调 |

**赛道彩蛋**：身强 vs 身弱的人即使日主一样，画风完全不同——
- 同是甲木日主、身强 → 出图用"金色阳光、外放、繁花"
- 同是甲木日主、身弱 → 出图用"晨雾、根系、苔绿"
**这就是为什么同月生的两个客户出来的图气质不一样**——这是你和"模板号"的核心区别。

## 二、《奇门遁甲》核心：八门系统决定壁纸 SKU

> **划重点**：奇门的八门 = **天然的 8 个产品 SKU**。
> 这是 v2.0 商业模型的最大升级。
> 老玩法：客户买 1 张本命壁纸 → 客单价 99
> 新玩法：客户买"八门一套" → 客单价 499，复购×4

## ⚠️ v2.1 三个关键技术点

### 1. 客户长相一致性（最难的技术点）

> 62 张图必须**看起来都是同一个人**，否则订阅产品立刻崩盘。

**解决方案**（按工具优先级）：

- **首选：即梦「角色一致性」功能**（最稳定）
  - 上传客户 3 张正脸照 → 创建"角色 ID" → 全年 62 张图都用这个 ID
- **次选：Midjourney --cref 参考链接**
  - 把客户照片传到 imgur → 拿 url → 每个 Prompt 加 `--cref [url] --cw 80`
- **进阶：Nano Banana 的多图融合**
  - 客户照 + 节庆参考图 → 输出"客户长相 + 节庆服饰"

### 2. 本命色锚点（v2.1 灵魂）

> 不管节气怎么变，**每张图必须有客户本命色的 25-40% 占比**，否则就和"通用节气壁纸"没区别。

实现方法：
- 节庆肖像 → **服饰主色 = 本命色**（最稳）
- 节庆壁纸 → **画面 1/4 区域留给本命色元素**（如本命色的烛光、麦穗、远山）

### 3. 节点提前推送（情感价值最大化）

> **节前 3 天**推送是黄金窗口——客户既来得及换头像/壁纸，又比"当天才发"显得有"被记住"的仪式感。

技术实现：
- 飞书/钉钉日历 + 提前 5 天提醒（你内部 SOP）
- 节前 3 天：邮件 + 微信 + 私域社群三通道推送
- **不要太早**（提前 10 天客户还没进入节庆情绪）
