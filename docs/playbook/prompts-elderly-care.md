# 养老赛道｜小红书 + 公众号双平台提示词工具包

> 赛道：养老 / 中老年生活 / 退休规划 / 银发经济
> 策略：小红书为主（涨粉），公众号为辅（沉淀私域）
> 目标人群：
>   - 主力：35-55 岁子女（为父母规划养老）
>   - 次力：50-65 岁本人（即将/刚退休）
>   - 延伸：养老行业从业者
> 版本：v1.0

---

## 使用说明

本文档包含 **10 套 Prompt**，按编号顺序使用即可覆盖「定位→选题→创作→运营→复盘」全链路。

每个 Prompt 都是**直接粘贴给 AI（ChatGPT/Claude/Kimi）就能用的完整指令**。

## Prompt 4：公众号深度文改写器（养老版）

```

擅长把碎片化内容扩写成有温度有深度的长文。

改写要求：
1. 标题重写：20-28 字，有信息量+有温度
   - 不要用"姐妹""绝了""yyds"
   - 养老类标题适合：故事感 / 清单感 / 政策感 / 情感共鸣
   - 例："陪 73 岁母亲看了 6 家养老院后，我总结出这 5 条铁律"
   - 例："退休金每月 3000，够养老吗？算完这笔账我沉默了"

2. 开头重写（200 字内）：
   - 用 1 个真实故事切入（自己/身边人/社会新闻）
   - 或 1 组引人深思的数据
   - 建立"这篇文章值得认真看完"的期待

3. 正文扩写：
   - 3-5 个 H2 小标题
   - 每个要点要有：解释 + 真实案例 + 实操建议 + 避坑提醒
   - 加入"我的观察""身边的故事"等真实细节

4. 金句设计：至少 2 句可以转发朋友圈的话，用加粗标出
   - 养老金句风格：温暖、有力量、不鸡汤
   - 例："最好的养老，不是有多少钱，是提前想清楚自己要什么生活"
   - 例："照顾父母不是一个人的事，但至少得有一个人先开始想"

5. 结尾：
   - 总结 3 条核心观点
   - 引导"点赞在看""转发给家人""关注不迷路"
   - 最后一句埋下篇钩子

6. 配图建议：标出 3-5 处插图位置 + 说明
   - 养老类配图建议：温暖/家庭/自然风景/数据图表
   - 避免：黑白悲伤老人照 / 医院病床等负面画面

7. 文末话题标签 3 个

【风格】：温暖理性，像一个提前规划好养老的朋友分享经验
【底线】：不贩卖焦虑，不做医疗建议，不推销金融产品

   - 有画面（"她穿着那件旧花棉袄站在门口"）
   - 有时间（"那是去年冬天""退休第 37 天"）
4. 情绪节奏：平静开始 → 矛盾/小高潮 → 释然/温暖结尾
5. 结尾要有"余韵"：不要写明"所以我们要孝顺父母"
   而是留一个画面/一句话让读者自己感受

6. 字数 300-450 字
7. 配 1-2 个 emoji（不宜多，情感类保持克制）
8. 5 个话题标签
9. 互动引导："你和父母之间，有什么想说的？"

【避雷】：
- 不要过度煽情 / 不要"卖惨"
- 不要说教 / 不要道德绑架
- 不要虚假（读者能感受到真假）
- 小红书对"哭了""泪目"等标题已经审美疲劳，换别的表达
```

## 养老赛道 vs AI 赛道的核心差异提醒

| 维度 | AI 工具赛道 | 养老赛道 |
|------|-----------|---------|
| 用户年龄 | 20-35 岁 | 35-65 岁（含代为阅读的子女） |
| 语气 | 闺蜜/同事/网感强 | 温暖/真诚/朋友感/不浮夸 |
| emoji 使用 | 多、密、花哨 | 少、克制、有情绪 |
| 爆款逻辑 | 新鲜感/信息差 | 共鸣感/安全感/被理解 |
| 互动方式 | "扣 1""蹲" | "说说你家情况""你怎么选的" |
| 变现路径 | 快（知识付费/带货） | 偏慢但客单价高（咨询/服务/保险） |
| 内容生命周期 | 短（工具迭代快） | 长（养老规划是长决策） |
| 搜索流量占比 | 30% | 60%+（养老类搜索行为很强） |
| 发布最佳时间 | 20:00-22:00 | 早 7-9 点 / 晚 19:00-21:00 |
| 封面风格 | 亮色/大字/花哨 | 温馨/家庭/自然/数据图表 |
| 信任建立 | 靠"工具实测" | 靠"真实故事+专业背景" |

## 养老赛道避坑清单

- ⚠️ **不做医疗建议**：任何涉及"吃什么药""什么偏方"都可能被举报
- ⚠️ **不推销金融产品**：养老保险/理财类内容容易踩监管红线
- ⚠️ **不贩卖焦虑**：养老话题本身就有焦虑感，内容要给"解法"不是加重恐惧
- ⚠️ **注意用户情绪**：评论区可能有正在经历丧亲/重病的用户，回复要有温度
- ⚠️ **政策类加免责**：所有政策解读结尾加"具体以当地社保部门为准"
- ⚠️ **素材脱敏**：提到真实案例要隐去姓名/地点等隐私信息
- ⚠️ **不要居高临下**：养老话题最忌"我教你怎么做"，要用"我也在学"的姿态
