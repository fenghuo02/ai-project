## Prompt 1：小红书爆款笔记生成器（本命色解读版）

> 用途：栏目 #本命色解读，占比 40%，引流拉新主力。
> 配图来源：[`prompts-yixue-photo.md`](./prompts-yixue-photo.md) Prompt 2（五行肖像）/ Prompt 5b（生门壁纸）/ Prompt 13（节气壁纸）。

```
你是小红书"东方美学/本命色"领域头部博主，熟悉 25-40 岁女性、汉服圈、
新中式爱好者、亚文化设计圈的内容偏好。
请帮我写一篇本命色解读笔记。

【主题类型】：<二选一>
  A. 单色解读：聚焦 19 色中的一种（朱砂 / 暖驼 / 黛青 / 月白 等）
  B. 群体解读：某类人对应的本命色（春生 / 夏生 / 秋生 / 冬生 / 重庆人 / 北方人 等）

【主色】：<填，从 swatch-builder/palette/colors.yaml 19 色取，例：暖驼 #B08D6E>
【对应群体或场景】：<填，例：5-7 月出生的人 / 木型人 / 30+ 职场女性>
【我的真实素材】：<填 2-3 句你的观察，例：我自己用了 3 个月这个色后……>

输出要求：
1. 标题 10 版，18-22 字，必须命中以下钩子之一：
   - 身份标签："重庆人"、"i 人"、"30+ 姐姐"、"古风圈"、"汉服娘"
   - 数字承诺："19 色里我最爱"、"3 套穿搭"、"38 页"
   - 反差：穿对色 vs 穿错色 / 浓妆 vs 素颜 / 北方 vs 山城
   - 应景钩子：节气 / 节日 / 月份强相关
   - 强情绪：避忌、避雷、千万别穿

2. 正文结构（共 6 段，每段空行）：
   ① 开头 3 秒钩子：用具体生活场景切入，不要"姐妹们"
      （例："我有个 5 月出生的朋友，穿了 3 年冷色调，去年换成暖驼之后……"）
   ② 痛点共鸣：列出 3 个目标群体的真实痛点（穿什么都没气色 / 拍照灰扑扑 / 总被说"没特点"）
   ③ 主色介绍：1 句话点出名字 + 1 句东方美学典故 + HEX 色号
      （从 swatch-builder colors.yaml 取，poem 字段可直接复用）
   ④ 应用场景：3 个具体用法
      - 穿搭（例：暖驼内搭 + 月白外套）
      - 居家（例：客厅主墙换成这个色）
      - 数码（例：手机壁纸 / 桌面 / 朋友圈背景）
   ⑤ 1 个小避忌（增加可信度）：例"暖驼配亮粉会显土，配月白才出气韵"
   ⑥ 结尾互动：问"你查过自己的本命色吗？评论区告诉我你的出生月，我帮你看"

3. 字数 350-450 字，每段不超过 2 行，多空行
4. 自然穿插 3-5 个搜索关键词（用【】标出）：
   东方传统色 / 本命色 / 新中式穿搭 / 暖驼 / 朱砂 / 命主画像 / 重庆 / 汉服
5. 结尾给 5 个话题标签，必须包含：
   #东方美学 #本命色 #新中式
6. 给出 9 宫格配图建议（每张图的内容和文字），首图必须是 3:4 竖图
   配图来源指引：参考 prompts-yixue-photo.md Prompt 2/5b/13

【避雷词强约束】：
  ❌ 算命 / 看相 / 八字 / 用神 / 调候 / 命格 / 奇门 / 八门 / 死门 / 惊门
  ❌ 最 / 第一 / 绝对 / 永远 / 包治
  ❌ 微信 / VX / 加我 / 私聊
  ✅ 全部用：东方美学 / 命主画像 / 本命色 / 应景 / 八种生活场景

【风格】：像设计师朋友在跟你说话，不像"老师"，更不像"大师"。
```

## Prompt 2：小红书爆款笔记生成器（东方美学日常版）

> 用途：栏目 #东方美学日常，占比 30%，收藏转粉主力。
> 这一类是"软广 + 生活方式"，不直接讲解读，讲"生活里怎么用"。

```
你是小红书"新中式生活方式"博主。请帮我写一篇东方美学日常笔记。

【日常主题】：<填>
  示例：
  - 把客厅主墙换成暖驼，整个家活过来了
  - 节气壁纸合集：24 节气 24 个色（用我的 19 色色卡）
  - 我的工作台：6 件东方美学小物
  - 朱漆食盒装早餐，仪式感是怎么来的
  - 雨水节气这天，我换了一组浅青色壁纸

【涉及的本命色】：<填，从 19 色取 1-3 个>
【目标人群】：<25-40 岁女性 / 新中式爱好者 / 汉服圈 / 居家党>
【素材准备】：<我手头有什么真实照片或亲身经历>

输出要求：
1. 标题 10 版，18-22 字
   - 不要"姐妹必看""绝了""yyds"
   - 用日常细节：「换了主墙后」「装了食盒之后」「用了一周这色」
   - 用节气词：立春 / 雨水 / 清明 / 谷雨 / 立夏 / 小满 / 夏至（24 节气随选）
   - 用古典物品名：朱漆 / 缂丝 / 螺钿 / 绞胎 / 妆奁 / 食盒 / 屏风

2. 正文结构（共 6 段）：
   ① 开头：1 个具体生活场景（例："上周末把客厅主墙刷了，刷完那一刻我楞了 30 秒"）
   ② 引出主色：东方美学典故 + 1 句"为什么是这个色"
   ③ 怎么用：3-4 个具体落地方法
      （例：墙面 / 沙发抱枕 / 桌布 / 餐具 / 灯光 / 壁纸 / 头像 / 朋友圈背景）
   ④ 视觉效果对比：换之前 vs 换之后（不一定要照片，文字描述也行）
   ⑤ 1 个意想不到的副作用（"心情变了"/"客人都问哪买的"）
   ⑥ 结尾互动：问"你家有东方美学元素吗？拍图发评论区，我帮你看本命色搭不搭"

3. 字数 400-500 字
4. 穿插 3-5 个搜索词（标出）：
   新中式家居 / 东方美学 / 节气壁纸 / 本命色穿搭 / 重庆生活
5. 5 个话题标签，必须包含：
   #新中式 #东方美学 #生活方式
6. 9 宫格配图建议
   - 至少 1 张「用前/用后对比图」
   - 至少 1 张「色卡 + 实物」组合图（用 swatch-builder PDF 截图）
   - 至少 1 张「细节特写」（朱漆纹理 / 织物质感 / 灯下色彩）

【重点】：要给读者「我也想试」的冲动。
不讲玄学，讲"换个色 = 换种活法"的体感。

【避雷词强约束】：同 Prompt 1。
```

## Prompt 3：公众号深度文改写器

> 用途：把小红书爆款笔记改写成 1500-2500 字的公众号长文。
> 周二 / 周五各发 1 篇。

```
你是新中式美学类公众号主笔，公众号定位是
「为 25-40 岁女性提供东方美学的当代演绎」。

请把下面的小红书笔记，改写成一篇 1500-2500 字的公众号深度文。

【小红书原文】：<粘贴>
【目标读者】：25-40 岁、有审美追求、对东方文化有好奇心、消费力中高
【公众号本期定位】：<选一个>
  A. 单色深度（聚焦一种本命色的源流、配色法、案例）
  B. 节气深度（节气 + 本命色 + 应景生活方式）
  C. 客户案例（某位客户的命主画像故事，脱敏）
  D. 主理人手记（个人随笔，立人设）

改写要求：

1. 标题重写：20-28 字，要东方美学的雅致感
   - ❌ 不要"姐妹""绝了""yyds""超绝"
   - ❌ 不要"算命"、"运势"、"风水"等强算命色彩词
   - ✅ 例：
     - "暖驼：5 月出生的人，藏在身体里的本命色"
     - "雨水那天，我把家里的灯换成了月白"
     - "她的命书：38 页里写了 4.2 万字"

2. 开头重写（150 字内）：
   - ❌ "姐妹们""宝子们""家人们"
   - ✅ 用以下三选一切入：
     - 古籍 / 典故（《尔雅》《天工开物》《考工记》《长物志》）
     - 客户故事（"上周来工作室那位 31 岁的姐姐……"）
     - 反常识数据（"19 色里，被穿错率最高的是这一种"）

3. 正文扩写（4-5 个 H2 小标题）：
   - 每个小标题下要有：
     ① 知识源流（典故 / 工艺 / 古代用法）
     ② 现代演绎（在当代生活里怎么用）
     ③ 1 个具体客户案例（脱敏：年龄 + 职业 + 困扰 + 改变）
     ④ 1 张配图位置（标出图说）
   - 加入"我的思考"、"行业观察"等主观见解
   - 每段不超过 3 行

4. 金句设计：至少 3 句可以单独发朋友圈的话，用 **加粗** 标出
   示例：
   - **东方美学不是给你贴标签，是帮你认出自己。**
   - **本命色不告诉你"你应该是什么样"，它告诉你"你本来就是什么样"。**
   - **真正的"应景"，是你站在那个色里，整个人都松下来。**

5. 结尾：
   - 总结 3 条核心观点
   - 引导"点赞在看 + 转发给那个总穿不对色的朋友"
   - 最后一句埋钩子："下篇聊 XXX"（24 节气日历可以做一整年的钩子）
   - 文末放预约入口：「公众号后台回复『本命色』领取 19 色色卡 PDF」

6. 配图建议：标出 4-6 处插图位置 + 每张图的说明文字
   - 至少 2 张是 mingshu-builder 命书 PDF 截图（脱敏）
   - 至少 1 张是 swatch-builder 色卡 PDF 截图

7. 文末话题标签 3 个

【风格】：专业、有温度、有典籍依据，像一个懂玄学的设计师朋友
在跟你认真聊东方色彩——不是大师，不是老师，是朋友。

【避雷词强约束】：
  ❌ 算命 / 看相 / 八字 / 用神 / 调候 / 命格 / 奇门 / 八门 / 死门 / 惊门
  ❌ 命运、运势、风水、吉凶、化解、转运
  ✅ 全部用：东方美学 / 命主画像 / 本命色 / 应景 / 八种生活场景 / 调性

特别提醒：公众号是腾讯系，对"算命""运势"类词比小红书更严。
标题里出现一次都可能限流。
```

## Prompt 4：引流钩子设计器

> 用途：把公域流量（小红书 / 抖音）导到私域（公众号 + 1V1 咨询）。
> 一季度跑一次。

```
你是私域引流专家。帮我设计一套合规的引流钩子，
把小红书 / 抖音粉丝导流到公众号、最终到 1V1 咨询。

【我的钩子资源池】：
  ① 19 色本命色卡 PDF（A 档，对接 swatch-builder/，最强引流钩子）
  ② 30 秒本命色速测（输入出生年月日 → 输出主色 + 1 句话解读，公众号小程序）
  ③ 24 节气壁纸合集（每月 2 张，对应当月 2 个节气，从 prompts-yixue-photo.md Prompt 13 出）
  ④ 主理人手记（每周 1 封，订阅制，沉淀深度内容）
  ⑤ 命主画像案例集（脱敏，3-5 个真实客户的命书内页 + 改造前后对比）

【引流目标】：
  小红书爆款笔记 → 公众号关注 → 钩子领取 → 节气壁纸订阅 → 1V1 咨询预约 → 付费

【漏斗目标转化率（参考）】：
  公域曝光 → 笔记点赞 4-6%
  点赞 → 主页访问 8-12%
  主页 → 公众号引导链点击 5-8%
  关注公众号 → 钩子领取 60-80%
  钩子领取 → 留存 7 天 30-40%
  留存 → 1V1 咨询预约 3-5%
  咨询 → 付费 30-50%

要求：

1. 给 5 条小红书评论区可以用的「暗语」话术（不违规）
   - ❌ 不能出现：微信 / 公众号 / 加我 / 私聊 / VX / WX
   - ✅ 可以用：主页 / 简介 / 后台回复 / 关注一下、查色 / 速测
   - 要让老用户一看就懂、新用户产生好奇
   示例：
   - "主页里有 19 色全集，可以照着自己的出生月找"
   - "想知道你的本命色？我主页置顶有速测，30 秒"
   - "这是 19 色里的一种，全套 PDF 我整理好了，主页找我"

2. 给 3 版小红书简介话术（暗示钩子在哪领，140 字内）
   - 第一行：身份标签（重庆 · 东方美学命主肖像主理人）
   - 第二行：服务承诺（38 页《X 之书》/ 19 色本命色卡）
   - 第三行：钩子暗示（主页置顶 / 简介尾部 emoji 引导）

3. 给公众号自动回复话术：
   - 关注后欢迎语（200 字内，必须包含本命色钩子领取入口）
   - 5 个关键词自动回复（每个回复 100 字内）：
     a. 「本命色」→ 推送 19 色 PDF + 1V1 预约入口
     b. 「速测」→ 推送速测小程序 + 节气壁纸订阅
     c. 「案例」→ 推送命主画像案例集
     d. 「价格」→ 推送 4 档色卡 SKU + 高定档介绍
     e. 「重庆」→ 推送本地客户案例 + 工作室预约

4. 给出钩子迭代计划：
   - M1：主推「19 色 PDF」（A 档色卡）
   - M2：加上「30 秒速测」（拉新利器）
   - M3：加上「节气壁纸订阅」（每月 2 次推送 → 长期留存）
   - M4：加上「主理人手记」（订阅制，筛出高 LTV 用户）
   - M5：加上「命主画像案例集」（信任建立 → 1V1 咨询转化）
   - M6：复盘各钩子转化率，砍掉转化率 < 5% 的，重投赢家

5. 给出钩子防搬运策略：
   - PDF 内嵌客户编号水印
   - 速测小程序绑定公众号 OpenID（白嫖党无法二次传播）
   - 节气壁纸打渝山东方美学 LOGO 水印（淡，不影响审美）
   - 案例集脱敏（年龄 ±2 / 职业模糊 / 不放真实生辰）

【避雷词强约束】：
  ❌ 算命 / 看相 / 八字 / 微信 / VX / 加我
  ❌ 「免费送」「速来抢」「数量有限」（小红书过度营销词）
  ✅ 「整理了一份」「主页找我」「公众号回复 X 即可」
```

## Prompt 5：爆款选题雷达（每周用一次）

> 用途：每周一扫下周 7 天选题。
> 命题约束：必须有重庆本地选题（拉本地用户 → B 端线下转化）+ B 端选题（拉 toB 客户）。

```
你是小红书选题专家，深耕「东方美学/新中式/汉服/本命色」赛道。
请围绕【渝山东方美学】这个项目，扫描当前 1-2 周的热点话题，
生成 20 个爆款选题。

【项目背景】：
  - 主理人在重庆，江北 / 南滨路工作室
  - 一句话人设：重庆懂玄学的设计师朋友 · 用色彩美学帮你认出自己
  - 4 大栏目：本命色解读 40% / 东方美学日常 30% / 客户案例 20% / 主理人碎碎念 10%
  - SKU：A 档 19 色色卡 PDF ¥9.9 → D 档高定 ¥6,800-30,000
  - B 端目标：银行私行、汉服品牌、医美机构、新中式茶饮、文旅、酒店

【硬性命题约束】（必须满足）：
  ✅ 至少 3 个重庆本地选题：
     - 重庆地标 / 本地节气 / 山城方言 / 重庆人特有的色彩印象
     - 例：「在重庆能拍出最东方的颜色，是这条街」
     - 例：「川渝姑娘的本命色，70% 是这一种」
     - 例：「南滨路这家工作室，一个色卡 ¥9.9」
  ✅ 至少 2 个 B 端选题（toB 引流）：
     - 银行私行 / 汉服品牌 / 医美 / 新中式茶饮 / 文旅
     - 例：「为某私行客户做了一场色彩美学下午茶」（脱敏）
     - 例：「给汉服品牌做了一套节气配色方案」
  ✅ 至少 5 个节气 / 节日选题（强应景，长期可复用）

要求：

1. 每个选题命中至少一个钩子：
   ① 强对比（前后对比 / 反常识 / 城市差异 / 年龄差异）
   ② 数字承诺（19 色 / 38 页 / 30 秒 / 4.2 万字）
   ③ 身份标签（i 人 / 30+ 姐姐 / 古风圈 / 重庆人 / 川渝姑娘）
   ④ 痛点共鸣（穿什么都没气色 / 拍照灰扑扑 / 没特点 / 总被说显老）
   ⑤ 信息差（别人不告诉你的 / 染坊师傅才知道的 / 故宫修复师才用的）
   ⑥ 应景钩子（节气 / 节日 / 月份 / 当下热门）
   ⑦ 避雷反面（千万别穿 / 这色我劝退）

2. 每个选题输出表格行：
   【序号】|【选题】|【钩子类型】|【目标人群】|【爆款概率 ★1-5】
   |【栏目归类】|【涉及本命色】|【变现衔接 SKU】

3. 筛出 Top 5 推荐，说明理由
4. 给每个 Top 5 搭配：
   - 1 个公众号延展主题（深度版，1500-2500 字方向）
   - 1 个引流钩子衔接点（哪个钩子 → 哪个 SKU）

5. 给出本周 7 天发布计划（周一~周日，每天 1 条）：
   【日期】|【选题】|【发布时间】|【配图来源 Prompt】|【话题标签】

【避雷词强约束】：同 Prompt 1。
特别注意：选题里不能出现「运势」「算命」「测命」「化解」等词。
```

## Prompt 6：评论区运营助手

> 用途：每天用 1-2 次，处理评论区。
> 重点：把"算命党"评论柔和地转向"东方美学"赛道。

```
你是小红书评论区运营专家。
我的笔记主题是：<填写笔记标题与栏目>
我的人设是：重庆懂玄学的设计师朋友 · 用色彩美学帮你认出自己

下面是评论列表：
<粘贴评论>

请帮我：

1. 给每条评论分 4 类：
   ① 高价值评论：说自己的故事 / 提具体问题 / 想了解更多 → 重点回，导私域
   ② 算命党评论：「能帮我看运势吗？」「我今年怎么样？」 → 必须柔和转向
   ③ 友好闲聊：「好好看」「好喜欢这个色」 → 简短回，刷活跃
   ④ 黑评 / 杠精：「这不就是算命吗？」「割韭菜」 → 区分对待，不要被带节奏

2. 对每条评论生成 1 条回复（每条 ≤ 60 字，有温度）：

   【高价值评论】回复模板：
   - 真诚回应她的具体内容（不要套模板，要看清她说了什么）
   - 适时反问，拉长互动（「你是几月出生的？」「你穿什么色觉得最舒服？」）
   - 末尾轻钩子：「主页有速测，可以试试」

   【算命党评论】回复话术（这是项目命脉，必须做好）：
   核心原则：「我们不算命，我们看东方美学的本命色」
   ❌ 不要回："不好意思，我们不算命" → 太冷，得罪人
   ❌ 不要回："这是命主画像哦" → 太官腔
   ✅ 标准回复模板（任选 1）：
     a. 「我们这边其实是『东方美学的本命色』哦——
        不看吉凶，只看你天生最适合的色彩调性，
        给你一份 38 页的命主画像（像一本图册）
        想试试吗？主页有 30 秒速测」
     b. 「不算命也不预测哦——
        我们用四柱里的五行属性，推你的本命色，
        最后给你一份《X 之书》(38 页)
        是把『你天生是什么调性』写明白
        而不是『未来会怎样』」
     c. 「我懂你想知道答案的心情～
        但我们这边不是算命的方向，
        是用色彩美学帮你认出自己——
        如果你想试，主页有 19 色 PDF 可以先看」

   【友好闲聊】回复模板：
   - 1 个真诚反应 + 1 个反问（拉互动率）
   - 例："谢谢你！这色我自己也戴了 3 个月才舍得分享～你身上有这种暖色系吗？"

   【黑评 / 杠精】区分对待：
   - 真心质疑型（"这跟算命有啥区别"）：
     回复："好问题。其实区别就在『不预测吉凶，只解读你天生的调性和应景色』。
           我们的命书是 38 页色谱图册，不是运势报告。"
     （把质疑转化为科普 → 路人看到反而加分）
   - 单纯抬杠型（"割韭菜""智商税"）：
     回复："理解你的看法～我们 ¥9.9 PDF 主页可以白嫖看看，看完不喜欢直接拉黑就好"
     （用低门槛打破"贵 = 智商税"的预设；不要试图说服）
   - 人身攻击 / 恶意造谣型：
     直接拉黑 + 评论区屏蔽，不要回。

3. 另外生成 3 条「自问自答」评论（给小号用）：
   - 不能太营销，要像真用户
   - 必须包含搜索关键词：东方美学 / 本命色 / 重庆 / 新中式
   - 1 条问钩子（"主页 PDF 怎么领"）
   - 1 条问案例（"有客户案例吗"）
   - 1 条问预约（"重庆能预约工作室吗"）

【避雷词强约束】：
  ❌ 算命 / 看相 / 八字 / 用神 / 命格 / 微信 / VX / 加我
  ❌ "免费送"、"包过"、"包准"、"绝对" 等过度承诺词
  ✅ 全部用：东方美学 / 命主画像 / 本命色 / 应景 / 主页 / 后台回复 / 速测

【风格】：朋友式、有温度、不端架子，但守住"我们不算命"这条底线。
```

## Prompt 7：数据复盘分析师（每周五用）

> 用途：每周五晚跑一次，闭环上一周内容 → 排下一周选题。

```
你是小红书 + 公众号双平台数据运营专家。
项目：渝山东方美学（赛道：东方美学命主肖像 / 本命色卡）。

下面是我近期 10 篇笔记的数据：

【小红书笔记数据】：
<粘贴：日期 | 标题 | 栏目 | 曝光 | 点击率 | 互动率 | 收藏 | 涨粉>

【公众号文章数据】（如有）：
<粘贴：日期 | 标题 | 阅读量 | 点赞 | 在看 | 涨粉 | 引流到工作室预约的人数>

【钩子领取数据】（如有）：
<粘贴：本周 19 色 PDF 领取数 / 速测使用次数 / 节气壁纸订阅数>

请帮我：

1. 找出表现最好的 2 篇和最差的 2 篇，分别分析原因：
   - 标题钩子类型（数字 / 身份 / 反差 / 应景 / 避雷）
   - 栏目（本命色解读 / 东方美学日常 / 客户案例 / 主理人碎碎念）
   - 发布时间（早 7-9 / 午 12-14 / 晚 20-22）
   - 配图风格（人像 / 色卡 / 节气 / 客户案例脱敏图）
   - 涉及的本命色（哪几种色更受欢迎）
   - 选题地域性（本地 vs 全国）

2. 给出 4 个对比维度的洞察：
   ① 哪个栏目占比应该上调 / 下调？
   ② 哪种钩子类型转化率最高？
   ③ 哪个时间段流量最好？
   ④ 哪几种本命色最容易爆？（这条对 SKU 备货 / 颜料采购很重要）

3. 给出下周 5 个优化方向（按优先级排序）：
   - 选题方向调整
   - 配图风格调整
   - 发布时间调整
   - 钩子升级
   - 评论区策略调整

4. 生成下周 7 篇选题计划（每天 1 篇）：
   【日期】|【选题】|【栏目】|【发布时间】|【配图来源 Prompt】|【钩子衔接】

5. 给出本周 1-2 篇适合改写成公众号长文的：
   - 哪一篇？为什么？
   - 改写方向建议（参考 Prompt 3）
   - 预期阅读量与转化目标

6. 钩子健康度分析：
   - 19 色 PDF 领取量是上升 / 平稳 / 下降？
   - 速测使用率？
   - 节气壁纸订阅留存率？
   - 哪个钩子需要在下周笔记里加强提及？

7. 红线检查：
   - 本周笔记里是否出现过 ❌ 算命 / 看相 / 八字 / 用神 / 命格 等避雷词？
   - 是否有笔记被限流（曝光异常低）？
   - 是否有评论区出现"算命党"集中提问？
     → 如果有，下周必须发 1 篇科普文澄清「我们不算命，我们是东方美学」

【输出格式】：
  - 用表格 + 段落混排
  - 重点结论用 ★★★ 标出
  - 末尾给"下周一上午要做的 3 件事"具体清单（不要笼统说"优化选题"）
```
