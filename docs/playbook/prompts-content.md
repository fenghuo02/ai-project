```
   - 发布时间调整
   - 钩子升级
   - 评论区策略调整

4. 生成下周 7 篇选题计划（每天 1 篇）：
   【日期】|【选题】|【栏目】|【发布时间】|【配图来源 Prompt】|【钩子衔接】

5. 给出本周 1-2 篇内容建议（v1.1 双向选择）：
   方向 A：哪一篇短视频/小红书原创素材，适合反向扩写成公众号长文（如客户案例 vlog 反响很好，可扩为客户案例长文）？
   方向 B：哪一篇公众号长文表现尤其好，下周可加一篇 B 站长视频跟进（科普向，详见 [`bilibili-onboarding.md`](./bilibili-onboarding.md) §2.2 类型 A 知识体系篇）？
   - 选择哪篇？为什么？
   - 衍生方向建议（参考 Prompt 3 公众号长文 / B 站长视频）
   - 预期阅读量与转化目标

6. 钩子健康度分析：
   - 19 色 PDF 领取量是上升 / 平稳 / 下降？
   - 速测使用率？
   - 节气壁纸订阅留存率？
   - 哪个钩子需要在下周笔记里加强提及？

7. 红线检查：
   - 本周笔记里是否出现过 ❌ 算命 / 看相 / 八字 / 用神 / 命格 等避雷词？
   - 是否有笔记被限流（曝光异常低）？
   - 是否有评论区出现"算命党"集中提问？
     → 如果有，下周必须发 1 篇科普文澄清「我们不算命，我们是东方美学」

【输出格式】：
  - 用表格 + 段落混排
  - 重点结论用 ★★★ 标出
  - 末尾给"下周一上午要做的 3 件事"具体清单（不要笼统说"优化选题"）
```

## 使用顺序建议（v1.1 四平台版）

```
v1.5 平台矩阵的每周流程（公众号原创主阵地 → 多平台分发）：

  周一上午：
    用 Prompt 5（选题雷达）扫下周选题
    输出 → 7 天选题计划 + 配图来源 Prompt 编号 + 平台分发标记

  周一 / 周三 / 周日（每周 3 条短视频）：
    用 short-video-script-prompts.md Prompt 1（三平台同发短视频脚本生成器）
    一份信息卡 → 抖音 30-45s + 视频号 45-60s + B站 60-90s 三个版本
    抖音先发 → 30 分钟后投视频号 + B站（避免搬运降权）

  周二 + 周五（每周 2 篇公众号长文 ⭐ 原创主阵地）：
    上午：用 Prompt 3 写公众号长文（输入：客户案例 / 节气节点 / 主理人观察 / 古籍引用）
    晚 20:30 推送公众号
    12h 内（晚 21:00 - 次日 9:00）：用 Prompt 1 把长文搬运成小红书图文版

  全周（每天）：
    用 Prompt 6 运营评论区（每条 ≥ 1 回复，算命党必处理）
    四平台共用同一套合规话术

  仅在以下场景启用 ✍️ 小红书原创（每月 2-4 篇）：
    - 节气当天首发（公众号长文还没排上）
    - 主理人 vlog（朝天门挑货 / 矿物颜料试色 / 工作室碎片）
    - 客户案例脱敏快讯（不需要 1500 字深度）
    用 Prompt 2 写

  每月最后一周日（B 站长视频 ⭐）：
    用 bilibili-onboarding.md §2.2 类型 A/B/C/D 模板
    5-15 分钟，月度 1 条，与本月某篇公众号长文配套（强复用）

  周五晚 22:00：
    用 Prompt 7 跑数据复盘（四平台数据全部喂入）
    输出 → 下周优化方向 + 7 天选题计划

  每月第 1 天：
    用 Prompt 4 刷新引流钩子（按 M1-M6 节奏推进）

  每季度第 1 周：
    用 Prompt 4 复盘钩子转化率，砍掉低于 5% 的
```

| 🔄 小红书图文搬运 | 8 篇 | **30 分钟**（公众号长文搬运）|
