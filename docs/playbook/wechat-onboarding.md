**公众号文案合规比小红书更严**——腾讯系对「算命」「运势」「测命」类词限流极重。
**所有自动回复 / 菜单栏 / 简介 / 群发文案禁止出现：**

- ❌ 算命 / 看相 / 八字 / 用神 / 调候 / 命格 / 格局 / 奇门 / 八门 / 死门 / 惊门

- ❌ 微信 / VX / 加我 / 私聊（公众号自动回复里**严禁**导流到个人号）

**统一替换为：**
- ✅ 东方美学 / 命主画像 / 本命色 / 应景色 / 调性
- ✅ 主理人 / 设计师 / 色彩研究 / 渝山

## 九、常见问题 FAQ

### Q1：个人订阅号一天只能推 1 次，够用吗？
够用。**频繁推送会掉粉**。每周 2 次稳定更新（周二 + 周五），反而涨粉更快。

### Q2：没开原创怎么办？
前 1-2 周正常发原创文章，积累几篇后公众号后台会邀请你开通原创。开通后文章才能被赞赏 / 转载授权。

### Q3：钩子资源放哪里？
- **首选**：腾讯文档（免费、可实时更新、无下载门槛、可水印保护）
- **备选**：百度网盘（需提取码，下载率略低）
- **避免**：直接发 PDF 文件（公众号会压缩变糊）

> PDF 内嵌客户编号水印，详见 [`prompts-content.md`](./prompts-content.md) Prompt 4 §5「钩子防搬运策略」。

### Q4：多久能开流量主？
订阅号关注数满 **500 人** 即可开通流量主（广告分成）。
关注数满 **100 人** 可开通赞赏功能。

### Q5：可以推荐微信加好友吗？
**严禁**。腾讯系会判定为"诱导加私域"，轻则限流，重则封号。
导流路径**只能**是「公众号 → 表单预约 → 工作室客服回访 → 加微信」。

### Q6：算命党在评论区刷「能帮我看运势吗」怎么办？
用 [`prompts-content.md`](./prompts-content.md) Prompt 6 的标准转向话术。
核心：**「我们不算命，我们看东方美学的本命色」**。

### Q7：B 端客户问「八字 / 命格」怎么回？
合作场景下也保持口径——
"渝山做的是**东方美学的色彩定制**，不做命理咨询。
合作内容是色彩方案 / 色卡定制 / 节气壁纸 / 私享活动。"

—— 这样既守住合规，又给 B 端一个清晰的合作边界。
