## 一、头像设计 5 大原则

1. **一眼识别**：缩到 40 px 还能看清主体（朱砂印 / 暖驼底）
2. **强记忆点**：「渝山」二字 + 朱砂印章感是核心识别符
3. **东方雅致**：不要卡通可爱风、不要 emoji 表情、不要黑眼圈打工人风
4. **和账号强关联**：色彩 + 东方 + 重庆元素
5. **跨平台统一**：小红书 / 公众号 / 视频号 / 抖音用同一张

## 二、6 个头像方案（按推荐度排序）

> 全部从 [`swatch-builder/palette/colors.yaml`](../../swatch-builder/palette/colors.yaml) 19 色里取色，
> 不再使用奶油黄 + 番茄红 + 薄荷绿那套 AI 工具号配色。

### 🥇 方案 1：朱砂印 · 渝山二字（最推荐 · 最有识别度）

**为什么推荐**：
- 印章感最东方，缩到 40 px 仍能识别
- 「渝山」二字直接打出 IP 名，省了昵称解释
- 米白底 + 朱砂方印是新中式品牌头像最经典的配方
- 可以无限复用在落款、贴纸、工作室名片

**视觉描述**：
```
- 圆形头像 512×512
- 底色：米白 #FAF7F2（mibai）
- 主体：一枚朱砂红 #C0392B（zhusha）的方形印章，
  印章占画面中央 60%，
  印面上是「渝山」二字（篆体 / 思源宋体 Heavy）
  字色：米白（阴文反白）
- 印章四周：一圈淡米麦 #F5E6C8 的水墨晕开效果（不要太满）
- 边缘：墨青 #2C3E50 细描边 2 px（增强辨识度）
```

**HTML 模板**：[`covers/avatar-yu-shan-seal.html`](./covers/)（PR C3 提供）

---

### 🥈 方案 2：暖驼底 + 篆体「渝」字（最易复制 · 最暖）

**为什么推荐**：
- 暖驼是主理人本命色，强化"色彩 = 我"的人设关联
- 单字最克制，缩多小都能看清
- 30 分钟可做完

**视觉描述**：
```
- 圆形头像 512×512
- 底色：暖驼 #C9A66B（nuantuo）
- 主体：超大「渝」字（篆体 / 方正小篆繁），墨青 #2C3E50
  字号：360+，居中
- 副文字：底部小字「东方美学」墨青 80%，思源宋体
- 角标：右上角一个小朱砂方印（20% 大小）
```

**HTML 模板**：[`covers/avatar-yu-camel.html`](./covers/)（PR C3 提供）

---

### 🥉 方案 3：色卡风（19 色身份证）

**视觉描述**：
```
- 圆形头像 512×512
- 底色：米白 #FAF7F2
- 主体：一组 19 色矿物色块组成的色环
  色块按色相过渡（朱砂 → 朱橘 → 暖驼 → 米白 → 嫩绿 → 墨青 → 紫雾）
  排成一个内环（缩到 40 px 时仍然显眼）
- 中央：墨青小字「渝山」+ 朱砂小印
- 整体克制，矿物质感
```

适合：希望强化"19 色"卖点的人。但识别度略弱，不如方案 1/2。

**HTML 模板**：[`covers/avatar-19-colors.html`](./covers/)（PR C3 提供）

---

### 方案 4：宋画团扇风（最雅致）

```
- 圆形头像 512×512（本身就是团扇形状）
- 底色：米白 + 米麦渐变
- 主体：一支宋画风的折枝（如朱砂红梅 / 暖驼银杏 / 嫩绿柳条）
  从右上角伸入画面 1/3
- 中央偏左：墨青「渝山」二字（思源宋体 Heavy）
- 四周：极细的墨青描边
- 风格：极简、留白、雅
```

适合：偏向高客单（D 档）潜在客户的视觉印象。

---

### 方案 5：主理人侧影（真人 IP）

```
- 圆形头像 512×512
- 底色：墨青 #2C3E50
- 主体：主理人本人的侧影剪影（米白色），穿暖驼大衣
  头略低，看不出五官细节，只看出"东方女性"轮廓
- 角标：右下角朱砂方印「渝山」
- 风格：克制神秘
```

适合：主理人本人愿意露面、想做强 IP 的方向。
**注意**：用真人剪影，不要清晰正脸——避免被 AI 换脸或被冒充。

---

### 方案 6：极简大字版（最商务）

```
- 圆形头像 512×512  
- 底色：墨青 #2C3E50
- 主体：超大「渝」字
  字体：思源宋体 Heavy 或方正小篆繁
  颜色：暖驼 #C9A66B，带轻微浮雕立体感
- 底部小字：「YUSHAN · 东方美学」（米白）
```

适合：B 端 BD 用，名片印章感最强。

## 八、头像进阶玩法

### 节气版本

保留主视觉，背景 / 配饰随 24 节气换：
- 立春：背景换嫩绿 #8FB573 + 一支柳条
- 立夏：背景换朱橘 #E67E22 + 一颗柿子
- 立秋：背景换暖驼 #C9A66B + 一片麦穗
- 立冬：背景换米白 + 一枝寒梅
- 春节：印章颜色加深为深朱砂 + 一对小灯笼角标
- 中秋：背景换月白 #FAF7F2 + 一轮淡墨月

> 节气头像建议每节气前 3 天换上，节气后 7 天换回主头像。
> 不建议每周换——会损失"账号是 XX"的稳定识别。

### 篆体印章延展（最强 IP 资产）

以「渝山」朱砂方印为核心，做一套：
- 圆印：用于主头像
- 方印：用于落款（每张笔记右下角，参考 [`covers-design.md`](./covers-design.md)）
- 长条印：用于公众号底部签名
- 小贴纸印：用于评论区表情包

> 这套印章是**渝山品牌的核心视觉资产**，强烈建议第 1 周就找一位会篆刻的设计师做出 PNG 透明底矢量，
> 后续所有视觉物料都从这一套里调用。

## 九、快速决策：哪个方案适合你？

```
如果你：
  ✓ 第一次起号 / 想用最强识别度          → 方案 1（朱砂印「渝山」）
  ✓ 想强化"色彩 = 我"的人设               → 方案 2（暖驼篆「渝」字）
  ✓ 想突出"19 色"产品差异化               → 方案 3（色卡风 19 色身份证）
  ✓ 主打高雅定位 / D 档高定客户           → 方案 4（宋画团扇折枝）
  ✓ 主理人愿意做真人 IP                    → 方案 5（侧影剪影）
  ✓ 主打 B 端 BD                          → 方案 6（极简大字版）
```

### 我个人最推荐：**方案 1（朱砂印「渝山」）**

- 识别度天花板（缩到 40 px 还认得出）
- 与 9 宫格右下角印章呼应（IP 一致性最强）
- 兼容性最好（所有平台都能识别）
- 制作最简单（HTML 模板 + 截图）
- 后续延展空间最大（节气版 / 不同字体 / 表情包）
