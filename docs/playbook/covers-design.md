# A. 9 宫格封面设计规范（第 1 篇笔记）

> 笔记标题：打工人必装的 5 个 AI 神器，用过真的回不去🔥
> 工具推荐：Canva / 稿定设计 / 创客贴
> 画布尺寸：**1242 × 1660 px**（3:4 竖图，小红书首图黄金比例）

---

## 一、统一视觉规范（9 张图都用）

### 配色方案（暖黄奶油风，年轻打工人最吃的一套）

| 色值 | 用途 | 场景 |
|------|------|------|
| `#FFF8E7` | 奶油白 | 背景主色 |
| `#FFD93D` | 暖黄 | 强调块/色块高亮 |
| `#2D2D2D` | 深炭黑 | 主标题/大字 |
| `#FF6B6B` | 番茄红 | 钩子字/数字/重点 |
| `#6BCF7E` | 薄荷绿 | 辅助点缀 |

### 字体

| 用途 | 字体（免费商用） |
|------|----------------|
| 主标题大字 | **阿里巴巴普惠体 Heavy** |
| 副标题 | **思源黑体 Bold** |
| 正文小字 | **思源黑体 Regular** |
| 数字强调 | **站酷庆科黄油体** 或 **HarmonyOS Sans** |

### 边距和安全区

- 上下各留 **80 px** 安全区（防被 UI 遮挡）
- 左右各留 **60 px**
- 核心信息集中在画面 **中央 70% 区域**

## 三、Canva 实操速成（5 分钟搞定）

### Step 1：新建画布
- 打开 Canva → 自定义尺寸 → 1242 × 1660 px
- 或直接搜"小红书封面"模板

### Step 2：导入配色
- 创建品牌色板：`#FFF8E7` / `#FFD93D` / `#2D2D2D` / `#FF6B6B` / `#6BCF7E`
- 每张图都从色板拖色，保证视觉统一

### Step 3：套用模板
- Canva 搜"小红书 干货"，挑一个对齐的模板
- 复制 9 份，分别改文字

### Step 4：批量操作
- 第 1 张做好后右键 → "复制页面" → 改文字
- 保证字体、字号、色块位置完全一致

### Step 5：导出
- 导出格式：**PNG 高清**
- 文件名按顺序命名：`01-封面.png` ... `09-引导.png`
