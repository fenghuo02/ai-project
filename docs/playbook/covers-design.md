## 一、统一视觉规范（9 张图都用）

### 配色方案（东方美学 · 黛青暖驼调）

> ⭐ 所有色值必须从 [`swatch-builder/palette/colors.yaml`](../../swatch-builder/palette/colors.yaml) 取。
> 本表的 5 个色值就是从 19 色里精选的封面专用子集。

| HEX | 名 | colors.yaml id | 用途 |
|------|------|------|------|
| `#FAF7F2` | 米白 | `mibai` | 背景主色 / 大留白 |
| `#F5E6C8` | 米麦 | `mimai` | 次底色 / 浅色块 |
| `#2C3E50` | 墨青 | `moqing` | 主标题 / 大字 |
| `#C9A66B` | 暖驼 | `nuantuo` ⭐ | 强调块 / 钩子色 |
| `#C0392B` | 朱砂 | `zhusha` | 数字 / 印章 / 重点 |
| `#D4AF37` | 烛光金 | `zhuguangjin` | 高定感点缀（限量用，不超过 5%）|

> 取色原则：**整体黛青 + 米白打底 → 暖驼做强调 → 朱砂做点睛**。
> 不要往里加奶油黄、薄荷绿、番茄红——那是 AI 工具号的旧体系，已废弃。

### 字体（免费商用）

| 用途 | 字体 | 备选 |
|------|------|------|
| 主标题大字 | **思源宋体 Heavy / Noto Serif CJK SC** | 站酷文艺体 / 方正宋刻本秀楷 |
| 副标题 | **思源黑体 Bold / Noto Sans CJK SC** | 阿里巴巴普惠体 Bold |
| 正文小字 | **思源黑体 Regular** | 苹方常规 |
| 数字强调 | **Bodoni 72 Bold** + 思源宋体 | DIN Bold |
| 落款 / 印章 | **方正小篆体** | 汉仪小篆繁 |

> 字体调性建议：**宋 → 雅；黑 → 现代**。封面主标用宋体出东方感，避免全黑体的"工具号"感。

### 边距和安全区

- 上下各留 **80 px** 安全区（防被 UI 遮挡）
- 左右各留 **60 px**
- 核心信息集中在画面 **中央 70% 区域**
- 右下角永远留出 **160 × 60 px** 的「主理人印章 / 渝山」位置

### 视觉锚点（重要，强化人设记忆）

每张图右下角必须有一枚**朱砂印章风**小标识：

```
┌──────────────┐
│              │
│              │
│              │
│       ┌────┐ │
│       │ 渝山│ │ ← 朱砂红 #C0392B 圆/方印
│       │  色 │ │   思源宋体 / 篆体
│       └────┘ │
└──────────────┘
```

字可以是「渝」「山」「色」三选一组合，或固定「渝山东方美学」。
字体首选篆体或宋体粗体。**这个标识 9 张图都必须出现，是 IP 锚点**。

## 三、Canva 实操速成（5 分钟搞定）

### Step 1：新建画布

- 打开 Canva → 自定义尺寸 → 1242 × 1660 px
- 或直接搜「**小红书封面**」模板

### Step 2：导入品牌色板

创建品牌色板，**只用这 5+1 个色**（不要往里加奶油黄、番茄红、薄荷绿）：

```
#FAF7F2  米白 mibai      ─ 背景
#F5E6C8  米麦 mimai      ─ 次底色
#2C3E50  墨青 moqing     ─ 主标题
#C9A66B  暖驼 nuantuo    ─ 强调（★）
#C0392B  朱砂 zhusha     ─ 数字 / 印章
#D4AF37  烛光金 zhuguangjin ─ 高定点缀（限量）
```

每张图都从色板拖色，保证视觉统一。

### Step 3：套用模板 / 用本仓库 HTML

- Canva 搜「**新中式 干货 笔记**」或「**国风 手册**」，挑一个对齐的模板
- 或者直接打开 [`covers/cover-01-main.html`](./covers/cover-01-main.html)（仓库内自带模板，浏览器开 → 截图）

### Step 4：批量复制改字

- 第 1 张做好后右键 → "复制页面" → 改文字
- 字体、字号、色块位置完全一致——保持"一系列"的感觉

### Step 5：导出

- 导出格式：**PNG 高清**
- 文件名按顺序命名：`01-封面.png` ... `09-引导.png`
- 每张文件名前缀对齐，发布时按字母顺序自动排好
