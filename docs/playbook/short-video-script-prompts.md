# 三平台同发短视频脚本生成器

> 目标：**一支拍摄 / 一套素材 → 抖音版 + 视频号版 + B 站版 三条脚本**
> 用法：每次拍完素材，按 §二 把信息填进 Prompt 1，AI 一次性吐出 3 个平台版本，你直接照着剪
> 版本：v1.0 · 2026 春（与业务计划书 v1.5 §3 「打法 1：抖音 + 视频号 + B 站三平台短视频同发」 + [`bilibili-onboarding.md`](./bilibili-onboarding.md) §2.3 配套）

---

## ⚠️ 平台合规底线（开工前必读）

短视频是**同时进 3 个平台审核系统**的内容——任何一个平台禁的词都不能用。

### 三件套对外命名（强制 · 详见 [`.kiro/steering/01-compliance-redlines.md`](../../.kiro/steering/01-compliance-redlines.md) §五）

| 内部沟通用语 | 短视频脚本里**只能**用 |
|---|---|
| 转运肖像 / 五行肖像 | **命主画像** ⭐ / 五行美学肖像 |
| 转运壁纸 / 开运壁纸 | **节气壁纸** ⭐ / 应景壁纸 |
| 命理书 / 命理报告 | **命书** ⭐ / 美学命书 |
| 转运照片 / 开运照片 | **节气写真** / 应景写真 |

### 三平台共同禁词（任何一个平台触发都全平台限流）

```
L1：算命 / 看相 / 八字 / 用神 / 调候 / 命格 / 奇门 / 八门 / 死门 / 惊门
L2：测命 / 化解 / 转运 / 改运 / 改命 / 开运 / 风水 / 吉凶 / 命运 / 运势 / 玄学 / 命理 / 大师 / 师傅
L3 否定句式陷阱：不算命 / 不是算命 / 不预测 → 用「不下结论、不做预测」
L4：微信 / VX / 加我 / 私聊 / 免费送 / 绝对 / 永远 / 包治 / 包准
L5 天干日主（短视频 100% 禁用）：丁火 / 庚金 / 壬水 / 乙木 等
```

> ⚠️ **L5 短视频和 B 站长视频差异**：B 站长视频可适度提天干日主（学术科普框架），但**短视频任何平台都禁用**——短视频时长太短，没有空间做「学术框架」铺垫，必然触发"算命"联想。

### B 站短视频独有约束

- 投稿设置必须勾选「视频内容由 AI 生成或合成」（含 AI 出图时）
- 投稿分区**绝不**选「玄学占卜 / 迷信民俗」，应选「生活 → 美妆 / 日常 / 家居家装」或「知识 → 文化习俗 / 设计创意」
- 详见 [`bilibili-onboarding.md`](./bilibili-onboarding.md) §一

## 一、三平台用户差异速查表（决定脚本调性）

| 维度 | 抖音 | 视频号 | B 站 |
|---|---|---|---|
| **用户画像** | 18-40 岁，重庆同城为主 | 25-50 岁，主理人微信熟人圈 + 朋友推荐 | 18-30 岁，知识型/审美型/设计型用户 |
| **观看场景** | 通勤 / 排队 / 睡前 | 朋友圈刷到 / 公众号底部入口 | 主动搜索 / 推荐流深度浏览 |
| **首屏耐心** | **0-3 秒**（划走概率最高）| 3-5 秒（半熟人光环）| **5-10 秒**（用户预期看「内容」）|
| **理想时长** | 30-45 秒 | 45-60 秒 | 60-90 秒（短视频上限） |
| **字幕密度** | 高（每 2-3 秒一行）| 中（每 3-4 秒一行）| 中低（每 4-6 秒一行）|
| **配乐风格** | 节奏强、重低音、网络热曲 | 温和、人声、家人朋友感 | **古风 / 民谣 / 纯音乐** |
| **结尾 CTA** | "评论 1 想要" / "私信暗号"（强）| "公众号回复 X 领" / "看更多在主页" | **不放强 CTA**，仅 "想了解更多关注我" |
| **强项内容形态** | 强对比 / 反差 / 数字承诺 | 故事感 / 人格 / 真实感 | 知识 / 慢节奏 / 文化考据 |

## 四、Prompt 3：5 类核心选题模板（直接复用 ⭐）

> 主理人开始时不会写脚本？以下 5 个模板直接套用，前 1-2 个月跑通后再用 Prompt 1 自由创作。
