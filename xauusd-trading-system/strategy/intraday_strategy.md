# XAUUSD 日内交易策略

> 适用周期：M5 / M15 入场，M30 / H1 过滤
> 目标：每日 1–3 笔高质量交易，单笔盈亏比 ≥ 1:1.8
> 风格：**顺势 + 关键水平回踩 / 突破回测**，不做反转、不做震荡区中段

---

## 1. 时段规则（北京时间）

| 时段 | 时间 | 操作策略 |
|------|------|----------|
| 亚盘 | 07:00–14:00 | **不主动交易**，只观察、记录亚盘高低（HOD/LOD） |
| 欧盘前 | 14:00–15:00 | 准备工作：读 D1 / H4 结构，标好关键水平 |
| **欧盘** | **15:00–18:00** | **黄金时段 1**：欧盘开盘后常出现假突破亚盘高低 → 反向机会 |
| 伦敦修整 | 18:00–20:30 | 谨慎，多为震荡 |
| **美盘前** | **20:30–21:30** | 数据公布期，**有数据则停手 ±15 分钟** |
| **美盘** | **21:30–01:00** | **黄金时段 2**：趋势延续 / 反转最常发生 |
| 美盘尾段 | 01:00–04:00 | 不做新单，只管理持仓 |

**核心原则**：**只在欧盘和美盘前半段开新单**，其他时间只管理已有仓位。

---

## 2. 入场模型（A / B / C 三套）

### 模型 A：关键水平回踩入场（首选，胜率最高）

**适用场景**：D1/H4 趋势明确（EMA50 与 EMA200 同向），价格回踩到关键支撑/阻力。

**入场条件**（必须全部满足）：
1. 主分析师剧本方向 = 当前操作方向
2. 价格触及以下任一关键水平 ±3 美元：
   - 前日高 PDH / 前日低 PDL
   - 亚盘高 / 亚盘低
   - H4 EMA21 / EMA50
   - 整数关口（每 10 美元，2300 / 2350 等）
3. M15 出现确认 K 线形态：
   - 多头：看涨吞没 / 锤子线 / 三连阳
   - 空头：看跌吞没 / 流星线 / 三连阴
4. RSI(14) 在 M15 上从超买 / 超卖区反转：
   - 多头：RSI 从 < 35 上穿 40
   - 空头：RSI 从 > 65 下破 60
5. **价格在 VWAP 同侧**（多头在 VWAP 上方，空头在下方）

**订单参数**：
- 入场：M15 确认 K 线收盘价 ± 0.5 美元限价
- 止损：确认 K 线另一端外 + 0.3×ATR(14) 缓冲
- 止盈 1：盈亏比 1:1（先平 50%，剩余移盈亏平衡）
- 止盈 2：下一个关键水平 -2 美元
- 最大持仓时间：4 小时未到 T1 → 手动平仓

---

### 模型 B：突破回测入场（次选）

**适用场景**：盘中突破亚盘 / 欧盘高低后，回踩突破位不破。

**入场条件**：
1. 价格突破关键水平 ≥ 3 美元后回踩
2. 回踩时 M5 不收破突破水平
3. 回踩成交量明显小于突破时成交量
4. EMA9 (M15) 与突破方向一致

**订单参数**：
- 入场：突破水平 ± 0.5 美元
- 止损：突破水平反向 1.0×ATR(14, M15)
- 止盈：突破前的箱体高度 1:1 投影

---

### 模型 C：假突破反转（高难度，不熟练者跳过）

**适用场景**：欧盘开盘 30 分钟内，价格刺破亚盘高/低后立即收回。

**入场条件**：
1. 仅在 15:00–15:45 北京时间使用
2. 价格刺破亚盘高/低 ≥ 5 美元
3. 5 分钟内回到亚盘区间内
4. 此时 RSI 在另一端出现背离

**订单参数**：
- 入场：回到亚盘区间内 2 美元处
- 止损：刺破极值外 2 美元
- 止盈：亚盘区间另一端

> ⚠️ 模型 C 月胜率约 55–60%，但盈亏比可达 1:3+。建议先用小仓位练 30 单后再放量。

## 3. 离场规则（比入场更重要）

### 主动止盈
- 触及预设止盈位 → 按计划平仓
- 触及关键水平 + 反向 K 线形态 → 立即平仓不等止盈

### 主动止损
- 触及结构止损位 → 必须平仓，**禁止挪止损**
- 持仓 4 小时未到 T1 → 手动平仓（时间止损）
- 出现以下情况立即手动平仓：
  - K 线形态完全反向（如多单出现 M15 看跌吞没在阻力处）
  - DXY 突然反向加速（与金价负相关失效）
  - 突发地缘事件 / 央行讲话

### 移动止损
- 浮盈 = 1R：止损移到入场价（盈亏平衡）
- 浮盈 = 2R：止损移到 +1R
- 浮盈 = 3R 以上：用 M15 EMA21 跟踪止损

## 4. 一日交易模板（早盘准备 → 收盘复盘）

### 14:00 准备阶段（10 分钟）
- [ ] 用 `prompts/01_master_analyst.md` 让 AI 输出当日剧本
- [ ] 在 TradingView 标好：PDH / PDL / 亚盘高低 / 整数关口
- [ ] 检查日历：今日有无 CPI / 非农 / FOMC

### 15:00–18:00 欧盘执行
- [ ] 等待信号出现（指标提示 + 价格行为）
- [ ] 信号出现 → 用 `prompts/02_intraday_signal.md` 二次确认
- [ ] GO → 下单；WAIT → 继续等；NO-GO → 跳过
- [ ] 单笔最大风险 = 账户 1%

### 21:30–01:00 美盘执行
- [ ] 21:15–21:45 不开新单（数据时段）
- [ ] 22:00 后按欧盘相同流程执行
- [ ] 累计盈利 ≥ 3R 后**减半仓位**保护成果
- [ ] 累计亏损 ≥ 2R 立即收工

### 01:30 收盘复盘（5 分钟）
- [ ] 截图所有交易写交易日志
- [ ] 标注：信号质量分 / 执行偏差 / 经验教训
- [ ] 更新次日 PDH / PDL

## 5. 红线（违反任何一条 → 当日停止交易）

1. ❌ 单日亏损 ≥ 账户 3%
2. ❌ 连续 2 单亏损（无论金额）
3. ❌ 在「不交易时段」开仓
4. ❌ 主分析师剧本明确写「今日观望」时仍交易
5. ❌ 数据公布前 ±15 分钟开仓
6. ❌ 挪动止损扩大风险
7. ❌ 报复性补仓 / 反手
