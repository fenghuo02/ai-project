# XAUUSD 趋势交易策略

> 适用周期：H4 / D1 入场，W1 定方向
> 持仓时间：1–10 个交易日
> 单笔目标盈亏比：≥ 1:2.5（理想 1:5）
> 风格：**只做回调，不做反转；多周期共振，宁缺毋滥**

---

## 1. 趋势判定（三层过滤）

### 第一层：W1（周线）— 决定能不能做
- **多头条件**：W1 收盘 > EMA20(W1)，且 EMA20 > EMA50
- **空头条件**：W1 收盘 < EMA20(W1)，且 EMA20 < EMA50
- **不交易**：W1 EMA20 与 EMA50 距离 < 1×ATR(W1) → 视为震荡

### 第二层：D1（日线）— 决定方向
- **多头有效**：D1 EMA50 上倾，最近 5 根 D1 至少 3 根高低点抬升
- **空头有效**：D1 EMA50 下倾，最近 5 根 D1 至少 3 根高低点压低
- **结构破坏**：D1 收盘破前 swing 低/高 → 趋势中断，立即清仓

### 第三层：H4 — 决定入场时机
- 只有当 D1 处于「**趋势中的回调**」状态才入场
- 回调判定：H4 RSI(14) 进入超买 / 超卖反向区
  - D1 多头时：H4 RSI 跌至 35–45 → 准备做多
  - D1 空头时：H4 RSI 升至 55–65 → 准备做空

---

## 2. 入场模型

### 模型 T1：EMA 回踩（最经典，胜率 ~65%）

**条件**（D1 多头为例，空头反向）：
1. W1 + D1 共振多头
2. 价格回踩到 D1 EMA20 或 H4 EMA50（取较高者）
3. H4 出现以下任一确认：
   - 看涨吞没 / 锤子线
   - H4 RSI 从 < 40 上穿 45
   - H4 MACD 柱状从负转正
4. 当前价距离 D1 EMA200 ≤ 3×ATR（防止过度延伸）

**订单参数**：
- 入场：H4 确认 K 线收盘价
- 止损：回调 swing 低 - 0.5×ATR(H4)
- 止盈 1：1:2 盈亏比（平 1/3 仓位）
- 止盈 2：前 swing 高（平 1/3 仓位）
- 止盈 3：1.618 斐波扩展位（剩余 1/3 用 D1 EMA20 跟踪）

---

### 模型 T2：突破回测（趋势加速时使用）

**条件**：
1. D1 收盘突破前关键阻力 / 支撑 ≥ 0.5×ATR(D1)
2. 突破后 1–3 个交易日内回踩突破水平
3. 回踩时 H4 不收破突破水平
4. 回踩成交量明显萎缩

**订单参数**：
- 入场：突破水平 ± 0.3×ATR
- 止损：突破水平反向 1.0×ATR(D1)
- 止盈：突破前箱体高度 × 1.618 投影

---

### 模型 T3：流动性扫荡反转（进阶，仅 W1 趋势末端使用）

**条件**：
1. 价格刺破多个交易日 swing 高/低 ≥ 1×ATR(D1)
2. 刺破当日收盘回到原区间内
3. 第二日开盘延续反向，且 D1 出现吞没
4. 配合 DXY 反向 + 美债收益率反向（宏观共振）

**订单参数**：
- 入场：第二日 D1 收盘 +/- 0.3×ATR
- 止损：刺破极值外 1.0×ATR(D1)
- 止盈：分批，T1=1:2，T2=1:3，T3=1:5

> ⚠️ T3 模型每年机会 4–8 次，但单笔可贡献年化 30%+。建议盯到形态再交易，不要硬找。

## 3. 加仓与持仓管理

### 加仓规则（金字塔加仓）
- 第 1 仓：100% 基础仓位
- 浮盈 ≥ 1.5R 且价格再次回踩 H4 EMA21 → 加 50% 基础仓位
- 浮盈 ≥ 3R 且 D1 收出新高/新低 → 再加 25% 基础仓位
- **总仓位上限**：175% 基础仓位（不无限加仓）
- **加仓止损**：所有仓位的止损统一移到 +0.5R

### 移动止损（必须执行）
| 浮盈 | 止损位置 |
|------|----------|
| 1R | 入场价（盈亏平衡） |
| 2R | +1R |
| 3R+ | D1 EMA20 跟踪 |
| 5R+ | H4 EMA50 跟踪（更紧） |

### 强制平仓信号（出现任一立即清仓）
- D1 收盘出现反向**完全吞没** + 长上/下影线
- D1 收盘破 EMA50（趋势核心被打破）
- W1 RSI 进入极端区（>75 / <25）+ 当周高低反转
- 重大政策反转（如 FOMC 鹰转鸽 / 鸽转鹰）

## 5. 趋势交易者的心理纪律

1. **不预测顶部 / 底部**。趋势会比你想象的走得更远。
2. **拿不住单 = 没有入场资格**。如果浮盈 2R 你就想跑，说明你的入场质量不够高。
3. **回调 = 朋友，不是敌人**。趋势中的回调让你拿到便宜筹码，不是反转信号。
4. **错过比错做更可贵**。这周没机会就关掉软件。
5. **趋势单的盈亏比不是 1:2，是 1:5+**。如果你设 1:1 跑路，长期一定亏。
