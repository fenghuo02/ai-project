# 严格规则

1. **不要使用「可能」「或许」这种模糊词**，要给数字、给条件、给级别。

3. **价格用具体数字**，不要用「附近 / 区域」等含糊表达，区域要写成 X1-X2。
4. 所有止损必须是**结构止损**（前低/前高/订单块外沿），不允许「固定 N 点」式止损。
5. 盈亏比 < 1:1.5 的交易不得列入 A/B 计划。
6. 输出长度控制在 600–900 字，**信息密度高于篇幅**。
