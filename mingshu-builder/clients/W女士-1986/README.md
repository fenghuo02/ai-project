## 一、命局推导（内部工作记录）

### 八字盘

| 年柱 | 月柱 | 日柱 | 时柱 |
|---|---|---|---|
| 丙寅 | 丁酉 | 己未 | 壬申 |
| 火木 | 火金 | 土土 | 水金 |

- **日主**：己土，生于酉月（金旺月），仲秋
- **五行势力**：火2（丙、丁双透）/ 木1（寅）/ 土3（己、未含库）/ 金3（酉、申）/ 水1（壬时）
- **强弱**：日主**偏弱**（酉月金旺泄己土；壬水时上侵）
- **调候用神**：**丙火 / 丁火**（年月双透，火生土暖局）⭐
- **喜神**：戊土（帮身固根）
- **忌神**：金（再加申酉，泄土愈盛）
- **类型一句**：**秋土待火 · 金叶满径**

## 三、本地生成 PDF

### 校验配置（30 秒）

```bash
cd mingshu-builder
python3 build.py --validate W女士-1986
```

预期输出：
```
[*] 校验 W女士-1986 ...
  [i] 图片：找到 0/40（找不到的会自动 CSS fallback）
  [!] 以下 40 个 key 在 images/ 没找到：portrait, kai, ...
[✓] 校验通过 · 1 条警告
```

### 生成 PDF（3-5 分钟）

```bash
# 首次需安装依赖（Mac）
brew install pango cairo gdk-pixbuf
pip install pyyaml jinja2 weasyprint

# 生成
cd mingshu-builder
python3 build.py W女士-1986
```

输出：
```
mingshu-builder/clients/W女士-1986/
├── output.html        ← 调试用 HTML
└── output.pdf         ← ⭐ 39 页命书 PDF（约 400KB）
```

> **沙盒环境跑不了**（缺 weasyprint），必须在主理人本地 mac/win 跑。
> Windows 推荐方案：用 GitHub Codespace（在 PR 里点 `Code → Codespace`）。

## 五、用作路演 demo 的 3 种姿势

### 姿势 1 · 现场口头介绍（5 分钟）

老师面前打开本 README，按下面顺序讲：

1. "这是 W 女士，1986 年生，八字 丙寅丁酉己未壬申"
2. "她日主己土，生于酉月，金旺泄气，需要丙丁火来生根" — 看「§一 命局推导」
3. "我们给她推了本命主色朱砂、辅色暖驼" — 看「配色推导」
4. "本命书会自动生成 39 页 PDF" — 打开 `output.pdf`（本地预生成的）
5. "搭配 6 张五行肖像 + 24 节气壁纸" — 打开 `images/jing.jpg` 给老师看

### 姿势 2 · iPad 翻页演示（10 分钟）

把 `output.pdf` 用 PDF 阅读器打开 iPad，逐页翻：

- P.1-P.5 序言 + 本命色 → "看，这是她的色卡"
- P.6-P.21 八门 → "这是她适合的 8 个生活场景，按 ta 的命局自动配色"
- P.22-P.28 七节日 → "她全年的节庆色彩"
- P.29-P.35 24 节气 → "全年 24 张应景行动建议"
- P.36-P.39 终章 + 色卡日历 → "包括最后的色卡日历"

### 姿势 3 · 把这本 PDF 打印一本（最震撼）

打印一份 39 页 PDF（彩色铜版纸 + 装订）放工作室：

- 老师面前**不放电脑**，直接拿出实物书
- 老师摸到纸的那一刻，理解这是认真的产品而不是 demo
