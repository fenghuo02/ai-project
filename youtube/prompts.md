# YouTube 创作者 AI 提示词手册

> 使用方式：每开一个新会话，先粘贴【0. 角色设定】让 AI 进入"YouTube 顾问"模式，
> 然后按需粘贴对应模块的提示词，把 `【...】` 里的变量替换成你的内容。

---

## 0. 角色设定（每个新会话先发这段）

```
你现在是我的 YouTube 频道首席顾问，同时扮演三个角色：
1. 内容策略师：精通 MrBeast、Ali Abdaal、Veritasium、李永乐、影视飓风等头部频道的成长路径与选题模型。
2. 脚本编剧：擅长 Hook-Retention-Payoff 结构，熟悉 YouTube 的 AVD（平均观看时长）与 CTR（点击率）优化。
3. 增长运营：熟悉 YouTube 算法、SEO、Shorts 分发逻辑、以及 AdSense / 品牌合作 / 联盟 / 会员 / 自营产品 五种变现路径。

回答规则：
- 默认用中文，除非我要求英文。
- 给出的建议必须可执行、带数据或案例支撑，避免空话。
- 当我提供的信息不足以给出高质量答案时，先反问我 2-3 个关键问题再动笔。
- 每次输出结尾加一个"下一步建议"，告诉我接下来最该做的一件事。

我的频道信息（如果还没有就写"待定"）：
- 语言：【中文 / 英文 / 双语】
- 方向：【例如：AI 工具教学 / 个人成长 / 科技评测 / Vlog / 知识科普】
- 目标受众：【年龄、职业、痛点】
- 变现目标：【例如：6 个月内开通 AdSense / 一年做到月入 1 万美元】
- 我的资源：【每周可投入时间、设备、专业背景、是否出镜】

收到请回复"已就位"并向我提问补全以上信息。
```

## 1. 频道定位诊断（第一步，最重要）

> 90% 的新人死在这一步——选了一个自己喜欢但没人搜索、或者竞争已经红海的方向。

```
请对以下频道方向做一次"定位可行性诊断"：

方向候选：【A. xxx / B. xxx / C. xxx】
我的优势：【你的专业/经验/资源】
每周产能：【能出几条长视频 + 几条 Shorts】

请从 5 个维度各打 1-10 分，并给出理由：
1. 需求规模（YouTube 上这个主题的总搜索/观看量级）
2. 竞争强度（头部账号有多强，新人突围难度）
3. 变现天花板（CPM 范围、是否容易接广告/卖课/带货）
4. 我的契合度（我的资源和这个方向的匹配度）
5. 长期护城河（3 年后这个方向会不会被 AI/同质化吞噬）

最后给出：
- 推荐方向 Top 1 及理由
- 一个可以验证这个方向的"30 天最小实验"方案（出几条视频、怎么衡量成功）
```

## 2. 选题挖掘（批量产出）

> 一次生成 20 条候选，你从中挑 3-5 条真正有感觉的，这样命中率最高。

```
基于我的频道定位【XXX】，请帮我挖掘 20 个视频选题，要求：

选题必须满足以下至少 2 条：
- 有明确的搜索需求（可以写成 "How to..."、"为什么..."、"..排行榜"、"..对比" 等形式）
- 有情绪钩子（争议、反常识、惊奇、焦虑、爽感）
- 可以在 8-15 分钟讲清楚
- 我能用现有资源拍出来（不要让我飞去南极）

输出格式（表格）：
| 序号 | 选题标题（带 hook） | 选题类型 | 预估搜索热度 | 预估 CTR | 我可执行的理由 |

类型分布要求：
- 5 个"教学/How-to"型（长尾 SEO 流量）
- 5 个"榜单/对比"型（高 CTR）
- 5 个"故事/人物"型（高完播）
- 5 个"热点/争议"型（爆款潜力）

最后推荐其中 3 个作为我的"前 10 期视频"的开篇。
```

## 3. 脚本大纲（先定结构再写正文）

```
请为这个选题写一份"高留存"视频大纲：

选题：【xxx】
时长目标：【8-12 分钟】
目标观众：【xxx】

结构要求（严格按 YouTube 留存曲线优化）：
1. 冷开 Hook（0-15 秒）：
   - 一句话价值承诺 + 一个视觉/情绪钩子
   - 给我 3 个不同风格的 Hook 候选（悬念型 / 反常识型 / 利益型）

2. 过渡 + 频道介绍（15-30 秒）：
   - 最多 10 秒自我介绍，并埋一个"继续看下去的理由"

3. 正文 3-5 个 Chapter：
   - 每个 Chapter 结尾设置一个"下一段预告"防止划走
   - 标注每段的"情绪波动"（好奇→顿悟→爽）

4. 高潮/反转（视频 70% 位置）：
   - 一个让观众觉得"赚到了"的核心信息

5. 结尾 CTA（最后 30 秒）：
   - 不要求"点赞订阅"，改成"给出下一个行动"（看下一条视频 / 下载 / 评论区讨论）

同时在大纲中标注：
- [B-ROLL] 需要插入的画面
- [字幕强调] 需要放大字幕的关键词
- [音效] 需要加音效的位置
```

## 4. 完整脚本（口语化 + 可直接念）

```
基于以下大纲，写成完整口播脚本：

【粘贴第 3 步产出的大纲】

要求：
- 口语化，第二人称"你"，让观众感觉你在单独跟他说话
- 平均每 20-30 秒一个"留存钩子"（设问、预告、反转、数据）
- 避免"众所周知"、"毋庸置疑"这类空话
- 专业术语第一次出现时用一句话解释
- 句子尽量短，方便念和剪辑
- 在需要停顿/换镜头的地方用 "//" 标记
- 每段开头用【时间戳 xx:xx】标注

总字数控制在 【时长 × 300 字】 左右（中文）。
```

## 5. 标题 A/B 生成器（CTR 是命脉）

```
为这期视频生成 10 个标题候选：

视频主题：【xxx】
核心价值：【观众看完能获得什么】

标题要求：
- 中文≤20 字，英文≤60 字符
- 每个标题只用一种"钩子类型"（不要堆砌）
- 必须和内容匹配，不做标题党

类型分布：
1. 数字型（"我用 AI 30 天..."）
2. 悬念型（"没人告诉你的..."）
3. 对比型（"A vs B 谁更..."）
4. 身份型（"程序员为什么都在..."）
5. 反常识型（"别再...了，真相是..."）
6. 利益型（"一条视频学会..."）
7. 问题型（"为什么...？"）
8. 故事型（"我花了 1 万块，只为..."）
9. 榜单型（"2026 年最..."）
10. 情绪型（带感叹/惊讶）

每个标题附：
- 预估 CTR（低/中/高）
- 适合的缩略图画面
- 一句话点评
```

## 6. 缩略图方案

```
为这期视频设计 3 个缩略图方案：

标题：【xxx】
视频核心：【xxx】

每个方案给出：
1. 主体画面（我的表情 / 物品 / 场景）
2. 文字（≤ 5 个汉字或 3 个英文单词，字号要大）
3. 配色（主色 + 对比色，给 HEX 值）
4. 情绪（惊讶 / 好奇 / 疑惑 / 得意）
5. 与标题的"反差点"（缩略图和标题不要重复信息，要互相补充勾起好奇）
6. 在手机端 120px 小图下是否还能看清（验证）

最后推荐一个最优方案并说明理由。
```

## 7. 描述 + 标签 + 章节（SEO 套装）

```
为这期视频生成发布所需的全套元信息：

标题：【xxx】
脚本大纲：【粘贴大纲】

输出：

1. 视频描述（前 2 行最关键，要包含关键词并勾起点击）：
   - 第 1 段（前 150 字）：价值承诺 + 核心关键词自然植入
   - 第 2 段：章节时间戳（至少 5 个）
   - 第 3 段：相关链接 / 社交媒体 / 联盟链接位
   - 第 4 段：#话题标签（3-5 个）

2. Tags（15 个，从泛到精）

3. 固定评论（Pinned Comment）：
   一个能引发讨论的问题 + 一个行动引导

4. 结束画面（End Screen）建议：
   推荐下一条视频 + 订阅按钮位置
```

## 8. Shorts 拆条策略

```
把这条长视频拆成 5 条 Shorts：

长视频脚本：【粘贴脚本】

每条 Shorts 要求：
- 时长 30-60 秒
- 第 1 秒必须有钩子
- 独立成立，不依赖长视频
- 结尾引导"完整版在主页"或"下一条"

输出：
| 序号 | Shorts 标题 | 切入点（长视频的哪一段） | 开头 hook | 结尾 CTA |

额外：给 3 条"原生 Shorts 选题"（不是从长视频切的，而是专门为 Shorts 设计的轻量选题）。
```

## 9. 数据复盘与迭代

```
我这条视频的数据如下，请帮我做复盘：

- 标题：【xxx】
- 发布时间：【xxx】
- 曝光：【xxx】
- CTR：【xx%】
- 平均观看时长：【xx:xx】
- 平均观看百分比：【xx%】
- 观众留存曲线关键点：【30s 掉了 xx%、3min 掉了 xx%...】
- 订阅转化：【+xx】

请从 3 个维度诊断：
1. 标题/缩略图（看 CTR）：低于频道基线 / 同行水平吗？问题在哪？
2. 开头（看前 30 秒留存）：Hook 是否有效？
3. 中段（看留存曲线陡降点）：哪里观众大量流失？对应脚本的哪一段？

最后给出：
- 这条视频的 3 条具体改进项
- 下一条视频应该保留/避免什么
- 是否值得做"重发/改标题/改缩略图"的实验
```

## 10. 变现路线图

```
基于我的频道现状【订阅 xxx / 月播放 xxx / 主题 xxx】，
给我一份未来 6 个月的变现路线图。

要求覆盖 5 种变现方式并排出优先级：
1. AdSense（YPP 会员计划）
2. 品牌合作 / 赞助
3. 联盟营销（亚马逊、数字产品）
4. 自营产品（课程、电子书、模板、会员）
5. 频道会员 / Super Thanks

输出：
- 每个月的重点目标（订阅数、播放量、收入）
- 每个月该重点推进哪种变现
- 3 个需要提前准备的"资产"（比如邮件列表、落地页、产品 demo）
- 避坑清单（新手最容易犯的 3 个变现错误）
```

## 11. 竞品拆解（每月做一次）

```
分析这个竞品频道，帮我提炼可复用的打法：

频道：【频道名 / 链接】
我观察到的数据：【订阅数、近 10 条视频平均播放、爆款是哪条】

请输出：
1. 他的"选题公式"（近 10 条标题的共同模式）
2. 他的"开头公式"（前 15 秒都在做什么）
3. 他的"缩略图风格"（颜色、文字、画面构成）
4. 他的"更新节奏"和"视频结构"
5. 他的"变现方式"（从简介、置顶评论、视频内容推断）
6. 我可以直接抄的 3 个点
7. 我不能抄、必须差异化的 1 个点
```

## 使用顺序建议

**第 1 周：定位**
- 粘贴【0】+ 【1】→ 确定方向
- 粘贴【11】→ 拆 3 个竞品

**第 2 周：储备选题**
- 粘贴【2】→ 产出 20 个选题，储备 10 条视频的 idea

**每条视频流程：**
【3】大纲 → 【4】脚本 → 【5】标题 → 【6】缩略图 → 【7】描述 → 拍摄剪辑 → 发布 → 【8】拆 Shorts

**每条视频发布 7 天后：**
- 粘贴【9】→ 复盘数据

**每月一次：**
- 粘贴【10】→ 调整变现策略
- 粘贴【11】→ 追踪竞品变化

## 附：新人最容易踩的 5 个坑

1. **追求完美，第一条憋半年**——前 10 条视频的目的是练手和找感觉，不是做爆款。
2. **只做自己喜欢的**——YouTube 是"观众搜索什么"驱动的平台，不是博客。
3. **标题和缩略图随便做**——它们决定 CTR，CTR 决定推荐，推荐决定一切。
4. **不做 Shorts**——Shorts 是目前涨粉最快的流量入口，别放弃。
5. **没开通广告就急着接商单**——先把频道数据做好看，议价权天差地别。
