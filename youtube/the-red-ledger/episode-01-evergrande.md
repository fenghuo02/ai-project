## 📖 CHAPTER 1 — The Boy Who Sold Dreams (0:30 - 2:30)

[BGM: slow piano, minor key]
[B-ROLL: black and white photo of rural Henan, 1980s China]

In 1996, // a thirty-seven-year-old steel mill worker from Henan Province //
walked into a bank in Guangzhou // and asked for a loan //
to build something he had never built before. //
An apartment complex. ///

[B-ROLL: young Xu Jiayin archive photo, Guangzhou construction site 1990s]
[SUB: Xu Jiayin · 1996]

His name was Xu Jiayin. //
And over the next twenty-five years, // he would become //
the third richest man in Asia. ///

[B-ROLL: Forbes list graphic, "2017 · Asia's Richest"]

But here's what most people get wrong about this story. //
Xu Jiayin did not build a real estate company.

[SUB: HE DIDN'T BUILD A COMPANY]

He built // a machine. //
A machine that turned future promises // into present cash.

[B-ROLL: stylized animation — blueprint transforming into stacks of money]
[SFX: mechanical gears engaging]

Here's how it worked. //
In China, // developers are allowed to sell apartments //
*before* they're built. //
Buyers pay a deposit — // sometimes the *full price* — //
for a concrete box that doesn't exist yet.

[SUB: PRE-SALE = PAY FOR AIR]

The developer takes that cash, //
and legally, // they're supposed to use it //
to finish the building. ///

[B-ROLL: half-finished apartment tower]

But in the 1990s, // regulation in China was a suggestion, // not a law. //
So developers like Xu // started doing something else. //
They took the deposit from *one* building, //
and used it to buy land for the *next* one.

[B-ROLL: animated chain — money flowing from Project A to Project B to Project C]
[SUB: THE CASH CAROUSEL]

And the next one. // And the next. ///

By 2009, // Evergrande was listed in Hong Kong, // worth seven billion dollars. //
By 2017, // it was worth over fifty billion. //
Xu Jiayin was the richest man in Asia.

[B-ROLL: champagne, fireworks, Hong Kong IPO footage]

But the real trick // wasn't selling apartments. //
It was what he did with the money // *before* the apartments existed. ///

[SFX: ticking clock accelerates]

## 📖 CHAPTER 2 — The Three Sets of Books (2:30 - 5:00)

[BGM: shift to tense strings]
[B-ROLL: Evergrande corporate logo, then pull back to reveal map of China with 280 dots lighting up]

By 2020, // Evergrande had thirteen hundred projects //
in over two hundred and eighty Chinese cities.

[SUB: 1,300 PROJECTS · 280 CITIES]

Two hundred thousand employees. //
One point six million apartments sold //
to regular families // who had paid in full //
and were still waiting to move in.

[SUB: 1,600,000 FAMILIES WAITING]
[SFX: soft heartbeat]

Officially, // the company had three hundred billion dollars in debt. //
But here's what the headlines didn't tell you.

[B-ROLL: stylized three ledger books slamming onto a desk]

There were three sets of books. ///

[SUB: THREE SETS OF BOOKS]

The first set // was the public one. //
Audited. // Filed in Hong Kong. //
This is the three-hundred-billion number // you've probably heard.

[B-ROLL: annual report document, "$300B" circled]

The second set // was the banking set. //
Loans from Chinese state banks, // commercial paper, // trust products. //
Most of this was technically disclosed — //
but buried in footnotes // no foreign investor ever read.

[B-ROLL: close-up of footnotes, red pen circling them]

And the third set // was the one nobody talks about. ///

[BGM: drone deepens]
[B-ROLL: dark hallway, single door at the end]

Off-balance-sheet financing. //
Wealth management products sold //
to Evergrande's *own employees*. //
Guaranteed returns of nine to twelve percent, //
marketed as "safe." //
Eighty thousand people bought them. ///

[SUB: 80,000 EMPLOYEES · $6B LOST]
[SFX: single piano note, cold]

Teachers. // Nurses. // Retirees. //
Low-level staff who were *told* //
that buying these products // was part of being loyal to the company. ///

When the music stopped, // that money was gone. ///

[B-ROLL: empty wallet, scattered receipts]

Now here's the question nobody in English-speaking media has asked properly. //
How does a company build this kind of machine // for twenty years //
without a single regulator // saying stop? ///

[SUB: WHO WAS WATCHING?]
[SFX: ticking clock returns, louder]

Because somebody // *had* to be watching.

## 📖 CHAPTER 3 — The Night Beijing Cut the Cord (5:00 - 7:30)

[BGM: slow building suspense]
[B-ROLL: Zhongnanhai exterior, Beijing government buildings at night]

August twentieth, twenty-twenty. //
The Chinese government, //
without warning, //
announced a new policy. ///

[SUB: AUGUST 20, 2020]

Three red lines.

[B-ROLL: three red horizontal lines drawn across the screen, one by one]
[SFX: three blade strikes]

Line one: // developers' liabilities //
must not exceed seventy percent of assets. //

Line two: // net debt must not exceed //
one hundred percent of equity. //

Line three: // cash on hand must cover //
short-term debt. ///

Break one red line, // you can't borrow more. //
Break two, // you can barely breathe. //
Break three — ///

[SUB: BREAK THREE · YOU ARE FINISHED]
[SFX: heavy boom]

Evergrande broke all three. ///

[B-ROLL: Evergrande stock chart falling off a cliff]

Now here is the part that should make you stop and think. //
For twenty years, // Evergrande grew // with the full cooperation //
of the Chinese state banking system. //
State banks lent them money. //
State-owned enterprises bought their wealth products. //
Local governments sold them land // at discounts. ///

[B-ROLL: handshake photos, ribbon-cutting ceremonies]

And then, // in a single afternoon, //
Beijing decided to pull the cord.

[SUB: WHY NOW?]

Why now? ///

[BGM: drops to near-silence]

The official answer // is that the housing market had become dangerous — //
too much debt, // too many empty apartments, //
too much risk to the financial system. //
All true.

[B-ROLL: empty ghost cities, drone footage]

But the *unofficial* answer, //
the one whispered in boardrooms // across Shenzhen and Shanghai, //
is different. ///

[B-ROLL: closed door, dim light underneath]

Xu Jiayin had become // too powerful. //
Too visible. //
Too independent // of the Party. ///

[B-ROLL: Xu Jiayin at charity galas, political conferences]

And in China, //
there is one rule // you do not break. //
No matter how much money you make, //
no matter how many apartments you sell — //
you do not become // bigger than the state. ///

[SUB: DO NOT BECOME BIGGER THAN THE STATE]
[SFX: slow, final drum beat]

September twenty-third, twenty-twenty-one. //
Evergrande missed an interest payment. //
Eighty-three million dollars. ///

[B-ROLL: news headlines flashing — Bloomberg, Reuters, FT]

In global terms, // it was pocket change. //
In political terms, // it was the signal //
that nobody was coming to save them.

## 📖 CHAPTER 4 — The $300 Billion Mystery (7:30 - 10:00)

[BGM: noir jazz, slow and cold]
[B-ROLL: night cityscape, Shenzhen from above]

And now we arrive // at the three questions //
that the Evergrande story leaves behind. ///

[SUB: THREE MYSTERIES]

---

[B-ROLL: pile of money burning]

**Mystery one.** // Where did the money actually go? ///

The official debt is three hundred billion. //
But forensic accountants now estimate //
that off-balance-sheet obligations //
add *another* one hundred and fifty billion.

[SUB: $300B ON THE BOOKS // $150B OFF THE BOOKS]
[SFX: two heavy impacts]

Four hundred and fifty billion dollars //
flowed through this machine. //
And when the liquidators arrived in twenty-twenty-four, //
they found // that the recoverable assets //
were worth // less than three percent of that. ///

[SUB: 3% RECOVERY]
[SFX: deep boom]

Ninety-seven cents of every dollar // simply vanished. ///

---

[B-ROLL: party officials at a meeting, blurred faces]

**Mystery two.** // Who was protecting him? ///

Evergrande did not grow alone. //
Its expansion required // thousands of land deals, //
bank loans, // permits, // and political cover.

[B-ROLL: handshakes with officials, signing ceremonies]

In twenty-twenty-three, //
Chinese anti-corruption authorities //
quietly detained // more than a dozen senior executives, //
including Xu Jiayin himself.

[B-ROLL: news footage, Xu Jiayin detention]
[SUB: XU JIAYIN · DETAINED 2023]

But the officials // who had approved //
every one of those thirteen hundred projects //
for twenty years — /// most of them //
have not been named publicly. ///

The Chinese state // is very careful //
about which parts of the story // become visible.

---

[B-ROLL: foreign investors, Wall Street trading floor]

**Mystery three.** // Why did nobody abroad see this coming? ///

Evergrande issued nineteen billion dollars //
in offshore US dollar bonds. //
BlackRock held them. // Fidelity held them. //
Allianz held them.

[B-ROLL: logos of major asset managers]

Credit rating agencies // rated these bonds //
investment-grade // as recently as twenty-twenty. //
Goldman Sachs research // called Evergrande's debt //
"manageable." ///

[SUB: "MANAGEABLE" — GOLDMAN SACHS, 2020]
[SFX: dry laugh]

When the liquidation order came down // in January twenty-twenty-four, //
recovery on those offshore bonds //
was projected at // less than three cents on the dollar. ///

[B-ROLL: stock tickers, red numbers falling]

Billions of dollars // of Western pension money, //
insurance premiums, // and retirement savings, //
were quietly vaporized. ///

The strangest part? //
Almost nobody in Western financial media //
has written about this // since. ///
