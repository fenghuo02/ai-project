# YouTube Channel · The Red Ledger

This folder contains all planning, scripts, and workflow assets for the YouTube channel **The Red Ledger** — a documentary-style, AI-narrated English channel covering finance, power, and collapse stories.

## Structure

```
youtube/
├── README.md                          # You are here
├── prompts.md                         # Reusable AI prompt library (11 modules)
│                                      # Use these for channel strategy, scripts,
│                                      # titles, thumbnails, SEO, and data review.
└── the-red-ledger/
    └── episode-01-evergrande.md       # Full production pack for Episode 01:
                                       # "Why Beijing Let Evergrande Get This Big"
                                       # Includes script, thumbnail guide,
                                       # publish checklist, and shorts cut list.
```

## How to use

1. Start a new AI session → paste Module 0 from `prompts.md` to load the advisor persona.
2. For each new episode, copy Modules 3 → 4 → 5 → 6 → 7 from `prompts.md`.
3. Save the final production pack as `the-red-ledger/episode-XX-topic.md`.
4. After publishing, run Module 9 (data review) 72 hours later.
